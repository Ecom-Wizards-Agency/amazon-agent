# Promoted Skills

Alexa skill builders can create campaigns to promote skills on Echo Show device home screens.

Promoted Skills is a self-serve ad product for promoting Alexa skills on Alexa channels, starting with Echo Show Home Screen.

Skill builders can create campaigns through Amazon Ads Console and view campaign performance there. Customers can invoke the skill by tapping the screen or speaking the hint text shown in the bottom left of the screen, such as "Alexa, open [skill name]".

## Getting started

1. Go to Alexa Developer Console.
2. Choose the skill to promote.
3. Visit the **Promote your skills** tab.
4. Check whether the skill is eligible.
5. Begin creating the campaign in Amazon Ads Console.
6. As campaigns run, review performance and attribution metrics in Campaign manager. Metrics update daily.
