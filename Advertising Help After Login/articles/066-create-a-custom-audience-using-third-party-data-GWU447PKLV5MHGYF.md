# Create a custom audience using third-party data

Use data collected off Amazon to create custom audience segments.

## How to create an audience

1. In the navigation side menu, click **Campaigns**.
2. Click **Audiences** in the side navigation.
3. Click **New audience**.
4. Click **Your data**.
5. Click **Next** on the tile for the audience type: Amazon Ads Tag, Off-Amazon conversions, or Hashed.
6. Follow the flow for the selected audience type.

## Amazon Ads Tag audience

The ad tag must be properly set up to create an audience based on it.

1. Enter audience name and description.
2. Create inclusion and/or exclusion rules.
3. Choose **match any** or **match all** from the first dropdown.
4. Click **choose event** and search for the desired rule.
5. Enter a lookback window.
