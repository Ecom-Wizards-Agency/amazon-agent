# Live Events Optimizer

This article explains how to set up and optimize Streaming TV ad groups for live events inventory such as sports, concerts, and cultural programming.

## Key Points

- Provides a unified workflow and supply access for live events.
- Uses infrastructure built for live events.
- Supports advanced optimization capabilities.
- Reduces operational friction for live event buying.

## Workflow

1. Create a new ad group with Live Events content type.
2. Configure products and services.
3. Select live events inventory.
4. Configure audience and location targeting.
5. Set up delivery and pacing.
6. Configure bidding settings.
7. Submit creatives for preregistration.
8. Review and launch the campaign.

## Inventory Reference

| Publisher type | Examples | Inventory |
| --- | --- | --- |
| Amazon owned properties | NBA League Pass, UEFA Champions League, WNBA, Amazon Music Live | Open Auction, Programmatic Guaranteed, Preferred Deals, Private Auction |
| Certified open internet publishers | Sports, concerts, award shows, cultural programming | Programmatic Guaranteed, Preferred Deals, Private Auction |

## Checkpoints

- Stop before launching or changing live-event ad groups.
- Save media type and strategy selections carefully.
- Submit creatives early for approvals.
- Monitor campaign performance during and after live events.
