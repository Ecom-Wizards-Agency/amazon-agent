# Amazon Marketing Cloud Home Page

This article explains the Amazon Marketing Cloud Home page and its main functions.

## Key Points

- AMC is a secure, privacy-safe analytics solution.
- The Home page gives access to Insights templates and recent activity.
- Users can customize the Home page by adding or rearranging Insights cards.

## Workflow

1. Go to the **Insights** section on the AMC Home page.
2. Click **Add insight**.
3. Select a template.
4. Edit the layout by dragging and dropping cards.

## Checkpoints

- Review the query before executing it.
- Confirm the selected Insights card is set up correctly before running.
- Stop before running queries that could create cost, expose sensitive reporting, or change shared dashboards.
