# Manage Users in a Manager Account

This article explains how to invite users, set permissions, edit access, remove users, and perform bulk user actions in a Manager Account.

## Key Points

- Users can be invited to the entire manager account or selected accounts.
- Existing permissions can be changed from the Users menu.
- Users can be removed from the Manager Account.
- Bulk actions are available for multiple users.

## Workflow

1. In the left menu, click **Administration**.
2. Click **Account access and settings**.
3. In the left menu, click **Users**.
4. Click **Invite users**.
5. Enter the user's email and click **Add**.
6. Choose **This manager account** or **Selected accounts**.
7. To edit permissions, locate the user and select **Change permissions**.
8. Select the permission level and click **Save permissions**.
9. To remove a user, click **Remove user** and confirm in the popup.

## Checkpoints

- Stop before inviting, changing, or removing a manager-account user.
- Confirm the email and selected accounts.
- Confirm whether access should apply to all accounts or only selected accounts.
