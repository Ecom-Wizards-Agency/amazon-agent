# Consent Requirements for Events Manager (AAT)

This article explains consent requirements for using Amazon Ad Tag in the UK and EEA.

## Key Points

- Amazon Ad Tag helps understand user events in a privacy-safe way.
- Consent must be communicated using supported consent frameworks such as IAB TCF, GPP, or ACS.
- Third-party consent management platforms can simplify ACS setup.

## Workflow

1. Use a third-party or advertiser-owned consent management platform.
2. Configure it to send the required consent signals.
3. Follow the CMP documentation for enabling ACS.

## Checkpoints

- Stop before deploying or changing tags.
- Confirm the CMP supports the needed consent framework.
- Confirm legal/privacy approval for UK and EEA use.
