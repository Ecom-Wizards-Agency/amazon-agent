# Create budget rules for sponsored ads campaigns

Budget rules increase daily budgets based on shopping events and campaign performance.

## Create a budget rule

1. In the navigation side menu, click **Campaigns**.
2. Next to the campaign name that you want to add a rule for, click **More**, then **Edit Settings**.
3. Select **Budget rules**.
4. Click **Add budget rule**.
5. For **Type**, choose schedule or performance.
6. For **Date range**, select a special event or enter a custom start and end date.
7. For special events, choose **Events**, then click the drop-down arrow to see recommended promotional events.
8. For **Rule recurrence**, choose daily to apply the rule every day, or weekly to select certain days for it to run.
9. For **Hours of day** on schedule rules, choose daily for all day, or choose a start and end time.
10. For **Performance metric** on performance rules, choose the performance metric and threshold for increase.
11. Set the budget increase amount.
12. Click **Add budget rule**.

> Note: Hours-of-day scheduling for budget rules is currently only available in the US, CA, UK, IN, and JP marketplaces.

## Budget rule status

After a budget rule is set, Campaign manager shows adjusted daily budget and rule status.

| Status | Details |
| --- | --- |
| Active | The rule is active and increased budget is used for budget enforcement for the day. |
| On hold | The rule is on hold because the performance criteria has not been met. |
| Paused | The rule is paused. |
| Upcoming | The rule will become active in the future. |
| Expired | The rule has reached its end date. |
