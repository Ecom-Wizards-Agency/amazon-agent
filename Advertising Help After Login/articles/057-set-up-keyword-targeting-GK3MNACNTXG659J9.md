# Set up keyword targeting

Keyword targeting lets advertisers add keywords to campaigns and choose match types to control when ads appear in customer searches.

## What is keyword targeting?

Keyword targeting chooses keywords to drive product discovery and sales. Use it when:

- The advertiser knows search terms customers use for similar products.
- The advertiser wants to set bids for individual keywords for more control over targeting and spend.

## Add new keywords to an existing campaign

1. In the navigation side menu, click **Campaigns**.
2. Select the campaign.
3. Select the ad group.
4. Click **Targeting** in the left-hand menu.
5. Click **Add keywords**.
6. Choose keywords and match types.
7. Browse suggested keywords, search, or upload a list.
8. Click **Add** next to each keyword.
9. Click **Save**.

## Tips for choosing keywords

1. Add at least 25 keywords in each campaign and ad group.
2. When using multiple match types for the same keyword, set different bids for each match type.
3. Use branded terms and similar product category keywords to expand reach.
4. Consider specificity. Very specific keywords may have low impressions but stronger conversion.
5. Include complementary keywords, such as "digital camera" for memory cards.
6. Replace low-performing keywords and increase bids on high-performing keywords using search terms report insights.
7. In Sponsored Products, use **Filter by** to choose Clicks or Orders for keyword recommendations.
8. Use suggested keyword groups to discover relevant keywords. Only one bid value is needed for each keyword group.
9. Use dynamic up and down bidding with keyword groups to adjust bids automatically based on shopper response and performance.

## Match types

Manual targeting campaigns have three match types:

- **Broad:** contains all keywords in any order and includes related keywords.
- **Phrase:** contains the exact phrase or keyword sequence.
- **Exact:** exactly matches the keyword or keyword sequence.

In Sponsored Brands, phrase match can also show ads on searches with similar meaning or intent, even if wording differs.

## Monitor keyword performance

After a campaign has run for two weeks, use the search term impression share (SIS) report to check which words perform best and which search terms generate traffic and clicks.

Suggested bid and suggested bid range are based on ads that won impressions over the past 7 days.

- **Impression share (IS):** percentage of impressions received on a term relative to all impressions the term generates. IS of 0% means no impressions were received.
- **Impression rank (IR):** where IS stands relative to other advertisers' impression share for the same term.
