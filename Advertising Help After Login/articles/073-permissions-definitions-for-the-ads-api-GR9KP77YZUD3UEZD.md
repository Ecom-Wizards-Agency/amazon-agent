# Permissions Definitions for the Ads API

This article explains Amazon Ads API permissions and access levels for integrations across sponsored ads and Amazon DSP.

## Key Points

- Third-party applications can manage accounts through the Amazon Ads API only when authorized.
- Sponsored ads and Amazon DSP have different permission definitions.
- API access should match the minimum scope needed for the intended workflow.

## Permission Areas

| Area | What to Review |
| --- | --- |
| Sponsored ads permissions | Campaign, reporting, and account-management permissions for sponsored ads. |
| Amazon DSP permissions | DSP access rights and supported API activities. |

## Checkpoints

- Stop before authorizing a third-party application.
- Confirm which API permissions are required and who owns the advertiser account.
- Avoid granting broad permissions when a narrower scope is enough.
