# Add Accounts to Your Manager Account

This article explains how to link or create accounts under a Manager account.

## Key Points

- Only manager account admins can link existing advertiser accounts.
- Linking accounts creates a hierarchy for easier management.
- Amazon DSP advertiser accounts created under the manager account may not be unlinkable.

## Workflow

1. In the left menu, click **Administration**.
2. Select **Account access and settings**.
3. Click **Accounts**.
4. Click **Add account**.
5. Choose whether to link an existing account or create a new sponsored ads/Amazon DSP advertiser account.
6. Follow the account-specific instructions.

## Visual Reference

![Administration icon](https://d369o5h5zn8mv7.cloudfront.net/images/Icons/Administration.png)

## Checkpoints

- Stop before linking or unlinking an account.
- Confirm you are a manager account admin.
- If you are not an admin, use the request access path instead of attempting a link.
- Confirm whether the account can be unlinked later before creating it under the manager account.

## Routing Use

Use this page when the user asks to add client accounts, connect accounts to a manager account, or organize Amazon Ads accounts under an agency structure.
