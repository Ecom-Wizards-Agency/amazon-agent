# Amazon Marketing Cloud Use Case Library

This article explains the AMC use case library, which contains ready-to-use custom solutions and SQL queries.

## Key Points

- The use case library provides custom solutions and SQL queries for AMC.
- It supports measurement and audience analysis use cases.
- Each use case can include business context and usage instructions.

## Workflow

1. Select the use case closest to the business need.
2. Review listed instructional queries and solutions.
3. Launch a solution and follow on-screen instructions, or select an IQ and review the instructions.
4. Filter by attributes such as features, ad products, and use case category.

## Checkpoints

- Stop before running or modifying SQL queries.
- Confirm the query matches the intended measurement or audience use case.
- Review date range, privacy thresholds, and output destination.
