# Manage Amazon DSP Campaigns With Bulksheets

This article explains how to manage Amazon DSP campaigns through bulksheets.

## Key Points

- Bulksheets can create, update, or archive campaigns.
- They can upload or update creative assets.
- They support audience management and failed-import troubleshooting.

## Workflow

1. In the navigation side menu, click **Campaigns**.
2. Access bulksheets from the Campaigns or Ad groups page.
3. In the ad group workflow, create multiple ad groups or recommended ad groups.
4. Download the template.
5. Review suggested ad groups.
6. Add required targeting and settings.
7. Save and upload the updated file.
8. Track status in **Bulk actions** under the Campaigns tab.

## Limits

| Limit | Value |
| --- | --- |
| Max upload file size | 2 MB |
| Ad groups/campaigns per upload | Up to 1,000 |
| Update operations per user/entity/day | Up to 5,000 |
| Ad groups/campaigns per download | Up to 7,500 |

## Checkpoints

- Stop before uploading a bulksheet.
- Confirm the correct account, advertiser, and campaign scope.
- Fix errors shown in the Upload issue column before uploading again.
- Keep a backup of the original export before making bulk changes.

## Routing Use

Use this page when the user asks to bulk create, update, archive, or troubleshoot Amazon DSP campaigns.
