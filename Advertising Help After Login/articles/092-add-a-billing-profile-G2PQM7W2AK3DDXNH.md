# Add a Billing Profile

This article explains how to add or remove billing profiles from an Amazon Ads manager account.

## Key Points

- Billing profiles are only available in manager accounts.
- Each country requires its own billing profile for ads to display.
- Valid payment information is required for ads to appear in a country.

## Workflow

1. Sign in to Amazon Ads.
2. Use the account name in the top-right to confirm you are in a manager account.
3. Click **Administration** in the left menu.
4. Select **Manager account billing dashboard**.
5. Choose the manager account and select **Billing profiles**.
6. Click **Add Billing Profile**.
7. Enter required billing, phone, and tax information.
8. For Canadian billing profiles, choose the relevant GST information if applicable.
9. Click **Save** or **Save as default**.
10. Link existing managed-service accounts if needed.
11. Notify account managers of the new billing profile address.

## Checkpoints

- Stop before adding, removing, or setting a default billing profile.
- Confirm manager account context first.
- Confirm country, tax, billing address, and payment responsibility before saving.
