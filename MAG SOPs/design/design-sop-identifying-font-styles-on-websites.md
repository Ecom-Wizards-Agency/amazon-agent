# Design SOP: Identifying Font Styles on Websites

Source: https://sop.myamazonguy.com/books/design-XyS/page/design-sop-identifying-font-styles-on-websites

## **Identifying Font Styles on Websites**

**Who is this for?**

This procedure applies to anyone who needs to analyze and identify the font styles used on a website, including graphic designers, web developers, and marketers.

**Objective**:

This SOP provides a step-by-step guide to identify the font styles used on a website, either by utilizing the browser’s developer tools or by using the Fonts Ninja extension.

Option 1: Using Browser Developer Tools

### **Option 1: Using Browser Developer Tools**

**Step 1: Open the Website**

- Navigate to the website you want to analyze.

**Step 2: Inspect the Element**

- Right-click on the text whose font style you want to identify.
- From the context menu, select "**Inspect**" or "**Inspect Element**." This will open the browser’s developer tools panel.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/C1Yimage.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/design-sop-identifying-font-styles-on-websites-image-rnzv2i11.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/W3oimage.png)

**Step 3: Navigate to the Computed Tab**

- In the developer tools panel, locate and click on the "**Computed**" tab. This tab shows the computed styles of the element you have selected.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/W57image.png)

**Step 4: Locate the Font Style Information**

- Scroll down in the Computed tab until you find the "**Rendered Fonts**" section. This will display the font family, postscript name, and other relevant font details used for the selected text.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/hctimage.png)

Option 2: Using the Fonts Ninja Extension

### **Option 2: Using the Fonts Ninja Extension**

**Step 1: Install Fonts Ninja Extension**

- Go to the following link in the Chrome browser:[Fonts Ninja Extension](https://chromewebstore.google.com/detail/fonts-ninja/eljapbgkmlngdpckoiiibecpemleclhh).
- Click "**Add to Chrome**" to install the extension.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/CbWimage.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/mRaimage.png)

**Step 2: Open the Website**

- Navigate to the website you want to analyze.

**Step 3: Activate Fonts Ninja**

- Click on the Fonts Ninja icon in your browser’s toolbar after installation.

**Step 4: Hover Over Text to Identify Font**

- Hover over the text you want to analyze. Fonts Ninja will display the font name, weight, and other relevant font details directly on the website.

  ![design-sop-identifying-font-styles-on-websites-image-tlew7uni.gif](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/design-sop-identifying-font-styles-on-websites-image-tlew7uni.gif)

**Step 5: View Detailed Font Information**

- Clicking on the Fonts Ninja icon will provide you with additional font details, including font family, size, line height, and other properties.

By following either of these methods, you can easily identify the font styles used on any website. Using developer tools gives you in-depth information on the rendered fonts, while Fonts Ninja provides an easier, more user-friendly option to quickly identify fonts.
