# Walmart SOP: Create Ticket to Walmart Seller Support

## **To File a Ticket to Walmart for Updating the Title, Description, Key Features, Images, etc.**

**First, prepare any proof of UPC.**

- Link to Manufacturer's website for specific item
- Image of the product with the product’s title and UPC in clear view
- Receipt from GS-1 and screenshot from GS-1 showing that you are the owner of the GTIN
- Other reputable retail website that shows matching UPC or Model Number

Then

1. Log in to Walmart seller central account. Go to [https://seller.walmart.com/](https://seller.walmart.com/)

**Provide the following:**

1. **Subject line**
2. **What were you trying to update?** - (Choose all the attributes that need an update)
3. **Where are you checking or verifying the content updates?** - Walmart.com
4. **Please provide the most recent feed ID** - Make sure you have the feed ID, located in Activity Feed
5. **Item ID** - Search in the Items tab
6. **Please provide description of the issue** - Explain problem
7. **CSV File Template** (see image below)

Download and Fill out the CSV file. Sample below.

**![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/0.8.jpg)**

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/3r4image.png)

Provide the following information

**Product ID (GTIN/UPC/ISBN)** - Can be GTIN/UPC/ISBN

**Incorrect Content?** - Title, Key Features, Description, Main Image, Secondary Image

**What is the correct content? Or image URL?** - Provide the correct content or image.

**Proof of UPC being correct**

Choose one from the list which one are you going to provide.

- Link to Manufacturer's website for specific item
- Image of the product with the product’s title and UPC in clear view
- Receipt from GS-1 and screenshot from GS-1 showing that you are the owner of the GTIN
- Other reputable retail website that shows matching UPC or Model Number

**URL to Proof of UPC**

If the proof is Image of the product that shows UPC. Upload the image to imgur.com, get the image link and put the link to this column.

Save the filled CSV file, return to the case creation page, tick the **"I understand and accept"** checkbox, upload the CSV file, click **Submit**, and the ticket ID will appear.

FAQs

1. What proof of UPC should I provide for my update request?

- You can provide one of the following proof types:

  - Link to Manufacturer's Website: Provide the manufacturer’s official website where the UPC is listed.
  - Image of the Product with UPC: Upload an image showing the product with its title and UPC clearly visible.
  - Receipt from GS-1: Provide the receipt or a screenshot showing that you are the owner of the GTIN.
  - Other Reputable Retail Website: If available, you can link to a retail website showing the same UPC or model number.

2. How do I verify that I have uploaded the correct CSV file before submitting?

- After filling out the CSV file:

  - Double-check the file for accuracy, ensuring that all required fields (Product ID, Incorrect Content, Correct Content) are filled.
  - Make sure the proof of UPC is included in the appropriate section.
  - Save the file and revisit the ticket creation page to upload it.
  - Preview the file before final submission to confirm all the data is correct.

3. What should I do if my content update request is not resolved?

- If the content update is not resolved or you haven't received a response:

  - Follow up by checking the case status in the Your Cases section under the Question Mark icon.
  - If the case is unresolved, respond directly within the case link to inquire about the status or to provide additional information.
  - You may need to resubmit your case or provide further clarification if the issue was not fully addressed.

4. How do I handle a case rejection or if the update is not applied?

- If Walmart support rejects the update or the changes are not reflected after submission:

  - Review the rejection reason if available and ensure that the details provided are correct.
  - If the issue persists, recontact support and ask for clarification or provide additional proof if needed.
  - Consider submitting the case again with updated or more detailed information, especially if there was a misunderstanding.

5. How can I check the status of my case after submitting a content update request?

- To check the status of your case:

  - Click the Question Mark icon located at the top-right of the page.
  - Select Your Cases from the dropdown menu.
  - Search the case ID or scroll through the list to find the status and any updates from Walmart support.

6. Can I edit or update my submission after uploading the CSV file?

- Once the CSV file is uploaded and the ticket is submitted, you cannot directly edit the submission.

  - If errors are identified before submission, correct them in the CSV file and re-upload.
  - If changes are needed after submission, reopen the case or create a new request if necessary.

7. What should I do if the content is not displaying correctly on Walmart after the ticket is processed?

- If changes are not reflected on the Walmart product page:

  - Wait for 24-48 hours for the system to update.
  - If changes are still not visible, contact Walmart support again using the case ID to investigate further.
  - Provide any additional evidence or clarification if needed.

8. What if the incorrect content is not listed in the fields I’m updating (e.g., title, description, images)?

- If you notice missing content fields in the request (e.g., title, description, images), ensure that the CSV file is correctly filled and that the fields match the issue you're reporting.

  - Double-check that you have selected the right product ID and associated it with the correct content.
  - If the problem persists, follow up with Walmart support and clarify which attributes are incorrect.

9. How can I ensure I’m submitting the correct product ID in my update request?

- The product ID (GTIN/UPC/ISBN) can be found in the Item ID field when searching for the item in the Items tab.

  - Ensure that the correct product ID is selected for the item you are updating.
  - If unsure, double-check the Item ID in the product catalog to confirm the correct match.

10. Is there a maximum number of content updates I can request for a single item?

- There is no maximum number for submitting content updates for a single item. However, each request must be clear and specific, detailing the necessary updates (e.g., title, description, images).

  - For multiple changes, it is advisable to include all the requested updates in one submission to avoid confusion or delays.
