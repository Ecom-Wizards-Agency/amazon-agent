## **Create an Offer for an Existing Item in Walmart (Seller Fulfilled Type)**

**Who is this for :** For individuals who want to create an offer for an existing Walmart listing.

**Objective :** Successfully create an offer for an existing item in Walmart and verify if the existing item information is all correct.

**Required attributes are:**

- SKU
- Product ID (only UPC, EAN, GTIN, ISBN are allowed)
- Product ID Type
- Fulfillment Type
- Selling Price
- Package Weight (lbs) if in a different weight unit please convert lbs.

**Instructions:**

1. Log in to Walmart seller central account. Go to https://seller.walmart.com/ ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1689291796416.png)

2. Go to Items under the Product Catalog. And click the Add Items button in the upper right corner. ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1689291833946.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/r0Mimage.png)

3. You will be redirected to the Add page. ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1689291859510.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/uOAimage.png)

4. Search the product ID in the search box to see if the listing already exists in the marketplace. If the product exists, a result will show up (like the image below). Click the Add item to add it to create the offer. ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1689291896864.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Cj2image.png)

FAQs

1. What does creating an "offer" for an existing item mean in Walmart Seller Center?

Creating an "offer" for an existing item means that you are adding your version of the product to a pre-existing listing. This allows you to sell the product while referencing an already established Walmart listing (known as the “parent listing”). This is particularly useful when multiple sellers offer the same product.

2. What are the required fields to create an offer for an existing item?

When creating an offer for an existing item on Walmart, the following fields are required:

- SKU (Stock Keeping Unit)
- Product ID (UPC, EAN, GTIN, ISBN)
- Product ID Type (e.g., UPC, GTIN)
- Fulfillment Type (e.g., Seller Fulfilled)
- Selling Price
- Package Weight (lbs) (Make sure to convert to pounds if using a different unit).

These are the minimum attributes to create the offer, but additional details, like product variations, may be required depending on the listing.

3. How do I ensure that my SKU format is valid?

Your SKU must follow Walmart’s specific formatting rules:

- It must be alphanumeric, with no special characters (other than dashes or underscores).
- Ensure that SKUs are unique for each item in your catalog.
- Walmart allows SKUs up to 50 characters long.
- Always make sure your SKU aligns with your internal inventory management system.

4. Can I add multiple variants of the same product when creating an offer?

Yes, you can create offers for multiple variants or multiple SKUs of a product. In this case:

- Each variant should have its own unique SKU and associated attributes (e.g., size, color).
- Variants can be added to the same parent listing if they belong to the same product.
- When creating the offer, ensure that the correct variant information (e.g., size, color, weight) is reflected for each SKU.

5. How do I verify if the existing listing’s category or attributes are correct?

To verify if the category or attributes of the existing listing are correct:

1. Search for the product ID in the search box under the Items section of the Product Catalog.
2. If the product exists, check its category and product details to ensure they match what is expected.
3. You can edit some fields (like price, weight, SKU) during the offer creation, but category and product ID should be accurate.
4. If any attributes are incorrect, contact Walmart support or create a new listing.

6. How do I update information for an existing offer?

Once your offer is created, you can update the following:

- Price: Adjust your price directly through the Items section.
- Description: To modify descriptions, you will need to use the bulk upload tool or contact support if you can't edit directly.
- Stock Level: Update stock levels in your Seller Center dashboard, ensuring that the inventory matches the listing.

Note: Some attributes like product ID and category cannot be updated without creating a new offer or contacting Walmart support.

7. What should I do if I encounter errors during the offer creation process?

1. Check the Feed ID in the Activity Feed to confirm whether the offer upload was successful.
2. If errors are reported, download the error report and look for common issues such as:

  - Invalid product ID or SKU.
  - Missing attributes, such as price or weight.
  - Invalid file format.
3. Fix the errors by updating the relevant fields and re-upload the file.
4. Common errors might include:

  - Product ID mismatch: Ensure the product ID (UPC, EAN) matches the Walmart database.
  - Price formatting issues: Make sure the price field does not contain special characters like currency symbols.

8. How long does it take for the offer to be processed and visible?

- Initial offer processing can take anywhere from 15 minutes to 2 hours depending on the complexity and number of items.
- For large updates or multiple SKUs, it may take up to 4 hours.
- You should see the updated offer on the product detail page (PDP) within 5-10 minutes. If not, it might take up to 24 hours to reflect.

9. How do I verify that my offer information is correct after submitting?

1. Go to the Product Catalog and check if the listing is visible under the Items section.
2. Click on the product and select View on Walmart.com to confirm if the changes (price, description, images) appear correctly.
3. Check Activity Feed for the Feed ID status to see if the upload was successful and if the changes were processed.

10. What should I do if the offer is not visible on Walmart.com?

If your offer is not visible after the expected processing time:

1. Check Activity Feed to confirm if there were any upload errors.
2. Verify if the offer appears in the Items section of your Seller Center.
3. If the problem persists, submit a support case through Walmart Partner Support for further assistance.
