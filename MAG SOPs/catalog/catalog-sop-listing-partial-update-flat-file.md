## Listing Update (Flat File)

**Who is this for :** This is intended for Amazon Sellers and other individuals responsible for updating listing information on Amazon.

**Objective :** To provide a step-by-step guide for Amazon Sellers to update listing attributes using flat files. This SOP is designed to help sellers who have a large number of listings and need to make changes to multiple attributes simultaneously. By following the procedures outlined in this SOP, sellers can avoid costly mistakes and save time, allowing them to focus on other aspects of their business.

**Instructions:**

FAQS

**1. What should I do if my flat file upload results in an error?**

- First, download the Processing Report under "Monitor Upload Status." The error message will indicate the issue. Common errors include missing required fields, incorrect formatting, or category mismatches. If you’re unsure how to fix it, check Amazon’s error code documentation or consult Seller Support.

**2. My listing updates aren’t reflecting on the product page. What should I do?**

- Amazon can take time to process updates. If your changes aren’t showing after 24 hours:

  - Check if the processing report shows any errors.
  - Confirm that your seller contribution is prioritized (Ctrl + F on backend values).
  - Try clearing cache or viewing the PDP in incognito mode.
  - If no gray fields appear for key attributes (like search terms), submit a case to Amazon.

**3. How do I revert changes if I made a mistake in my flat file upload?**

- If incorrect data was uploaded, create a new flat file with the correct information and re-upload it. If the mistake affected a critical listing field and doesn’t revert easily, submit a ticket to Amazon and request a manual correction.

**4. Do I need to back up my listings before making bulk updates?**

- Yes, always download the Category Listings Report before uploading any flat file updates. This serves as a backup in case errors occur, and you need to restore previous data.

**5. What should I do if my listing is suppressed or rejected after an update?**

- Check the Suppressed Listings tab in Seller Central for specific reasons.
- If caused by missing attributes, fill them in and re-upload the flat file.
- If flagged for compliance issues, submit the required documents through Amazon’s case system.

**6. How do I update parent-child variations in a flat file?**

- Ensure the Parent SKU and Child SKUs are structured correctly.
- Use "Parent" for the parent listing and "Child" for the variations.
- Fill out the Relationship Type column (e.g., variation) and the correct Variation Theme (e.g., Size, Color).
- Submit the file under PartialUpdate to avoid breaking the relationship structure.

**7. What if I can't find my product category when downloading a flat file template?**

- Ensure you’re searching for the exact category name as it appears on your live listing.
- If it's still missing, check if the product falls under a restricted category requiring approval.
- As a last resort, contact Amazon Seller Support and request assistance.

**8. Where do I submit a ticket if my updates are not saving in the backend?**

- Go to Seller Central > Help > Get Support > Fix a Product Issue > Product Detail Page Not Updating and submit a case. Include the SKU, ASIN, a screenshot of the current backend values, and the requested changes.
