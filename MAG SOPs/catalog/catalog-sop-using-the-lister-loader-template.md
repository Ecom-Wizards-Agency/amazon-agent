# Catalog SOP: Using the Lister Loader Template

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-using-the-lister-loader-template

## **Using the Listing Loader Template**

**Who is this for :** Individuals need a template for creating or adding their products that already exist in the Amazon catalog.

**Objective :** This is to provide efficient and streamlined functionality for Amazon sellers to match products, reduce manual effort, minimize data requirements, and upload directly to Amazon from Excel.

**Instructions:**

1. If you have multiple ASINs for a single UPC or EAN, consider using the **"Listing Loader Template."**

2. Sometimes, when you click on "Add a Product" from the "Catalog" menu in Seller Central and enter a UPC or EAN, you may see more than one ASIN. In that case, review all ASIN product pages and select the one with the best content, rankings, and reviews.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Using-the-Lister-Loader-Template-1.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/o6himage.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Using-the-Lister-Loader-Template-2.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/laMimage.png)

3. Once you decide which ASIN to use, return to Catalog and click "Add Products via Upload."

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Using-the-Lister-Loader-Template-3.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Zeyimage.png)

4. Then, under "Download Spreadsheet," click "Get Listing Loader" to the right of "Product Matching only."

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Using-the-Lister-Loader-Template-4.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/lvSimage.png)

5. On the Pop-Up Window, choose the exact template by Marketplace of "Listing Loader Template" to download it.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Using-the-Lister-Loader-Template-5.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/BfCimage.png)

6. In the Listing Loader Template, make sure to hit 'Enable Editing' first.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Using-the-Lister-Loader-Template-6.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/NA3image.png)

7. Insert the ASIN you want to use in the "ASIN Hint" field.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Using-the-Lister-Loader-Template-7.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/tyJimage.png)

**Note:**

- If you're using Brand Registry, use the UPC or EAN as the Product ID instead of the ASIN. You can map the ASIN to the UPC or EAN using the Listing Loader.
- If you're using the Listing Loader for a product with only one ASIN on Amazon, don't put the ASIN in the "ASIN Hint" field.
- Unlike the regular flat file, the Listing Loader has some fields not available in the Inventory Loader Template.

8. Once you complete the file, save and upload it to Seller Central using the "Add Products via Upload" option from the "Catalog" menu. Choose your file type, select your file, and upload it. You can skip the "Check File" step if you prefer.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Using-the-Lister-Loader-Template-8.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/rwaimage.png)

9. Check the processing report to ensure there are no errors. Your products will be uploaded to your catalog.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Using-the-Lister-Loader-Template-9.png)

FAQs

1. What is the Listing Loader Template?

The Listing Loader Template is a tool designed for Amazon sellers to efficiently add products to their catalog by matching existing products on Amazon. It allows sellers to reduce manual effort and upload product details directly from an Excel file.

2. When should I use the Listing Loader Template?

Use the Listing Loader Template when you have multiple ASINs for a single UPC or EAN, or when you want to match existing products in the Amazon catalog to streamline the listing process.

3. How do I find the correct ASIN to use for my product?

If you see multiple ASINs for the same UPC or EAN, review the product pages for each ASIN. Choose the ASIN with the best content, highest rankings, and most favorable reviews to ensure your product appears alongside the best listing.

4. How do I download the Listing Loader Template?

After selecting the ASIN, go to Catalog > Add Products via Upload. Then, click "Get Listing Loader" under "Download Spreadsheet" next to "Product Matching only." Choose the appropriate marketplace and download the template.

5. What should I do after downloading the Listing Loader Template?

Once downloaded, open the template and click 'Enable Editing'. Then, insert the ASIN in the "ASIN Hint" field.

6. What should I do if I’m using Brand Registry?

If you’re using Brand Registry, use the UPC or EAN as the Product ID instead of the ASIN. You can map the ASIN to the UPC or EAN in the Listing Loader.

7. What if the product only has one ASIN on Amazon?

If your product has only one ASIN on Amazon, do not insert the ASIN in the "ASIN Hint" field.

8. How do I upload the completed file to Amazon?

Once the template is completed, save the file and go back to Catalog > Add Products via Upload in Seller Central. Choose your file type, select your file, and upload it. You can skip the "Check File" step if you prefer.

9. How can I verify if the upload was successful?

After uploading, check the processing report to ensure there are no errors. If the file uploads successfully, your products will be added to your catalog.

10. Can I use the Listing Loader Template for products that don’t exist in the Amazon catalog?

No, the Listing Loader Template is designed for adding or matching products that already exist in the Amazon catalog. If you need to create a new product listing, use a different upload method, such as the Inventory Loader.
