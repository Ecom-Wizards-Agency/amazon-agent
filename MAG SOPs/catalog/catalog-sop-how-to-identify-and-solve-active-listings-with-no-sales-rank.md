# Catalog SOP: How To Identify & Solve Active Listings With No Sales Rank

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-how-to-identify-solve-active-listings-with-no-sales-rank

## **How To Identify & Solve Active Listings With No Sales Rank**

**Who is this for :** This is for all individuals and sellers with listings that do not display any BSR/ Sales Rank.

**Objective :** To present a clear and concise set of instructions for identifying Active Listings that lack a Best Seller's Rank (BSR) and resolving the issue to display Sales Rank. The following step-by-step guide outlines the necessary actions to address this matter efficiently.

**Identify Active Listing with No Sales Rank**

1. Go to Seller Central Account
2. Click Manage All Inventory

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%202.50.38%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Ektimage.png)

3. Make sure that Sales Rank is listed on the filters. If not, click on “Preferences”

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%202.50.47%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/JVnimage.png)

4. Tick the Sales Rank checkbox then click “Save Changes” at the bottom.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%202.51.03%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/emGimage.png)

5. Search for the ASIN that is active with No Sales Rank.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%202.51.13%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/qJyimage.png)

Once you have identified the Active Listing with No Sales Rank /BSR, pull the category listings report and check if the ASIN is assigned to a category and the item type is not missing. (Check the backend as well)

6. Once logged in to seller central, under the Inventory tab click “Inventory Reports” and download the Category Listing Report (CLR).

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%202.51.24%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/LtMimage.png)

7. If in any case that there is no Category Listing Report (CLR) option on the Inventory Reports, create a ticket to Amazon requesting to enable the Category Listing Report (CLR).

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%202.51.37%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/EMWimage.png)

8. Once Category Listing Report is downloaded, open the downloaded file then go to the “Review” section and click “Unprotect sheet”. Please turn on the filter and locate the problematic listing (SKU) by searching it through the Seller SKU column.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%202.51.47%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/kMTimage.png)

Note: Make sure you are only updating the problematic listing/s.

9. Make sure to fill in the correct product type and item type and upload the flat file. Afterward, create a ticket and provide the batch ID of the flat file upload.

**No Sales Rank Due To Adult Tagging/Flagging How To Identify Adult Flagging**

After filing a ticket to Amazon regarding the missing sales rank, you will receive many different reasons behind the issue. One of them is Adult Tagging. This is when your product listing was wrongfully tagged as an “adult/explicit product” due to different reasons like:

- You added the keyword “Adult” in the backend
- A customer review has deemed your product to be an “explicit item”

Their case response will look something like this:

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%202.52.37%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/GA1image.png)

To solve the issue, you may proceed with filing a ticket similar to the previous steps with a few revisions:

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%202.52.52%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/7Zdimage.png)

**Describe the Issue, change the email address, and hit Send. Make sure to remember 5 pointers:**

1. Explain what the product really is
2. Reference any existing case proving that Amazon flagged the listing as "adult"
3. Include the affected ASINS
4. Describe the contact reason as Product Listing Flagged As Adult
5. Change the default email to your work email address

If the ASIN/s you supplied were not detected by the system, you may proceed with the "contact an associate" section of the said ticket.

FAQs

**1. Why does my active listing have no Sales Rank (BSR)?**

- A listing may not display a Best Seller Rank due to:

  - No sales history (BSR updates only after a product sells at least once).
  - Incorrect category assignment (Amazon does not rank improperly categorized products).
  - Buy Box suppression (if your listing is suppressed, it may lose visibility in ranking).
  - Adult/Explicit content flagging (Amazon removes BSR from adult-tagged items).
  - Listing suppression due to missing product details (incomplete backend attributes).

**2. How can I check if my ASIN previously had a Sales Rank?**

- Go to Amazon Seller Central > Reports > Business Reports > Detail Page Sales and Traffic.
- If the ASIN had previous sales, check Historical BSR data via third-party tools like Keepa or CamelCamelCamel.

**3. How do I check if my listing is suppressed or missing the Buy Box?**

- Go to Seller Central > Inventory > Manage Inventory.
- Click on Suppressed Listings to see if Amazon has flagged the product.
- If there is no Buy Box, it could be due to pricing issues or listing errors.

**4. How long does it take for Amazon to update my Sales Rank after fixing the issue?**

- If the ASIN had no sales, BSR appears only after the first sale.
- If you updated category/item type information, Amazon typically updates within 24–72 hours.
- If flagged for adult content, it may take 5–7 business days after Amazon removes the restriction.

**5. How can I check if my category update request was processed?**

- Go to Seller Central > Inventory > Upload Status and check the batch ID status.
- If the update was rejected, download the error file and correct the mistakes.
- If no change occurs after 72 hours, open a case with Amazon Support.

**6. What if Amazon refuses to enable the Category Listing Report (CLR)?**

- Try downloading the Listing Quality and Suppressed Listings Report instead.
- Use Amazon’s flat file templates for category updates if CLR is unavailable.
- Contact Seller Support and provide specific ASINs needing updates.

**7. What are common errors when updating Product Type and Item Type Keyword?**

- Mismatched category and item type → Ensure the item type keyword aligns with the correct Amazon category.
- File formatting errors → Use a UTF-8 encoded CSV file and do not modify required columns.
- Incorrect flat file version → Always download the latest template from Seller Central > Inventory > Add Products via Upload.

**8. How can I confirm if Amazon removed the Adult Flag from my listing?**

- Go to Manage Inventory and check if the Sales Rank reappears after 48–72 hours.
- Try running Sponsored Brands or Sponsored Display Ads—if the product is still flagged, Amazon won’t allow ad creation.
- If the issue persists, reopen the case with Amazon Support referencing the previous case ID.
