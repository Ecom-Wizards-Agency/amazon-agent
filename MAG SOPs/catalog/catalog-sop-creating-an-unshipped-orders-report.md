# Catalog SOP: Creating an Unshipped Orders Report

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-creating-an-unshipped-orders-report

## Creating an Unshipped Orders Report

**Who is this for :** All individuals who need to create an unshipped order report.

**Objective :** To create an unshipped order report to track all orders that have been placed but not yet shipped. This allows businesses to know what needs to be sent out, preventing any overlook or delay.

**Instructions:**

1. To generate an Unshipped Orders Report, go to the "Orders" tab in Seller Central and select "Order Reports" from the drop-down menu.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Creating-an-Unshipped-Orders-Report-1.png)

2. On the next page, click the "Request" button under the "Unshipped Orders" tab.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Creating-an-Unshipped-Orders-Report-2.png)

3. If desired, select the "Include a column indicating the sales channel for each order" checkbox to add this column to the report.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-Creating-an-Unshipped-Orders-Report-3.png)

FAQs

1. Where can I find the "Unshipped Orders" report in Seller Central?

To access the Unshipped Orders Report, navigate to the Orders tab at the top of Seller Central, then select Order Reports from the drop-down menu. On the next page, click the "Unshipped Orders" tab.

2. How do I request the Unshipped Orders Report?

Once you're on the Unshipped Orders tab, click the "Request" button to generate the report. This will initiate the process of creating the report for unshipped orders.

3. Can I include the sales channel in the Unshipped Orders Report?

Yes, you can include a sales channel column in the report by checking the box labeled "Include a column indicating the sales channel for each order". This will add the sales channel information for each order to your report.

4. How can I customize the columns in the Unshipped Orders Report?

To customize the columns, click on the "Add or remove order report columns" link in the top-right corner of the page. From there, you can select or deselect the columns you want to include or remove in the report. After making your selections, click "Save changes" to apply the updates.

5. How do I know when my Unshipped Orders Report is ready for download?

After requesting the report, you will see the Report Status initially marked as "Request Submitted" and Download Action as "Not Ready". Click the "Refresh" button at the top-right of the table until the Report Status changes to "Processing Complete" and a clickable link for downloading the report appears under the Download Action column.

6. How do I download the Unshipped Orders Report once it's ready?

Once the Report Status changes to "Processing Complete", a clickable link will appear in the Download Action column. Click the link to download the report.

7. Can I download multiple Unshipped Orders Reports at once?

No, you can only request and download one report at a time. After downloading the current report, you can generate another by following the same process.

8. How long does it take for the Unshipped Orders Report to be processed?

The report generation time may vary depending on the size of the dataset. After requesting the report, click the "Refresh" button periodically until the Report Status changes to "Processing Complete".

9. What happens if the Unshipped Orders Report fails to generate?

If the report does not generate or is delayed beyond a reasonable time:

- Double-check your customization settings and make sure the report was requested correctly.
- Try refreshing the page.
- If the issue persists, contact Seller Support for further assistance.

10. What file format will the Unshipped Orders Report be available in?

The Unshipped Orders Report will typically be available in .csv format, which you can open in Excel or other spreadsheet applications for further analysis.

11. How can I track the progress of the Unshipped Orders Report?

You can track the progress of the report by checking the Report Status in the table. Initially, it will show as "Request Submitted" and will change to "Processing Complete" once the report is ready for download.

12. Is there a limit to the number of orders in the Unshipped Orders Report?

There is no specific limit mentioned in the SOP for the number of orders, but the report will typically include all unshipped orders within the selected timeframe. If you are handling a very large dataset, you might want to filter by a specific date range to ensure the report loads correctly.
