# Catalog SOP: Downloading an All Statements Report

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-downloading-an-all-statements-report

## Downloading an All Statements Report

**Who is this for :** For sellers on Amazon Seller Central who need guidance on obtaining an "All Statements" report for their seller account settlement.

**Objective :** The objective of this SOP is to guide sellers in retrieving the "All Statements" report from their Amazon Seller Central account. It provides instructions on how to access and download the settlement report, which includes important financial information related to the account activity and balance.

**Instructions:**

1. To obtain an “All Statements” report, go to the “Payments” tab in Seller Central and select “Payments” from the Drop-Down Menu.![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1716133424723.png)

  [![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/hRQimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/hRQimage.png)
2. On the next page, navigate to “All Statements” Tab to view all available statements.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684344726906.png)

  [![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/fdKimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/fdKimage.png)
3. If desired, use the Select Date Range filter to specify the time period for which you want to generate the report.

  [![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/aSrimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/aSrimage.png)

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684345267340.png)
4. Click "Search".

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684345612582.png)

  [![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/FaYimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/FaYimage.png)
5. Look for the settlement period you're interested in and locate the corresponding Download buttons in the Actions column. There are three options for downloading the report:

  1. Download XML: This option provides an XML file that you can open in programs like Microsoft Excel.
  2. Download Flat File: Selecting this option will download a tab-delimited text file.
  3. Download Flat File V2: Specifically designed for Fulfillment by Amazon sellers, this file can be used with the Settlement Summary Template to create a pivot table for a simplified display of order details. You can learn more about this report in the Settlement Summary Report documentation.

FAQs

1. Where can I find the "All Statements" report in Seller Central?

To access the All Statements report, navigate to the Payments tab at the top of Seller Central, select Payments from the drop-down menu, and then click on the "All Statements" tab to view all available statements.

2. How can I specify the time period for the All Statements report?

You can use the Select Date Range filter on the All Statements page to specify the time period for which you want to generate the report. After selecting the desired date range, click "Search" to filter the results.

3. What formats are available for downloading the All Statements report?

There are three download options available:

- Download XML: Provides an XML file that can be opened in programs like Microsoft Excel.
- Download Flat File: Downloads the data as a tab-delimited text file.
- Download Flat File V2: Specifically for Fulfillment by Amazon (FBA) sellers, this file can be used with the Settlement Summary Template to create a pivot table for simplified order details.

4. What is the difference between the "Download Flat File" and "Download Flat File V2"?

- Download Flat File: This is a basic, tab-delimited text file containing all statement data.
- Download Flat File V2: This version is tailored for FBA sellers and can be used with the Settlement Summary Template for easier analysis of order details via a pivot table.

5. How do I search for a specific settlement period in the All Statements report?

After navigating to the All Statements tab, use the Select Date Range filter to choose the time period, and then click "Search". The results will show the available settlement periods. Locate the desired settlement period and download the report from the Actions column.

6. What should I do if I need help understanding the "Download Flat File V2" format?

If you need guidance on using the Download Flat File V2, it is recommended to refer to the Settlement Summary Report documentation, which explains how to use this file with the Settlement Summary Template to create a pivot table for better data visualization.

7. Can I download multiple reports at once?

No, you need to download each report individually by selecting the Download button next to the specific settlement period you're interested in.

8. What does the "Actions" column in the All Statements tab show?

The Actions column displays the available download options for each settlement period. From here, you can choose to download the report in one of the three available formats (XML, Flat File, or Flat File V2).

9. Are there any limitations on the number of settlement periods I can view or download?

There is no specific limitation on the number of settlement periods you can view in the All Statements report, but you must select and download them individually.

10. How do I know if the download was successful?

Once you click the Download button for a selected settlement period, the file will begin downloading to your computer. You can check your browser's download section to verify that the file has downloaded successfully.
