# Logistics SOP: Download an Amazon Fulfilled Shipments Report

## **Download an Amazon Fulfilled Shipments Report**

**Who is this for :**For Amazon sellers who are responsible for order fulfillment, logistics, and customer service within an organization using Amazon's fulfillment services.

**Objective :**To outline the process for downloading an Amazon Fulfilled Shipments Report. This report provides comprehensive information on shipments that have been fulfilled by Amazon, including order details, shipping information, and tracking numbers. It enables stakeholders to effectively track and manage their shipped orders, ensuring accurate and timely delivery to customers.

**To generate an Amazon Fulfilled Shipments Report, follow these steps:**

1. Go to the "Reports" tab in Seller Central and select "Fulfillment" from the dropdown menu.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-14%20at%202.01.03%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/VNpimage.png)

1. Under the "Sales" header on the left side of the page, click on "Amazon Fulfilled Shipments".

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-14%20at%202.01.10%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/RFFimage.png)

1. On the next page, select the event date range by choosing from the available options in the dropdown menu or by selecting "Exact Dates" and entering the "From" and "To" dates from the calendar icon.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-14%20at%202.01.17%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/I1Vimage.png)

1. Click on the "Request .csv Download" button to download the report in Excel format, or click on the "Request .txt Download" button to download it in text format.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-14%20at%202.01.23%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/RjOimage.png)

1. Check the "Report Status" column in the table below for the status of your report, which will initially show as "In Progress".

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-14%20at%202.01.29%E2%80%AFAM(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/zvWimage.png)

1. Once the report is ready for download, the status will change to "Download".

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-14%20at%202.01.35%E2%80%AFAM(2).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/9ioimage.png)

1. Click on the "Download" button in the "Report Status" column to download your report.

FAQs

1. How long does it take for an Amazon Fulfilled Shipments Report to be generated?

Most reports are generated within a few minutes, but larger reports covering longer timeframes may take up to 15 minutes. If the report is still "In Progress" after 30 minutes, refresh the page or try again later.

2. What information is included in the Amazon Fulfilled Shipments Report?

This report contains:

- Order ID
- ASIN and SKU
- Shipment ID
- Carrier name and tracking number
- Ship date and estimated delivery date
- Destination country
- Fulfillment center ID
  This data helps track orders, reconcile shipments, and analyze fulfillment performance.

3. What should I do if my report fails to generate?

If the report does not generate, try the following:

- Refresh the page and check if the request is still pending.
- Reduce the date range and try again.
- Ensure there are fulfilled orders within the selected timeframe.
- If the issue persists, contact Amazon Seller Support.

4. Can I automate report downloads instead of manually generating them?

Yes, you can use Amazon’s Selling Partner API (SP-API) to automate report generation. Alternatively, third-party tools like inventory management software may integrate with Amazon reports for automatic retrieval.

5. How far back can I pull an Amazon Fulfilled Shipments Report?

Amazon allows you to pull reports for up to 90 days at a time. If you need older data, generate multiple reports with adjusted date ranges or use Business Reports under Reports > Business Reports.

6. What format should I choose for downloading the report (.csv or .txt)?

- .csv (Excel Format): Recommended for easy viewing and filtering in spreadsheet software like Excel or Google Sheets.
- .txt (Text Format): Useful for bulk processing or integrating data with other systems.

7. What should I do if the "Download" button does not appear?

- Confirm the report status is marked as "Ready" instead of "In Progress".
- Refresh the page and check again.
- Try generating the report again with a smaller date range.

8. Can I download reports for multiple marketplaces at once?

No, Amazon Fulfilled Shipments Reports are specific to each marketplace. If you sell in multiple regions (e.g., US, UK, CA), you must download separate reports for each Seller Central account.

9. Where can I find tracking numbers for shipments in the report?

Tracking numbers are listed in the Carrier Name and Tracking Number columns. You can use this information to track shipments directly with the respective carrier.

10. Is there a way to schedule these reports for regular downloads?

No, Amazon does not currently offer a built-in scheduling option for Fulfilled Shipments Reports. However, if you need reports regularly, consider using the Amazon Selling Partner API (SP-API) or third-party tools that automate data retrieval.
