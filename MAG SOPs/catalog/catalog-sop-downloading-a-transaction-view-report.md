# Catalog SOP: Downloading a Transaction View Report

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-downloading-a-transaction-view-report

## Downloading a Transaction View Report

**Who is this for :** For sellers using Seller Central who need to download a Transaction View Report. It provides step-by-step instructions on accessing and customizing the report, as well as downloading it to review their account transactions within a statement period.

**Objective :** To guide sellers in downloading a Transaction View Report in Seller Central. The report allows sellers to view and analyze their account transactions, including orders, refunds, and Amazon-initiated charges or credits.

The Transaction View page shows account transactions within a statement period, up to the prior day's close. Transactions can include orders, refunds, and charges or credits initiated by Amazon.

You can adjust the report settings by filtering the view based on transaction types. Here are the options:

- All transaction types:

  Displays all transactions (default setting)
- Order Payment:

  Shows buyer payments and seller fees
- Refund:

  Displays order refunds
- Chargeback Refund:

  Refunds initiated by buyers through their credit card company
- A-to-z Guarantee Refund:

  Refunds related to A-to-z Guarantee
- Service Fees:

  Fees for services like FBA storage
- Shipping services purchased through Amazon:

  Includes shipping service purchases, refunds, and carrier adjustments
- Paid to Amazon:

  Amount paid to Amazon when sales for a statement period are less than fees and charges
- Other:

  Service-related fees and other Amazon-initiated credits and charges

**To download a "Transaction View" report, follow these steps:**

1. Go to the "Payments" tab on the top menu of Seller Central and select "Payments" from the drop-down menu.![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1716134182305.png)

FAQs

1. Where can I find the Transaction View report in Seller Central?

To access the Transaction View report, navigate to the Payments tab at the top of Seller Central, then select Payments from the drop-down menu. Afterward, click the Transaction View tab to proceed.

2. What information does the Transaction View report include?

The Transaction View report displays account transactions within a statement period, including:

- Orders: Buyer payments and seller fees.
- Refunds: Order refunds, A-to-z Guarantee refunds, chargeback refunds, and Amazon-initiated refunds.
- Charges/Credits: Amazon service fees, shipping service purchases, and other Amazon-initiated charges or credits.

3. Can I filter the Transaction View report by transaction types?

Yes, you can filter the report by various transaction types, including:

- All transaction types (default setting)
- Order Payment
- Refund
- Chargeback Refund
- A-to-z Guarantee Refund
- Service Fees
- Shipping services purchased through Amazon
- Paid to Amazon
- Other (service-related fees and other Amazon-initiated credits and charges)

4. How do I customize the Transaction View report?

To customize the report, you can use the following options:

- Account Type
- Transaction Type
- Transaction Status
- Dates
  After setting your filters, click the Update button to apply the changes.

5. Is there a limit on the number of transactions I can view in the Transaction View report?

Yes, the report is limited to displaying a maximum of 600 transactions. If you need to view more transactions or additional columns, you should refer to other reports, such as Disbursements or Date Range Reports.

6. How do I download the Transaction View report after applying the filters?

Once you have customized the report and clicked Update to apply your filters, you can click the Download button to generate a report containing the displayed transactions.

7. How far back in time can I view transactions in the Transaction View report?

You can view transactions from the last 24 months. Any transactions older than this will not be available for review in the Transaction View report.

8. What should I do if I need to see more than 600 transactions or additional data?

If you need a complete list of transactions or additional columns not displayed in the Transaction View table, you should utilize the Payment reports available on the Disbursements or Date Range Reports tabs for more detailed data.

9. Is the Transaction View report updated in real-time?

No, the Transaction View report reflects transactions up to the prior day's close. This means that transactions after the day's close may not appear until the following day.

10. Can I view a specific order using the Transaction View report?

Yes, you can search for a specific order within the report by entering the order details in the search bar, or by filtering based on transaction types and dates.

11. How do I know if my Transaction View report was successfully generated?

After clicking the Download button, the report will be generated. You can check the Report Status to ensure that the report is processed correctly and ready for download.
