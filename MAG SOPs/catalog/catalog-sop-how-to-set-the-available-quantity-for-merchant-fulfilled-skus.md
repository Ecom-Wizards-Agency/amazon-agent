# Catalog SOP: How to Set the Available Quantity for Merchant Fulfilled SKUs

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-how-to-set-the-available-quantity-for-merchant-fulfilled-skus

## **How to Set the Available Quantity for Merchant-Fulfilled SKUs**

**Who is this for?** Sellers or account representatives who are responsible for managing the Amazon catalog.

**Objective:** To ensure accuracy in inventory levels and the fulfillment of customer orders in a timely manner.

**Instructions:**

1. Go to the "Inventory" on the top menu of Seller Central and select "Manage Inventory" from the options.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/QDximage.png)

2. Find your new Merchant Fulfilled SKU on the page.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-U9C9ZT7S.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/rCgimage.png)

3. Enter the actual quantity available in the "**Available**" field and click the "**Save all**" button located at the top right side of the table to save your changes.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-GR9UYYYH.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Qvnimage.png)

Once you have established the available quantity, customers can place orders on SKUs with available units for sale at any time.

FAQs

**1. If I sell on multiple platforms, will my Amazon inventory update automatically?**

- No, unless you use an inventory management tool that syncs across platforms. If you sell on eBay, Shopify, or Walmart, you should manually adjust inventory or use software like Sellbrite, Linnworks, or InventoryLab.

**2. How do I set inventory for pre-order or made-to-order items?**

- For products not immediately available:

  - Use "Release Date" settings in Seller Central for pre-orders.
  - Set longer handling times under Manage Inventory > Edit SKU > Handling Time.
  - 3. What happens if I set my available quantity to 0?
  - Your SKU remains active but marked as "Out of Stock", meaning customers cannot purchase it.
  - If you want to pause sales temporarily, use "Close Listing" instead of setting inventory to 0.

**4. How can I prevent overselling if inventory updates are delayed?**

- Set a buffer stock (e.g., if you have 10 units, list only 8).
- Enable "Restock Alerts" in Seller Central.
- If selling across multiple platforms, use an automated inventory management system.

**5. Can I update inventory levels for multiple SKUs at once?**

- Yes. Instead of editing SKUs one by one:

  - Go to Seller Central > Inventory > Inventory Loader File.
  - Download the template, update available quantities, and re-upload the file.

**6. Can I use a flat file to update available quantity?**

- Yes. Use "Inventory Loader" or "Price & Quantity" flat file uploads in Seller Central. This is the best option for large catalogs.

**7. Will Amazon notify me when inventory is low?**

- Yes, you can set low-stock alerts under Settings > Notifications > Inventory Alerts to receive email reminders.
