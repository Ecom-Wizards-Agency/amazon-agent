# Vendor Central SOP: How to Download CLR in Vendor Central

Source: https://sop.myamazonguy.com/books/catalog/page/vendor-central-sop-how-to-download-clr-in-vendor-central

## **How to Download CLR in Vendor Central**

**Who is this for :** Category Listing Report can be useful for vendors who want to get a comprehensive overview of their products' categorization on Amazon.

**Objective :** This SOP (standard operating procedure) will help you to download the CLR in Vendor Central for company use. This report can help vendors identify any discrepancies in their product categories and ensure that their products are appearing in the correct categories for optimal visibility and sales.

**Instructions:**

1. Access the account by going to [Vendor Central](https://vendorcentral.amazon.com/).

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/catalog-sop-how-to-download-clr-in-vendor-central-image1.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/h3mimage.png)

2. Once logged in make sure to verify that you are on the correct account, if not click the account name at the left top then click See all to see the complete list of accounts.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202024-08-06%20at%2012.36.59%E2%80%AFAM(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/1b0image.png)

3. Choose the account that you need to work on.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202024-08-06%20at%2012.38.30%E2%80%AFAM(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Lagimage.png)

4. Hover the mouse over Items at the upper left corner then from the dropdown options select Catalog.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/MACimage.png)

5. On the Catalog page, scroll down to the bottom, look for “Results per page:”, click on it, and select the highest number available.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/catalog-sop-how-to-download-clr-in-vendor-central-image5(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/DQ9image.png)

6. Scroll back up to the top and tick the upper box to select all the listings. Verify the total number of selected listings to download After selecting all, click on the Download spreadsheet button (it will take a moment before the file gets ready to be downloaded, depending on the number of listings).

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/catalog-sop-how-to-download-clr-in-vendor-central-image6(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/A4limage.png)

7. You will be redirected to the , find your spreadsheet in “Spreadsheet download history” (check for the most recent file as well as the total of listings number) and download the generated spreadsheet.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/catalog-sop-how-to-download-clr-in-vendor-central-image7.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/2Tbimage.png)

8. The downloaded file will serve as Category Listing Report.

FAQs

**1. Do all vendors have access to the Category Listing Report (CLR)?**

- No, CLR is not always enabled by default for all vendors. If you don’t see the option to download it, you may need to request access through Amazon Vendor Support.

**2. How do I confirm that I’m working on the correct marketplace?**

- Before downloading the CLR, check the top-right corner of Vendor Central.
- Make sure the correct country marketplace (e.g., US, UK, CA) is selected to avoid pulling incorrect data.

**3. What should I do if the "Download Spreadsheet" button is missing?**

- Ensure you have admin or catalog management permissions in Vendor Central.
- Try using a different browser or clearing cache.
- If the option is still unavailable, open a case with Amazon Vendor Support to request CLR access.

**4. What key information is included in the Category Listing Report?**

- The CLR contains critical product categorization details, such as:

  - ASIN & SKU: Identifies your product.
  - Product Type & Item Type: Determines how Amazon categorizes the item.
  - Browse Node ID: Indicates where the product appears in Amazon’s category structure.

**5. My report has thousands of listings and won’t download. What can I do?**

- If you have a large catalog, try these steps:

  - Filter your listings by category and download smaller sections instead of the entire catalog.
  - Check your internet speed and avoid downloading during peak hours.

**6. How can I confirm if category updates I made have been applied?**

- Wait 24-48 hours after updating your product categories.
- Re-download the CLR and check if the Product Type, Item Type, or Browse Node has changed.
- If changes are not reflected, submit a flat file update again or contact Amazon Support.

**7. Are there alternative ways to download product category data if CLR is unavailable?**

- Yes. If CLR is not accessible, try:

  - Listing Quality Report (provides category and listing improvement suggestions).
  - Bulk Listing Report (offers detailed ASIN data for catalog management).
