# Logistics SOP: Setting up Return Settings for Merchant Fulfilled SKUs

Source: https://sop.myamazonguy.com/books/catalog/page/logistics-sop-setting-up-return-settings-for-merchant-fulfilled-skus

## **Setting up Return Settings for Merchant Fulfilled SKUs**

**Who is this for :** For Amazon sellers who are involved in managing merchant-fulfilled SKUs, including inventory managers, warehouse managers, and customer service representatives.

**Objective:** The objective of this process is to establish a clear and consistent system for setting up return settings for merchant-fulfilled SKUs.

**Instructions**

1. Click on the “Settings” tab located on the top right-hand side of your Seller Central Dashboard and select “Return Settings” from the drop-down menu.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-15%20at%201.48.05%E2%80%AFAM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/lxcimage.png)
2. On the next page, under the “General Settings” tab, choose your preferred option for receiving return request emails with links to authorize, close or reply
3. Select your “Automated Return Rules” according to your preferences.
4. Choose whether you want to handle the return mailing label yourself or have Amazon handle it.
5. Choose your preferred “Shipping Carrier” from the drop-down menu.
6. Decide whether you or Amazon will supply the “Return Merchandise Authorization Number”.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-15%20at%201.48.12%E2%80%AFAM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/XByimage.png)
7. You can write a return instruction by clicking the "Click here to write instructions" link.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-15%20at%201.48.20%E2%80%AFAM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/iEHimage.png)
8. Type your “Return Instructions” email response in the text box available below and then "Submit".

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-15%20at%201.48.28%E2%80%AFAM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/RWBimage.png)
9. Click the “Save Settings” button located at the bottom of the page to save your changes
10. Next, go to the “Manage Return Address” settings and enter your address details for the returns.
11. Finally, click the “Save Settings” button at the bottom of the page to save your return address settings

FAQs

1. What happens if I don’t set up return settings for merchant-fulfilled SKUs?

If you don’t configure return settings, Amazon will apply default return policies based on their marketplace rules. This could lead to unauthorized returns, delays in processing, or customer dissatisfaction due to unclear return procedures.

2. Can I set different return rules for different SKUs?

Yes, Amazon allows sellers to create customized return rules based on product category, condition, or price. You can configure this under “Automated Return Rules” in your Return Settings.

3. What is the difference between Amazon-handled and seller-handled return mailing labels?

- Amazon-handled labels: Amazon generates prepaid return labels based on their return policies.
- Seller-handled labels: You must provide the return label and manage the return process manually.

4. Can I require customers to contact me before returning an item?

In most marketplaces, Amazon automatically approves returns that fall within their policy guidelines. However, you can set up Return Instructions to request customers to contact you first, but compliance with Amazon’s return policies is required.

5. What if my return address is outside the country of sale?

If your return address is outside the country where you sell, Amazon may require you to:

- Provide a local return address.
- Offer a full refund without requiring a return if a local return isn’t possible.

6. How do I handle returns for items that are not eligible for returns?

For non-returnable items, ensure:

- The product is marked as non-returnable in your listing.
- Your Return Instructions clearly state why the item is non-returnable.
- Customer service is prepared to handle refund or exchange requests appropriately.

7. Can I charge a restocking fee for returns?

Yes, sellers can charge a restocking fee for certain returned items, but it must comply with Amazon’s Restocking Fee Policy. This is usually applicable to opened or used items.

8. How do I update my return address?

You can update your return address under “Manage Return Address” in Return Settings. Ensure the address is accurate, as Amazon requires a valid return location for merchant-fulfilled orders.

9. What should I include in my return instructions?

Your return instructions should:

- Clearly explain how customers should process returns.
- Provide any additional steps (e.g., contacting support before returning).
- Mention any product-specific return policies, if applicable.

10. How do I confirm that my return settings have been saved?

After configuring your settings, click “Save Settings” at the bottom of the page. Then, revisit the Return Settings section to confirm that your preferences have been applied.
