# Catalog SOP: Updating Product Listing Images

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-updating-product-listing-images

## **Update Product Listing Images**

**Who is this for :** Individuals who want to update product listing images.

**Objective:** The purpose of this objective is to have a step-by-step guide on how to update product listing images manually and through inventory file upload. You can update the images in two ways. One would be through each listing within the inventory page. You would manually need to edit each listing, delete, move the existing image, or upload a new one. The second method would be using upload files.

**Add an image to an existing listing (Manual):**

1. From your dashboard, go to the Inventory drop-down menu, and select **Manage Inventory**.
2. Click the **Edit** drop-down for each product on the far right, and select**Manage Images.**
3. Go to the **Images** tab and scroll down to the section titled **Your image recommendations.**
4. Click on **Upload** for any of the empty image slots. The **MAIN** image is the top left image slot in this section of the page.
5. Select an image from your computer and click Open.
6. Click Save at the bottom right of the page to add your image to the listing.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%202.08.23%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/iuCimage.png)

**Add an existing image to a flat file:**

1. From your dashboard, go to the Inventory drop-down menu, and select **Manage Inventory.**

2. Click the **Edit** drop-down for each product on the far right, and select **Manage Images**.

3. Hover your mouse over the image and right-click **Copy Image Address (Mac User)** or **Open the image in a new tab (If you’re using Windows).**

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%202.08.32%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/vupimage.png)

4. It will generate a direct link that you can paste into your flat file.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%202.08.39%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/vzWimage.png)

5. Open your flat file and paste the Direct Link into the images URL column and upload your file.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%202.08.47%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/0nSimage.png)

**NOTE**: If the images are not reflecting after you tried manually adding them and uploading a flat-file - Open a case to Amazon. Remember that if Amazon rejects your request, always continue to insist on them.

FAQs

**1. What are the two methods for updating product listing images?**

- You can update images using two methods:

  - Manual Update: Edit each product listing through the "Manage Inventory" section.
  - Flat File Upload: Add image URLs to the appropriate column in the inventory file and upload the updated file.

**2. How do I set the main image for a product listing?**

- To set the main image:

  - Go to Manage Inventory and select Edit > Manage Images.
  - In the Images tab, upload the desired image in the top-left slot labeled as the MAIN image.
  - Click Save to confirm the update.

**3. What is the correct way to add images via a flat file?**

- To add images using a flat file:

  - Obtain the image's direct URL by right-clicking the image (select "Copy Image Address" for Mac or "Open image in a new tab" for Windows).
  - Paste the URL into the images URL column in your flat file.
  - Upload the updated flat file via Add Products via Upload in Seller Central.

**4. Why are my updated images not showing on the product detail page?**

- This issue can occur due to Amazon’s review process or a delay in updating the detail page. If the images do not reflect after a manual update and flat file upload, open a case with Amazon Seller Support using the provided case template.

**5. What should I include in a case to Amazon for image updates?**

- When contacting Amazon:

  - Provide the ASIN of the product.
  - Mention that you’ve already tried manual updates and flat file uploads.
  - Include a Batch ID if available.

**6. How can I verify that the uploaded images meet Amazon's requirements?**

- Ensure that your images:

  - Meet Amazon's image guidelines (e.g., minimum 1000px on the longest side, white background for the main image).
  - Are in an accepted format such as JPEG, PNG, or TIFF.
  - Do not contain logos, watermarks, or promotional text.

**7. What should I do if Amazon rejects my image update request?**

- If Amazon rejects your request:

  - Review their feedback to identify any issues with the images.
  - Make the necessary adjustments to meet their requirements.
  - Reopen the case and insist on pushing the updates if you believe the images comply.

**8. Can I upload images for multiple listings simultaneously using a flat file?**

- Yes, a flat file allows you to update images for multiple listings at once. Ensure each image URL is correctly added to the respective product's images URL column before uploading.

**9. How long does it take for image updates to reflect on the detail page?**

- Images usually take 15–30 minutes to update after a successful upload. However, in some cases, it may take up to 24 hours. If the update doesn’t appear within this timeframe, open a case with Amazon.

**10. Can I update the images without a direct image URL?**

- No, flat file uploads require a direct image URL. For manual updates, you can upload images directly from your computer via the Manage Images option.

**11. What is a Batch ID, and how do I include it in my case?**

- A Batch ID is a unique identifier provided after uploading a flat file. It helps Amazon track your request. Include it in your case template to assist Seller Support in locating your file upload.

**12. What should I do if I encounter errors during a flat file upload?**

- If errors occur:

  - Review the error report provided by Amazon.
  - Correct any issues in the flat file (e.g., missing URLs, incorrect ASINs).
  - Re-upload the corrected file.

**13. Is there a limit to the number of images I can upload for a listing?**

- Yes, Amazon typically allows up to 9 images per listing, including the main image. Ensure all image slots are utilized effectively.

**14. Can I update A+ Content images using these methods?**

- No, these methods apply only to standard product listing images. To update A+ Content images, use the A+ Content Manager in Seller Central.

**15. Can I replace an existing image, or do I need to delete it first?**

- You can upload a new image to replace an existing one. There is no need to delete the old image beforehand; the new image will overwrite the previous one when uploaded.
