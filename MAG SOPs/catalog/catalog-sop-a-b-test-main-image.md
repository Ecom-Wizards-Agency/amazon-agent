# Catalog SOP: A/B Test (Main Image)

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-ab-test-main-image

## **A/B Test (Main Image)**

**Who is this for:** Individuals who wished to conduct A/B split testing within seller central.

**Objective:** The purpose of this document is to provide a step-by-step guide on how to create A/B split testing to submit alternate content for your products to collect insights such as traffic and conversion. Split testing or AB testing, on Amazon, is an excellent way of optimizing your listings through experiments with product content, images, and pricing. Four crucial elements of the listing affect conversion rate: Image, Title, Product Features, Description, and Price.

**Seller Central A/B Testing**

Access the Amazon Seller Central page by clicking the URL: https://sellercentral.amazon.com/

- Go to Brands, then select Manage Experiments.

- On the experiments dashboard, click Create a New Experiment.

  - You can run an experiment for A+ Content, Bullet Points, Product Images, Product Title A+ Brand Story and Multi-Attribute.

**Main Image A/B Testing**

- Search the ASIN for A/B testing. It will display if the ASIN is Eligible for the experiment, then click Select Reference ASIN.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-AB-Test-Main%20Image-3(1).png)

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/xthimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/xthimage.png)

- If the ASIN is part of a variation, you will need to select first the whole parentage as the reference ASIN.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-AB-Test-Main%20Image-4(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/2iqimage.png)

- Enter a descriptive test name, duration, and desired start date. You may also enter an optional hypothesis around what you wish to learn from the test.

  - Experiment Name
  - Hypothesis - Brief description of the experiment and the desired outcome.
  - Duration - Minimum of 4 weeks to a maximum of 10 weeks.

    - Start Date will be automatically set 1 week after you created the experiment, but you can still set it at a later date.
    - End Date will be automatically set depending on the duration of the experiment.
    - You can tick To significance which automatically ends the experiment when there is enough time data to declare winner.
    - There’s an option now to Automatically publish the winning version.

      - Note : There is no definite timeline for the A/B test but if there is already a **“strong”** evidence we can go ahead and stop the A/B testing also if the client is satisfied with the results and instructs to go ahead with the winning result.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-AB-Test-Main%20Image-5(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/1Iaimage.png)

- Uncheck all other Child ASINs that are not included in the experiment.

  - When you run an experiment on a variation family, you can submit an experiment for one or more of the child ASINs, but you always have to submit for the parent ASIN.
  - Upload the 2nd or B image you are putting up against the current main image.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-AB-Test-Main%20Image-6(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/HnKimage.png)

- Once all information has been entered, click Schedule Experiment. You will be routed back again to the experiments landing page.

FAQs

1. **What makes an ASIN eligible for A/B testing?**

- An ASIN is eligible for A/B testing if it meets Amazon’s criteria, including product category, sales volume, and other requirements. If an ASIN is not eligible, you may need to adjust your product or experiment parameters or contact Amazon support for more information.

1. **Are there any specific requirements for the images used in the A/B test?**

- Yes, the images used for A/B testing should meet Amazon’s image quality standards, including a high resolution (at least 1000 pixels on the longest side) and proper product representation. The A/B test requires you to upload an alternate “B” image, which will be compared to the current “A” image to analyze the performance difference.

1. **What is a hypothesis in A/B testing, and how do I create one?**

- A hypothesis in A/B testing is a brief description of what you expect to learn from the test. For example, you may hypothesize that changing the main image to a lifestyle shot will increase conversions. A good hypothesis should be clear, measurable, and specific, guiding your expectations for the outcome.

1. **How do I track and interpret the insights collected during an A/B test?**

- Insights from the A/B test include traffic metrics (e.g., clicks) and conversion metrics (e.g., purchases). Amazon provides these insights through the Seller Central dashboard after the test concludes. Use this data to evaluate which version of your content (e.g., main image) drives higher performance.

1. **What does "To Significance" mean in A/B testing?**

- The “To Significance” option automatically ends the experiment once enough data has been collected to determine a statistically significant winner. This ensures that the results are reliable and not due to random chance. If the data does not show a significant difference, the test may continue until enough data is gathered.

1. **What happens after an A/B test ends?**

- After the test ends, you can review the results. If the experiment was set to automatically publish the winning version, the better-performing content will be applied to your product listing. You may also receive a report with key performance metrics that show which variant performed better in terms of traffic and conversions.

1. **How do results from A/B testing apply to child ASINs in a variation family?**

- When running an A/B test for a parent ASIN, you must include the parent ASIN in the experiment. You can select one or more child ASINs for testing, but the results will apply to the parent ASIN. If you wish to apply the winning version to all child ASINs, you must do so manually.

1. **How do I interpret the results of my A/B test?**

- To interpret your A/B test results, compare the traffic and conversion metrics of the two variants. Look for statistical significance to determine if one variant outperforms the other. Use these insights to make informed decisions on which product content (e.g., image) should be kept.

1. **Are there any limitations on A/B testing on Amazon?**

- Yes, there are some limitations. A/B testing is generally available for eligible ASINs in specific product categories, and you may only run a certain number of experiments at the same time. Additionally, you may only test specific content elements, such as the main image, title, and description.
