# Catalog SOP: Schedule Brand Store Versions

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-schedule-brand-store-versions

## **Schedule Brand Store Versions**

**Who is this for :** Individuals who are uploading an occasion-themed brand store banner.

**Objective :** To minimize re-uploading different occasion-themed brand store banners.

Amazon Brand Stores allow you to showcase your brand and products in a multi-page, immersive shopping experience. No matter the size of your brand, the Amazon Brand Store gives you an immersive place to introduce audiences to your story, mission, and products.

You can create multiple versions of your Store and schedule them to run during specific periods of time. This is very useful during holidays, simplifies the process, and saves extra work.

**Creating Amazon Store Versions**

- Go to**Stores**, then **Manage Stores**.![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/schedule-brand-store-versions-image-yx1e7cvy.jpg)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Uqcimage.png)

- Click **Edit Store** in line with the brand you are working on.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Image%202(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/gzAimage.png)

- Click the **gear button** of the version you want to duplicate.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/2023-03-31_18-50-12%20(1).jpg)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Mq3image.png)

- Click **Copy version** button

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/schedule-brand-store-versions-image-81dopm7t(1).jpg)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/ubrimage.png)

- **Fill in** the Version name box.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/schedule-brand-store-versions-image-dx5qw5tf(1).jpg)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/tnnimage.png)

- Click **Edit now** to start customizing the new brand store version.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/schedule-brand-store-versions-image-qfh3m0nn(1).jpg)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/3Fiimage.png)

- You can customize the hero image (banner) or other stuff that suits the scheduled time.
- To replace the banner image, just click the image itself. A right sidebar will pop out, then click Upload New to upload the new banner image.

  - The banner image must be at least**3,000 x 600 pixels** in size.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/schedule-brand-store-versions-image-ijo5ci46(3).jpg)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/oOOimage.png)

- Once the new hero image is uploaded, click **Done**.
- Click **Submit** for Publishing button.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/schedule-brand-store-versions-image-qjdh91xl(2).jpg)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/lOCimage.png)

- In the pop-up window, select **Request publishing dates**.

  - Fill in the *Start and End dates* you want the version of the brand store to go live.
  - The bottom part will display other versions' start and end dates.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/2023-03-31_18-55-04%20(1).jpg)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/45Bimage.png)

- Click **Submit**.

**NOTE:**

- If there is no other version besides the original and duplicate versions, the original version will go live right after the end date of the duplicate version.
- If there are other versions you want to go live after the duplicate version's due date:

  - Wait for the duplicate version to go live first.
  - Edit the other version's start date (the date right after the end date of the duplicate version)
  - Submit the other version.
- Only 1 version can have no end date.

FAQs

1. **How do I handle scheduling multiple Brand Store versions with overlapping dates?**

- To avoid conflicts, ensure that each version has a unique start and end date. If two versions have overlapping dates, the one with the earliest start date will take precedence. You may need to adjust the start and end dates to ensure there is no overlap or conflict.

1. **What happens after I click "Submit for Publishing"?**

- Once you submit the brand store version for publishing, Amazon will review the changes. If everything is in order, the version will be published according to the start and end dates you specified. Note that there may be a slight delay in publishing, depending on the review process.

1. **What should I do if my new banner image doesn’t meet the required size or format?**

- If your banner image doesn’t meet Amazon’s size or format requirements (at least 3,000 x 600 pixels), you’ll be prompted to upload a valid image. If you encounter any issues, double-check the image size and resolution, or consider using a different image that meets the criteria.

1. **Do I need to update other content for each version of my Brand Store?**

- No, you don’t need to update other content (such as product listings or text) for each version unless you want to create a completely unique experience for that specific time period. The main focus for versioning is usually the banner image, but you can customize other elements if needed.

1. **What happens if I don’t set an end date for a version?**

- Only one version can have no end date. If a version is set with no end date, it will remain live indefinitely until you manually update or remove it. This means that no other versions will be able to go live until you specify an end date for the version without one.

1. **How can I revert changes or cancel a scheduled version?**

- If you need to revert changes or cancel a scheduled version, you can go back to the "Manage Stores" page and edit or delete the version. If you have already submitted it for publishing, you can modify the start/end dates or make adjustments directly within the version settings.

1. **Are there other customization options besides the hero image?**

- Yes, while the primary focus is often on customizing the hero image (banner), you can also edit other elements of your brand store. These include text, product showcases, promotional tiles, and other content areas. The level of customization depends on your store layout and the features available.

1. **What should I do if I accidentally schedule two versions with conflicting start/end dates?**

- If you accidentally schedule two versions with conflicting dates, Amazon will prioritize the earliest version and adjust the schedule accordingly. If you want to avoid any issues, review the start and end dates carefully to ensure they don’t overlap. You may need to adjust one of the versions if necessary.
