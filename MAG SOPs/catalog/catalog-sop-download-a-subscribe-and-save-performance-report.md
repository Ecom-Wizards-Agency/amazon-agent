# Catalog SOP: Download a Subscribe and Save Performance Report

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-download-a-subscribe-and-save-performance-report

## **Download a Subscribe and Save Performance Report**

**Who is this for :** Intended for individuals who are responsible for monitoring and analyzing the performance of the Subscribe and Save program. This SOP involves assessing the effectiveness of the program and identifying opportunities for improvement.

**Objective :** To outline the process for downloading a Subscribe and Save Performance Report. This report provides valuable insights into the performance of the Subscribe and Save program, allowing stakeholders to analyze key metrics and make data-driven decisions.

1. From the Seller Central home page, click the 3-line bars located on the left side. Then, select "Reports" and choose "Fulfillment.

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/rXGimage.png)

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684346949099.png)
2. Under the "Sales" header on the left side of the page, click the "Subscribe & Save Performance" link.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684347454131.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Agcimage.png)
3. On the next page, select the desired "Event Date" by clicking on the calendar icon next to the text box.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684347521701.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/CCpimage.png)
4. Click the "Request .csv Download" button to download the report in Excel format, or the "Request .txt Download" button for a text format.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684347572810.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/S3Kimage.png)
5. In the table below, the "Report Status" column will indicate the status as "In Progress".

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/W6Nimage.png)

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684347631706.png)
6. Once the report is ready for download, the status will change to "Download".

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684347745766.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/4Ssimage.png)

  7. Click the "Download" button in the "Report Status" column to download the report.

FAQs

1. Where can I find the Subscribe and Save Performance Report?

You can access the report in Seller Central by clicking the 3-line menu on the left, selecting Reports > Fulfillment, and then navigating to Sales > Subscribe & Save Performance.

2. What key metrics are included in the Subscribe and Save Performance Report?

The report provides insights into subscription trends, retention rates, order frequency, revenue impact, and customer behavior related to the Subscribe and Save program.

3. Can I choose a custom date range for the report?

Yes, you can select a specific "Event Date" using the calendar icon to define your desired reporting period.

4. What file formats are available for downloading the report?

The report can be downloaded in Excel (.csv) format by clicking "Request .csv Download" or in text (.txt) format by selecting "Request .txt Download."

5. How can I track the progress of my report request?

After requesting the download, check the "Report Status" column in the table below. The status will first show "In Progress" and update to "Download" when the report is ready.

6. How long does it take for the report to be generated?

The processing time depends on data volume, but typically, the report will be available within a few minutes.

7. What should I do if my report download fails or does not appear?

If the report does not generate or download successfully, try refreshing the page and checking the "Report Status" again. If the issue persists, clear your browser cache or try a different browser. If the problem continues, contact Amazon Seller Support for assistance.
