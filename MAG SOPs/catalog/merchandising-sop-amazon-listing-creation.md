# Merchandising SOP: Amazon Listing Creation

Source: https://sop.myamazonguy.com/books/catalog/page/merchandising-sop-amazon-listing-creation

## **Amazon Listing Creation**

**Who is this for?:**For individuals who want to create new listings on their Amazon Seller Central account.

**Objective:**This document aims to provide a step-by-step process on how to create new product listings on Amazon.

**Instructions:**

##### **PRE-REQUISITE 1: GS1 Certification Verification**

**Important Reminder!** Amazon has been conducting listing takedowns when the brand is not the owner of the barcode, and/or when the barcode is not GS1 certified.

To verify if the brand owns the barcode, follow the steps below:

**Method 1: Quick Search**

1. Go to gs1.org.

2. Hover your mouse over **Services**> **Verified by GS1**

3. Enter the barcode and hit “**Search**”

5. You should find the owner's Information just as shown in the picture below:

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-IZJISKLU.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Qt7image.png)

**Method 2: Detailed Search**

1. Go to [GS1 Link](https://gepir.gs1.org/index.php/search-by-gtin).

2. Hover your mouse over **Services** and **Click Who owns this barcode?**> **Search by GTIN**

3. Enter the UPC on the search box

4. Click the dropdown and select “*Ownership*”

5. Click **Search**

6. The detailed ownership details will look like the image below:

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1680126486636.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/4Tvimage.png)

Next Steps:

a. To avoid having your listings taken down by Amazon, it is advisable to purchase barcodes from GS1 if you are not the registered owner of the barcodes in the GS1 database. b. You may proceed with the following step if there isn't any problem with the barcode ownership.

##### **PRE-REQUISITE 2: Amazon Barcodes Or Manufacturer Barcodes Settings**

1. Before creating any listing for FBA, it is essential to verify the Barcode Preferences.

2. Go to **Settings**> **Fulfillment by Amazon**and check the **FBA Product Barcode Preference** → Change this to ***Amazon barcode***if you want to use FNSKUs/Amazon barcodes. Using Amazon barcodes will generate an FNSKU starting with X00 while the manufacturer barcode displays the ASIN.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/ShJimage.png)

3. Go to **Settings**> **Fulfillment by Amazon > Optional Service** and check the following setup:

a. Default - Who Labels? → Change the settings to Amazon if you use FNSKUS. Change the settings to “**Merchant”**for Manufacturer Barcodes

Default - Who Preps? → You can leave this alone as this is customizable when setting up shipments, but you may update this if Amazon will do the prepping.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/9ptimage.png)

**Steps To Create The Flat File**

1. Download the correct flat file template. SOP can be found [here](https://sop.myamazonguy.com/books/catalog/page/merchandising-sop-download-a-category-specific-template).

2. Fill out the product type column. All values that Amazon requires you to input will have a red coloring. The required information varies depending on the category.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/chlimage.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1680126749761.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/k5Gimage.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1680126792363.png)

***Important Notes:***

*a. Seller SKUs cannot be changed once the listing has been created.*

*b. The Standard Price and Quantity cells are not highlighted in red but they are still required.*

*c. If the listing you’re creating is FBA,****you don’t have to fill out Quantity****. If it’s FBM, put the accurate number.*

*d.To add images, you can use the Amazon backend link, copy and paste the direct image URL into the respective flat file cell.*

3. On the “Update Delete” column, use **Update**.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-5L6ZTT07.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Ktwimage.png)

4. Under the Fulfillment Center ID, use **AMAZON_NA** (NA for North America, CA for Canada, MX for Mexico, etc) if it’s Fulfilled by Amazon and **DEFAULT** if it’s Fulfilled by Merchant. If it’s left blank, it will default to FBM.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-MS18MY3W.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/437image.png)

*Important Note: For any FBA listing, fill out the columns (1)****“batteries_required****”, (2)****“supplier_declared_dg_hz_regulation1****, (3)****pesticide marking****, and (4)****pesticide regulation status****under the Compliance section.*

5. Add “New” to the Item Condition column.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-MRCNCJKR.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/JJmimage.png)

6. Set the offer start date to the day you're ready to launch the product.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-94ZA8AU7.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/C6Vimage.png)

7. Save the flat file and upload it on Seller Central. Go to Inventory > Add Products via Upload > Upload your Inventory File. Refresh the page first before attaching the file and make sure you’re on the right Amazon store. Then click **Upload file**.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1680126904539.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/WnLimage.png)

8. Once the file has been uploaded and processed, it will generally take 15 minutes for Amazon to process it. It may also take up to 24 hours for all the info/images to be reflected in the new listing. Go to the Manage Your Inventory page and search the SKUs to verify if they have been successfully created.

***Important Note: Monitor the Upload Status. If it says “Complete Drafts”, download the Processing Report to see what caused the error and fix it accordingly. Getting errors in the processing report does not necessarily mean that the flat file was not processed. Verify on the Manage Inventory Page.***

FAQs

1. How do I determine which flat file template to use for my product?

Amazon provides different flat file templates based on product categories. To find the correct one:

- Go to Seller Central > Inventory > Add Products via Upload > Download an Inventory File.
- Select the category that matches your product.
- Download the template and review the required fields.

2. What should I do if my flat file upload fails or has errors?

If your upload encounters errors:

- Download the Processing Report under Upload Status.
- Review error messages and correct any missing or invalid fields.
- Common issues include missing required attributes, incorrect barcode formatting, or unsupported image URLs.
- Re-upload the corrected file.

3. What are Amazon's image requirements for product listings?

- The main image must have a pure white background (RGB 255,255,255).
- Images should be at least 1000 x 1000 pixels for zoom capability.
- No watermarks, logos, or additional text overlays are allowed.
- The product should take up at least 85% of the image frame.

4. How do I create a variation (parent-child) listing?

To create a variation listing:

- Use the flat file template for the specific category.
- Define a Parent SKU and set its Parentage column to "Parent".
- For child SKUs, enter the same Parent SKU, specify the variation theme (e.g., size, color), and enter values accordingly.
- Ensure all child SKUs have unique SKUs, barcodes, and prices.

5. Are there character limits for SKUs, product titles, and descriptions?

Yes, Amazon has limits:

- SKU: Up to 40 characters.
- Product Title: Typically 200 characters, but may vary by category.
- Bullet Points: Each bullet can be up to 500 characters.
- Product Description: Up to 2000 characters.

6. Can I change a listing after it has been created?

Yes, but some fields cannot be changed, such as SKU. To edit:

- Go to Manage Inventory and click Edit on the listing.
- If changes don’t reflect, try updating via flat file using "PartialUpdate" in the Update Delete column.

7. What should I do if GS1 verification shows a different owner than expected?

- Double-check the barcode entry in gs1.org.
- If the barcode is registered to another company, Amazon may reject the listing.
- You may need to purchase a GS1-certified barcode or contact Amazon Seller Support for verification issues.

8. If I change my barcode preferences, will it affect existing listings?

No, barcode preferences only apply to new listings. However, switching from Manufacturer Barcode to Amazon Barcode (FNSKU) requires relabeling products already in FBA warehouses.

9. How do I check if my listing has been successfully created and is active?

- Go to Manage Inventory and search for the SKU.
- Check for suppressed listings under Inventory > Fix Your Products.
- If the listing is inactive, check for missing attributes or category approval issues.

10. Do I need approval to create listings in restricted categories?

Yes, some categories require approval before listing products. To check:

- Go to Seller Central > Catalog > Add a Product and search for your item.
- If approval is required, click Request Approval and follow Amazon’s process.
