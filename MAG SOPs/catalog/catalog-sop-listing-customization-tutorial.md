## **Listing Customization Tutorial**

**Who is this for :** This is intended for the Account Associates who will be creating listing customizations.

**Objective :** To ensure that Account Associates have the necessary knowledge and tools to effectively create listing customizations.

**Instructions:**

1. Create your listing. Check this tutorial on how to do this: [SOP: Listing Creation](https://myamazonguy.document360.io/docs/popeplus-catalog-sop-amazon-listing-creation-1). If you already have one, you can skip this step.

2. Once you have created the skeleton listing, look for it on Inventory > Manage All Inventory.

FAQs

1. What is a listing customization?

A listing customization allows you to provide customers with options to personalize or select different variations of a product on your Amazon listing. This feature helps enhance the customer shopping experience and can increase sales.

2. Do I need to create a listing before customizing it?

Yes, you need to have an active listing with a purchasable offer before you can customize it. Make sure your inventory is available and the offer start, release, and launch dates are correctly set.

3. How can I find my listing to customize it?

Once you've created the listing, go to Inventory > Manage All Inventory. Look for your listing and click on the arrow to reveal the dropdown menu, where you will select Add/Edit Customization Information.

4. Can I customize a listing if the product is not yet available for purchase?

No, you must have available inventory, and the Offer Start Date, Release Date, and Launch Date should not be set in the future. The product must be listed as purchasable before customization can be applied.

5. How do I add customizations to my listing?

Click on Add Customization, then choose the appropriate customization template for your product. You can add up to 15 customizations for each listing. After adding the customizations, click Publish to save your changes.

6. How can customers view and use the customization options?

When the customization is set up correctly, a Customize Now button will appear on the product detail page (PDP). Customers will need to click this button before they can purchase the product with their customized options.

7. Can I edit the customization after publishing it?

Yes, you can go back to Manage All Inventory and edit the customization options as needed. Just be sure to save any changes before exiting.

8. Can I use any image for the customization section?

Yes, you can upload the main product image for the customization section. If the title is too long, trim it to fit into the label field properly.

9. How many customization options can I add to a listing?

You can add up to 15 customizations for each product listing.

10. What should I do if I don’t see the customization options after publishing?

If the customization doesn’t appear after publishing, ensure that the listing has been properly set with available inventory, and check that the Customize Now button is enabled. If the issue persists, review the customization template for accuracy.
