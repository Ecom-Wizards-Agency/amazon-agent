## ASIN IS NOT ELIGIBLE FOR A+ CONTENT

**Who is this for**: This is for all individuals and sellers that are getting a warning message with their ASIN in A+ Content. That warning message states that "You are unable to add content to this ASIN because our system does not recognize this ASIN as part of your brand"

**Objective**: This Standard Operating Procedure (SOP) will help you understand and fix this issue and make that ASIN eligible for A+ Content.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/sop_-asin-is-not-eligible-for-a-content-image-y08qncdg(2)(6).jpg)

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/f7fimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/f7fimage.png)

- **This message means that the ASIN is not properly mapped to the brand. This is usually caused by inaccurate listing data on the ASIN.**

**How to access the A+ content page (Check the ASIN):**

- Go to Advertising > A + content Manager> Start Creating A+ Content > Create Basic> Provide Content Name (Test)> Put the ASIN in the Bulk Upload Bar

Step 1

- Update the data by selecting Edit on that ASIN in your Manage Inventory and go to Vital Info and put the Brand name and all other required attributes.

**Note:** *if you do not see the Brand Name under Edit Listing anymore, it would mean that the account already transitioned to Amazon's new Edit listing interface. In this case, skip step 1 and proceed to step 2 onwards.*

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/sop_-asin-is-not-eligible-for-a-content-image-weop28ak(1)(1).png)

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Ko6image.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/Ko6image.png)

- Make sure that the brand name listed on the ASIN matches the brand name approved by Amazon's Brand Registry (case sensitive). These need to match 100%. Adding spaces or hyphens will lead to a mismatch.

**Step 2**

- Download CLR and upload a FULL UPDATE on the ASIN

**Important :**Do not create a case before trying this method

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/sop_-asin-is-not-eligible-for-a-content-image-sgubcwzn.png)

**NOTE:** **Double-check all the information before uploading the feed file**

- If the listing has more than 1 SKU - Make sure to update both of them

For example, if the listings have FBM and FBA.

- Product ID must be the ASIN, PT, and ITK must be the correct value based on the category of the listing.
- **REVIEW THE SALE PRICE IF THERE IS A SALE PRICE IN THE BACKEND SINCE THIS IS NOT CAPTURED IN CLR**

- Update the “Update Delete” column to “Update” and save the file.

- Wait until the inventory file is processed +15 mins and make sure it is uploaded successfully and keep the BATCH ID

FAQs

#### 1. What does the warning message "You are unable to add content to this ASIN because our system does not recognize this ASIN as part of your brand" mean?

This warning indicates that the ASIN is not properly mapped to your brand, which is usually due to inaccurate listing data on the ASIN. It typically happens when the brand name on the listing doesn’t match the one registered in Amazon's Brand Registry.

#### 2. How can I check if my ASIN is eligible for A+ content?

To check if your ASIN is eligible for A+ Content:

1. Go to Advertising > A+ Content Manager > Start Creating A+ Content.
2. Enter the ASIN in the Bulk Upload Bar and try creating content.

#### 3. How do I update the ASIN’s brand name to make it eligible for A+ Content?

To update the brand name:

1. Go to Manage Inventory in Seller Central.
2. Find the ASIN and click Edit on the listing.
3. Ensure the Brand Name is entered correctly in the Vital Info section.

  - The brand name must match exactly what is registered in Amazon's Brand Registry, including case sensitivity and spaces or hyphens.

If the brand name field is no longer available (due to Amazon's new Edit Listing interface), skip Step 1 and proceed to Step 2.

#### 4. What do I do if I can't find the Brand Name under the Edit Listing section?

If the Brand Name field is not available, it means your account has transitioned to Amazon's new Edit Listing interface. In this case, skip Step 1 and proceed with the following steps.

#### 5. What is the next step after updating the brand name?

Once the brand name is updated, download the Category Listing Report (CLR) and upload a Full Update for the ASIN:

1. Ensure all the data is correct (ASIN, Product ID, and PT/ITK values).
2. Update the “Update/Delete” column to “Update” and save the file.
3. Upload the feed file under Add Products via Upload in Seller Central.

#### 6. What if my listing has more than one SKU (e.g., FBM and FBA)?

If the listing has more than one SKU, make sure to update all SKUs. For example, if you have both FBM and FBA listings for the same product, update both.

#### 7. How do I check if the feed file was uploaded successfully?

After uploading the feed file, wait at least 15 minutes for the inventory file to process. Check the Batch ID to confirm the file has been successfully uploaded.

#### 8. How long does it take for the ASIN to become eligible for A+ Content after the upload?

It may take up to 24 hours for the ASIN to be eligible for A+ Content after the update is processed.

#### 9. What should I do if the ASIN is still not eligible for A+ Content after the update?

If the ASIN is still not eligible after 24 hours, contact Amazon Seller Support. Provide the following details in your case:

- ASIN: [Enter ASIN]
- Brand Name: [Enter brand name]
- Batch ID: [Enter batch ID]
- Attach a screenshot of the error message.

Here’s a sample message you can use:

"Amazon,

We are unable to add A+ content on our ASIN (Enter ASIN). It says it's not eligible.

It appears that our ASIN is not properly mapped to our brand name.

We uploaded a feed file with a full update but it hasn't fixed the issue.

Brand Name: [Enter brand name]

Batch ID: [Enter batch ID]

See attached screenshot."

#### 10. Can I skip the "Check File" step during the upload process?

Yes, you can skip the "Check File" step during the upload process if you’re confident that your feed file is correctly formatted.
