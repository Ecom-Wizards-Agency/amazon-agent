# Catalog SOP: Merging Variations

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-merging-variations

## Merging Variations

**Who is this for :** This is intended for sellers using Seller Central to manage their inventory and who need to merge variations and add variations from one Parent ASIN to another. It is useful for those who want to optimize their product listings and reduce clutter on their inventory pages.

**Objective :** To provide instructions on how to merge variations and add variations from one Parent ASIN to another manually or via a template.

To merge variations and add variations from one Parent ASIN to another, you can do so manually or via the Template.

**MANUAL METHOD**

- If you are doing this manually, go to "Manage All Inventory" from the "Inventory" tab Drop Down menu in Seller Central. Make sure that the ASINs you want to merge meet the following stipulations: the Brand name should be the same for all ASINs, and the ASINs should be in the same category. Then, delete the parent ASIN for the ASINs you wish to add to another existing Parent ASIN, making the Child ASINs stand-alone ASINs. It's advisable to delete the Parent ASIN with the least number of Child ASINs. Make sure that you only delete the Parent ASIN and not the Child ASINs as well by clicking on the little arrow alongside the Parent ASIN beforehand.
- Once these Child ASINs are stand-alone ASINs, click on the "Edit" button to the right of the Parent ASIN you wish to add these Child ASINs to.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-07%20at%201.24.10%E2%80%AFAM.png)

- Under the "Variations" tab, add the two new variations as you would if you were creating new variations. Copy all the information exactly from the now stand-alone Child ASINs to the new variations you are setting up under the Parent ASIN. You can also choose to rename the Parent ASIN title to make it more relevant to the newly configured variations. Once you are happy with your changes, click the "Save and Finish" button at the bottom of the page.

- To merge Variations via Template, you may use the Category Listings Report through :

  - Reports→Inventory Reports→Select Report Type→Category Listings Report→Request Report

- Download the CLR once ready and filter out the stand alone SKU/ASIN which you are to merge to the existing Parent.
- Make sure that you fill in all the same information under the "Variations" section as it stands for these newly formed stand-alone Child ASINs (SKU, etc.) and insert the new Parent ASIN they are being mapped to. Here you would select the "Partial Update" item from the Drop Down Menu provided in the "Update Delete" Field on the sheet for the variations, and you would only be adding the stand-alone ASINs from the deleted Parent ASIN to this other Parent ASIN. You wouldn't need to re-add all the Child ASINs already under this Parent ASIN. Make sure that you fill in all the required information and fill in the "Item Type Keyword" field to ensure you are using the same category as you did for the Parent ASIN you are mapping these Child ASINs to.

If the Parent ASIN is new, you would need to fill in all the information for the Parent ASIN as you would when creating a new Listing and select "Update" from the "Update Delete" Field. Save this file in .txt format before uploading it under the "Check and Upload Your Inventory File" tab on the "Add Products Via Upload" Page.

Note that while merging ASINs to the existing variation, you need to use the same variation theme as of the parent to the ASIN you are going to add manually. However, you can change the variation theme if you follow the steps to edit/create variations steps via file template.

FAQs

#### 1. What does it mean to merge variations?

Merging variations refers to the process of combining different Child ASINs under one Parent ASIN, either by manually adding them or using a template. This helps streamline product listings, reduce clutter, and enhance product visibility by consolidating related variations (such as size or color) into one parent listing.

#### 2. How can I merge variations manually?

To merge variations manually:

1. Go to "Manage All Inventory" in the "Inventory" tab in Seller Central.
2. Ensure the ASINs you want to merge meet the following conditions:

  - Same Brand Name.
  - Same Category.
3. Delete the Parent ASIN of the ASINs you wish to merge. Ensure you only delete the Parent ASIN, not the Child ASINs.
4. Make sure the Child ASINs become stand-alone ASINs.
5. Click "Edit" next to the Parent ASIN you want to add the Child ASINs to.
6. Under the "Variations" tab, add the new variations as you would for creating new variations.
7. Copy all relevant information from the stand-alone Child ASINs to the new variations.
8. Optionally, rename the Parent ASIN title for better relevance.
9. Click "Save and Finish" to finalize the changes.

#### 3. How do I merge variations using a template?

To merge variations via a template:

1. Download the Category Listings Report (CLR) from Reports > Inventory Reports in Seller Central.
2. Filter out the stand-alone ASINs you want to merge into the existing Parent ASIN.
3. In the Variations section, input the same information for these Child ASINs and insert the new Parent ASIN they are being mapped to.
4. Select "Partial Update" from the "Update/Delete" dropdown menu.
5. Do not re-add Child ASINs that are already listed under the Parent ASIN.
6. Ensure the "Item Type Keyword" field is filled out and corresponds to the Parent ASIN’s category.
7. If the Parent ASIN is new, fill out all the information for it and select "Update" in the "Update/Delete" field.
8. Save the file as a .txt format and upload it under "Check and Upload Your Inventory File" on the "Add Products via Upload" page.

#### 4. Do I need to use the same variation theme when merging ASINs?

Yes, when merging ASINs, the variation theme of the new Child ASINs must match the variation theme of the Parent ASIN. However, you can change the variation theme if you're using the file template to edit or create variations.

#### 5. What if I need to merge variations from multiple Parent ASINs?

You can repeat the same process for each Parent ASIN, ensuring that the Child ASINs are merged into the correct Parent ASIN, maintaining the proper variation theme and category.

#### 6. Can I merge variations if the Parent ASIN is new?

Yes, if the Parent ASIN is new, you will need to provide all the necessary details for the Parent ASIN as if you are creating a new listing. Ensure you select "Update" in the "Update/Delete" field during the file upload process.

#### 7. Can I change the variation theme when merging ASINs?

Yes, you can change the variation theme when using the file template method. However, the variation theme must match the Parent ASIN when merging via manual methods.
