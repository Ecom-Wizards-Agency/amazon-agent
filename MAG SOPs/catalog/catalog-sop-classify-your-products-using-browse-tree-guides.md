## Classify Your Products Using Browse Tree Guides

**Who is this for :** This is for individuals who would like to classify their product's browse node ID.

**Objective :** This SOP aims to provide a step-by-step guide on how to classify the reader's products using Amazon's Browse Tree Guides (BTGs). BTGs are a hierarchical set of categories used to organize Amazon's products, making it easier for customers to navigate and find specific products.

**To determine the category of your product using browse node ID, you can follow the steps below:**

1. Go to this help page: [Inventory file templates](https://sellercentral.amazon.com/help/hub/reference/1641).
2. Look for Text-file Templates (for the CA marketplace) or Templates, style guides, and BTG for specific categories (for the US marketplace) and Choose the category for your product.

FAQs

1. **What is a Browse Node ID, and why is it important?**

- A Browse Node ID is a unique numerical identifier used by Amazon to classify products into specific categories. Selecting the correct node helps:

  - Ensure your product appears in the right category filters.
  - Improve search discoverability.
  - Prevent misclassification issues that might reduce visibility.

1. **How do I choose the correct Browse Node ID if multiple options are available?**

- Find the most specific node related to your product.
- Look at competitor listings in your category to see their node ID.
- If unsure, start with the closest match, monitor performance, and adjust if needed.

1. **Where do I enter the Browse Node ID in my listing?**

- If Adding a New Product Manually:

  - Go to Catalog > Add a Product.
  - Select I’m adding a product not sold on Amazon.
  - Enter the Browse Node ID in the category selection step.
  - If Updating an Existing Listing via Flat File:
  - Download the Inventory File Template for your category.
  - Locate the Recommended Browse Node (item_type) column.
  - Enter the correct Browse Node ID.
  - Save and upload the file via Catalog > Add Products via Upload.

1. **How long does it take for Browse Node ID changes to take effect?**

- Changes typically process within 15–30 minutes, but in some cases, it may take up to 24 hours to reflect on the product detail page.

1. **What if my Browse Node ID update isn’t reflecting?**

- Check for processing delays (wait at least 24 hours).
- Re-upload the flat file with a partial update.
- Clear browser cache and check from an incognito window.
- If the issue persists, submit a ticket to Amazon Seller Support with your ASIN and the correct Node ID.

1. **Can Amazon change Browse Node IDs over time?**

- Yes, Amazon periodically updates, merges, or removes browse nodes. It’s a good practice to:

  - Regularly check the latest BTGs to ensure accuracy.
  - Monitor changes in Seller Central notifications regarding category updates.
  - Verify your product classification every few months.
