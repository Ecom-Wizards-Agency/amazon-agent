## SAFE-T Claims

**Who is this for :** For individuals who need to make a Safe T claim.

**Objective :** To help us learn the process of filing Safe T claims and ensure that all Safe T claims submitted are processed accurately and efficiently.

The Seller Assurance for e-Commerce Transactions (SAFE-T) process allows you to file a claim for reimbursement when Amazon manual payment is required for your order. At Amazon's sole discretion, you may be issued a reimbursement in cases where Amazon determines that you were not at fault.

1. Access Seller-Central

2. Navigate to SAFE-T Claims by going to Orders → Manage SAFE-T Claims

**NOTE:** Safe-T Claims are ONLY For MFN Orders that used Amazon's auto returns program

FAQs

1. **What makes an order eligible for a SAFE-T claim?**

- To file a SAFE-T claim, the order must be a Merchant Fulfilled Network (MFN) order that was processed through Amazon's auto returns program. Additionally, you must have already filed a refund for the order before submitting the claim. Claims can only be filed after a refund is processed and if it was Amazon's error or if Amazon determines you were not at fault.

1. **Why do I need to file a refund before submitting a SAFE-T claim?**

- The refund must be filed first because SAFE-T claims are meant to address situations where Amazon deems there to be a discrepancy with the return or refund process, and you need manual reimbursement. The refund acts as confirmation that the item was returned, and the SAFE-T claim is a way of recouping that loss.

1. **What types of issues can I file a SAFE-T claim for?**

- SAFE-T claims are typically filed for situations where you, as the seller, need a manual reimbursement after an issue with an MFN order. Common scenarios include items returned through Amazon’s auto returns program that were not properly processed or returned in a condition that doesn’t match the original order (e.g., the buyer returns the wrong item or damaged goods). For example, if the customer returns an empty box or the wrong product, you can file a claim to be reimbursed.

1. **What types of proof should I provide when filing a SAFE-T claim?**

- To file a SAFE-T claim, you must provide evidence to support your case. This might include:

  - Photos of the returned item (e.g., if a customer returns something that’s not the original product).
  - The packing slip showing the original order details.
  - Any communication you had with the customer (e.g., an email or message indicating the issue). The more detailed and specific your evidence is, the better Amazon can process your claim quickly.

1. **How long does it take for Amazon to process my SAFE-T claim?**

- Amazon typically responds to SAFE-T claims within a few business days, but the exact processing time can vary depending on the complexity of the case. Keep an eye on the Claim Status in the SAFE-T Claims Dashboard for updates. If there are delays, you may need to contact Amazon Seller Support for clarification.

1. **What should I do if my SAFE-T claim is rejected or denied?**

- If your claim is rejected, review the reason for denial in the claim status or notification from Amazon. In many cases, you can file an appeal or reopen the case by providing additional documentation or clarifying the situation. If the issue persists, you can contact Amazon Seller Support for assistance.

1. **What does the SAFE-T Claims Dashboard show about my claim status?**

- The SAFE-T Claims Dashboard allows you to track the status of your claim, including:

  - Whether the claim is under review,
  - If additional information is required,
  - Whether the claim has been approved or denied,
  - If a reimbursement has been issued. If you're unsure about what the status means, you can always contact Seller Support for further clarification.

1. **Can I submit multiple SAFE-T claims for the same order?**

- Typically, only one SAFE-T claim can be filed per order. If your claim is denied or partially resolved, you may need to provide more evidence and work with Seller Support to rectify the issue. If you believe that multiple issues occurred with the same order, make sure to detail all of them in a single claim.
