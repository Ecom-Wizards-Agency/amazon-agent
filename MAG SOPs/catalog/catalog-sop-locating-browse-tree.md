# Catalog SOP: Locating Browse Tree

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-locating-browse-tree

## **Locating Browse Tree**

**Who is this for :** For anyone responsible for populating Category-Specific Templates for products on Amazon, particularly those who need to determine the most accurate category for a product.

**Objective :** This SOP outlines the steps to locate the correct category for a product by using the Browse Tree Guide in the Category-Specific Inventory Files.

**Steps:**

1. Open the Category-Specific Template and locate the "Item Type Keyword" column.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1721689865968.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/njSimage.png)
2. Normally, the cell is is not editable. But, it can be replaced using the Browse Tree Guide. You can use the guide to put the accepted replacement categories per Amazon.
3. Go to [this site](https://sellercentral.amazon.com/help/hub/reference/G1641?mons_sel_mkid=amzn1.mp.o.ATVPDKIKX0DER&mons_sel_mcid=amzn1.merchant.o.A3OO04JJ922PXW&mons_sel_persist=true) and go to "Use the following template versions".
4. Select the "Browse Tree Guide" alongside the relevant category of the template you have downloaded.
5. Open the file and review the worksheet named "Instructions" on how to use the Browse Tree Guide.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1721689134141.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/FiLimage.png)
6. On the second tab, named after the specific category you are using, locate the relevant text in the "Item Type Keyword" or "Query" field for US-based Templates, or the number in the "Node ID" field for UK and European-based Templates.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1721689601567.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/8kEimage.png)
7. Copy and paste the relevant text or number into the "Item Type Keyword" column of your Category-Specific Inventory File Template.
8. Repeat steps 7-8 for each product you are populating in the Category-Specific Template.
9. Using these Browse Nodes will ensure that your product is entered in the correct category for easier searching by customers.

FAQs

1. What is the Browse Tree Guide, and how does it help with product categorization?

The Browse Tree Guide is a tool provided by Amazon to help sellers identify the correct category for their products. It contains a structured list of accepted category keywords and node IDs for different product types. By using the guide, sellers can ensure that their products are accurately classified, making it easier for customers to find them.

2. What should I do if the "Item Type Keyword" column is not editable in the Category-Specific Template?

The Item Type Keyword column is typically non-editable, but you can still update it by using the Browse Tree Guide. The guide provides a list of valid keywords or node IDs that can be copied and pasted into the template for the correct category.

3. How do I find the correct Browse Tree Guide for my template?

After opening your Category-Specific Template, visit the provided website and go to the "Use the following template versions" section. Select the appropriate Browse Tree Guide corresponding to the category of your product, download the file, and follow the instructions in the "Instructions" worksheet.

4. What is the difference between the "Item Type Keyword" and the "Node ID" fields in the Browse Tree Guide?

- Item Type Keyword: This field is used for US-based templates and contains the text descriptions for product categories.
- Node ID: This is used for UK and European-based templates and represents the numeric ID for a specific category.

5. How do I populate the "Item Type Keyword" column in the Category-Specific Template?

After reviewing the Browse Tree Guide, locate the relevant text in the "Item Type Keyword" (for US templates) or "Node ID" (for UK and European templates) column in the guide. Copy the appropriate keyword or node ID and paste it into the Item Type Keyword column of your Category-Specific Inventory File Template.

6. Do I need to repeat the process for each product in the Category-Specific Template?

Yes, you need to repeat the process for each product you are categorizing. For every product, locate and enter the correct Item Type Keyword or Node ID from the Browse Tree Guide to ensure accurate categorization.

7. What if I can’t find a suitable category for my product in the Browse Tree Guide?

If you are unable to find a suitable category, check the product’s description and intended use to identify similar categories. If necessary, you can contact Amazon Seller Support for guidance on which category to choose.

8. How do I know if I have correctly populated the category for my product?

By using the correct Item Type Keyword or Node ID, your product will be assigned to the appropriate category, making it easier for customers to find. You can verify the category by reviewing your listing after it has been updated.

9. Is the Browse Tree Guide available for all product categories?

Yes, the Browse Tree Guide covers a wide range of product categories, but availability may vary. Make sure you download the correct guide for the specific category you are working with.

10. How long does it take for the changes to reflect after updating the Item Type Keyword in the template?

After updating the Item Type Keyword in the template and uploading it, changes should reflect within 24 hours. However, it may take a bit longer during high-volume periods or if additional product data processing is required.
