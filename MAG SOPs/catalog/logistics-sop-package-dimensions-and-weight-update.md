# Logistics SOP: Package Dimensions and Weight Update

Source: https://sop.myamazonguy.com/books/catalog/page/logistics-sop-package-dimensions-and-weight-update

## **Package Dimensions and Weight Update**

**Who is this for :** This SOP is intended for individuals who manage FBA (Fulfillment by Amazon) listings and are responsible for updating package dimensions and weight to lower FBA fees and get reimbursements.

**Objective :** The objective of this SOP is to provide clear and concise instructions for updating package dimensions and weight for FBA listings to lower FBA fees and get reimbursements, using one of the three available methods: manual edit, feed file, and calling seller support, or Cubiscan request.

**Note :** Package dimensions and weight updates are only applicable for FBA listings.

> **IMPORTANT UPDATE!**
>
> Starting June 20, 2025, the maximum box length for Fulfillment by Amazon (FBA) orders in the US will increase from 25 to 36 inches.
>
> This change allows you to use longer cartons across multiple sales channel and to meet industry standards.
>
> The maximum box dimensions will now be as follows:
>
> •**Length: 36 inches**
>
> **•Width: 25 inches**
>
> **•Height: 25 inches**
>
> **•Weight: 50 lb**
>
> If you've already sent in packages under the new dimensions, they’ll be received as standard size boxes with no defects or fees.
>
> For more information, go to FBA shipping and routing requirements.
>
> [**https://sellercentral.amazon.com/help/hub/reference/G200141510**](https://sellercentral.amazon.com/help/hub/reference/G200141510)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%205.02.08%E2%80%AFAM.png)

- Key in the correct dimensions and weight and then click “Save and finish”.
- Wait for 15 minutes to 24 hours before the changes reflect on Amazon’s end.

**2. Feed the File & Calling Seller Support**

- Download a Category Listing Report (CLR). Request CLR through Reports > Inventory Reports.
- Search the FBA SKU in the CLR and go to its Fulfillment section:

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%205.02.24%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/5K2image.png)

- Key in the dimension values and weight values and save the file.
- Go to Catalog>Add Products via Upload>Upload your spreadsheet>Upload the CLR File>Input your email address>Click Upload File.
- Get the batch ID (for reporting purposes and see the error message if any).
- If successful, wait 15 minutes to 24 hours before the changes reflect on Amazon’s end.
- If NOT, go to SC Help, file a case, and call/email SC support to push the changes

FAQs

**1. Can I update package dimensions and weight for all FBA products?**

- Not all products are eligible for updates. Amazon may restrict updates for:

  - Hazmat items
  - Products already in transit or at an FBA warehouse
  - Certain high-volume ASINs with automated verification systems

**2. How do I verify that my package dimension/weight update was applied?**

- Go to Manage All Inventory > Edit ASIN > More Details to check if the new values are displayed.
- Download a Category Listing Report (CLR) and confirm if the updates are reflected.
- If fees remain unchanged, use the FBA Revenue Calculator to check if dimensions are affecting costs.

**3. What should I do if Amazon rejects my package dimension/weight update?**

- Check for errors in the uploaded feed file or manual entry.
- Submit a case through Seller Support and request manual verification.
- Provide proof, such as:
- Manufacturer specifications
- Product packaging images with a measuring tape
- FBA shipment labels showing accurate weight/dimensions

**4. What if Amazon updates my dimensions, but FBA fees increase instead of decreasing?**

- Compare the before and after fee breakdowns using the FBA Revenue Calculator.
- If the update led to an incorrect increase, submit a reimbursement claim with Seller Support.

**5. How long do I have to request an FBA reimbursement for incorrect fees?**

- Amazon allows up to 18 months for requesting reimbursements due to incorrect FBA fees.
- Check Reports > Payments > Fee Preview to track any historical overcharges.

**6. How do I dispute FBA fee overcharges caused by incorrect dimensions?**

- Open a case in Seller Central > Help > FBA Fee Investigation.
- Provide evidence of incorrect dimensions being used to calculate fees.
- If Amazon denies the claim, escalate the issue and request a Cubiscan verification.

**7. What should I do if Amazon refuses to update my package dimensions even after using all three methods?**

- Submit a Cubiscan request to Amazon and provide before-and-after measurement photos.
- Request an FBA audit via Seller Support, providing proof of incorrect measurements.
- If the issue persists, escalate to Amazon’s Leadership Team through a support case.
