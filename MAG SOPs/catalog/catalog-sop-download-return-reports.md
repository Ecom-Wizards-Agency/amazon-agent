# Catalog SOP: Download Return Reports

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-download-return-reports

## Download Return Reports

**Who is this for :** Intended for individuals who are responsible for tracking and analyzing returns within a company or organization. It is designed for personnel who require access to the system or application that generates return reports and need to download the reports for further analysis, reporting, or decision-making purposes.

**Objective :** To outline the process for downloading and generating return reports for a specific system or application. This SOP provides a step-by-step guide for authorized personnel to access the necessary reports and retrieve relevant data related to returns.

**To generate a Returns Report:**

1. Navigate to the "Reports" tab on the top menu of Seller Central and select "Return Reports" from the drop-down menu.![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/catalog-sop-download-return-reports-image-1.png)

2. On the next page, choose the type of returns you want to include in the report from the provided drop-down menu, such as "All Returns" or "Prime Returns".

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/NYIimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/NYIimage.png)![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/catalog-sop-download-return-reports-image-2.png)

3. Select the date range for the report from the next drop-down menu, which includes options like "Last 1 Day", "Last 7 Days", "Last 30 Days", or "Exact Dates".![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/catalog-sop-download-return-reports-image-3.png)

4. If you select "Exact Dates", enter the "From" and "To" dates in the pop-up text boxes.![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/catalog-sop-download-return-reports-image-4.png)

FAQs

1. Where can I find the "Return Reports" in Seller Central?

To access the Return Reports, navigate to the Reports tab at the top of Seller Central, then select Return Reports from the drop-down menu.

2. How do I choose the type of returns I want to include in the report?

After selecting the Return Reports tab, you'll be able to choose the type of returns you want to include from the drop-down menu. Options include "All Returns" or "Prime Returns", depending on the data you wish to analyze.

3. How do I select the date range for the Return Report?

You can select the date range from the drop-down menu. Available options include:

- Last 1 Day
- Last 7 Days
- Last 30 Days
- Exact Dates (to select a custom range)

If you choose "Exact Dates", you can enter the "From" and "To" dates in the pop-up text boxes to specify your desired range.

4. What should I do if I want a custom date range for the Return Report?

If you need a specific date range, select "Exact Dates" from the drop-down menu and then enter the desired "From" and "To" dates in the text boxes that pop up.

5. How do I generate the Return Report?

After selecting the return type and date range, click the "Request" button to generate the report. Then, click the "Refresh" button located at the top right corner of the "Generated Reports" table to update the list of reports.

6. How can I download the generated Return Report?

Once the report is generated and appears in the "Generated Reports" table, you can download it as either a .xml or .tsv file by clicking the respective download link next to the report.

7. What file formats are available for downloading the Return Report?

You can download the Return Report in the following formats:

- .xml (for structured data)
- .tsv (tab-separated values)

8. How long does it take to generate the Return Report?

The time required to generate the report may vary depending on the size of the dataset. However, once you click "Request", the report should be available shortly after refreshing the table.

9. What should I do if the Return Report is not generating?

If the report is not generating, try the following steps:

- Click the "Refresh" button to ensure the latest data is loaded.
- Double-check your date range and return type selection.
- If the problem persists, contact Seller Support for assistance.

10. How can I verify the report before downloading it?

After generating the report, ensure that it appears in the "Generated Reports" table with the correct date range and return type. If everything looks accurate, you can proceed to download it.

11. Can I download multiple Return Reports at once?

No, each report needs to be downloaded individually. However, you can generate multiple reports by selecting different date ranges or return types one at a time.

12. How can I view detailed information in the Return Report?

After downloading the report in .xml or .tsv format, you can open it in an application like Microsoft Excel or any text editor to view the detailed information, including individual return transactions.

13. What should I do if I need data from a previous time period but it’s not showing in the report?

If the data you need is not showing in the report, it might be due to the date range or type of returns selected. Double-check the date range and report settings, or refer to additional reports if available (such as older transaction reports).
