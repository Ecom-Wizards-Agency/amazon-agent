# Catalog SOP: Setting the Available Quantity for Seller-Fulfilled Prime SKUs

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-setting-the-available-quantity-for-seller-fulfilled-prime-skus

## **Setting the Available Quantity for Seller-Fulfilled Prime SKUs**

**Who is this for :** This is intended for Amazon Sellers and other individuals that want to set the Available Quantity for their FBM listings.

**Objective :** To provide a step-by-step guide for Amazon Sellers to update the Available Quantity for their Seller-Fulfilled Prime SKUs.

1. Go to Seller Central's top menu and select "Inventory," then click on "Manage Inventory" from the dropdown menu.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%2011.51.08%E2%80%AFPM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/KOIimage.png)
2. Find your new SKU, which should now be listed as Seller/Merchant Fulfilled.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%2011.51.16%E2%80%AFPM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/a2Simage.png)
3. You can set the quantity available for this SKU in the "Available" field next to it.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%2011.51.24%E2%80%AFPM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/f0Fimage.png)
4. If you're not ready to ship this SKU yet, you can temporarily set the "Available" field to 0.
5. Once you've made the necessary changes, click "Save all" at the top right of the table to save your edits.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-05%20at%2011.51.32%E2%80%AFPM.png)

  FAQs

  **1. Who can update the Available Quantity for Seller-Fulfilled Prime SKUs?**

  - Only Amazon Sellers managing Seller-Fulfilled Prime (SFP) listings can update the Available Quantity. This does not apply to FBA listings.

  **2. What happens if I set the Available Quantity to zero?**

  - The SKU remains active, but customers will see it as "Out of Stock." If left at zero for an extended period, the listing may become inactive.

  **3. Can I update the quantity for multiple SKUs at once?**

  - Yes, you can use bulk uploads via a CSV file in Inventory Loader or the Inventory File Template in Seller Central.

  **4. How long does it take for quantity updates to reflect on my listing?**

  - Updates usually appear within 15 minutes but can take up to 24 hours depending on system processing.

  **5. Can I use APIs or third-party inventory management tools to update the Available Quantity?**

  - Yes, Amazon supports inventory updates via its Selling Partner API (SP-API) and third-party tools that integrate with Seller Central.

  **6. Are there any restrictions on how often I can change the Available Quantity**?

  - No, but excessive changes in a short period may trigger account reviews or listing errors.

  **7. What should I do if I can’t update the Available Quantity?**

  - First, check for system errors or listing restrictions. If the issue persists, contact Seller Support for troubleshooting.

  **8. Will changing the Available Quantity affect my Prime eligibility or Buy Box status?**

  - Reducing stock levels may impact Buy Box competitiveness. Keeping consistent inventory levels helps maintain Seller-Fulfilled Prime eligibility.
