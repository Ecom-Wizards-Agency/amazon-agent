## Supplemental Article: Understanding Order Handling Capacity on Amazon Seller Central

**Introduction:**

As an Amazon seller, ensuring timely delivery is key to maintaining a successful business. The **Order Handling Capacity** (OHC) feature on Amazon Seller Central provides a solution for managing order fulfillment, especially during high-demand periods. It allows sellers to set a limit on the number of orders they can handle within a given timeframe. This helps prevent delays, protects your account health, and ensures a smoother customer experience.

**What is Order Handling Capacity?**

The **Order Handling Capacity** feature allows sellers to set a daily limit for the number of orders they can handle. When that limit is exceeded, the handling time for additional orders will automatically be extended by an extra day. This feature helps you maintain timely shipping, even during unexpected spikes in demand.

For example, if you set your order handling capacity to 100 orders per day and receive 120 orders in one day, the first 100 orders will be processed within the set default handling time (e.g., one day). The remaining 20 orders will have an additional day added to their handling time, resulting in a two-day handling time for those orders.

**Benefits of Using Order Handling Capacity**

1. **Better Control Over Order Fulfillment:**

  - Helps sellers handle fluctuations in order volume, especially during busy seasons like holidays or unexpected surges in popularity.
  - Reduces the likelihood of exceeding handling times, which could negatively impact account health.
2. **Accurate Delivery Promises:**

  - By setting a limit on your capacity, customers will see more accurate delivery times. For instance, when your capacity is reached, buyers will see a delivery promise that is one day later, aligning with the additional handling time.
3. **Enhanced Customer Experience:**

  - Accurate delivery times help customers trust your ability to fulfill orders, improving their shopping experience and increasing the likelihood of purchase.
4. **Protects Account Health:**

  - Ensures that orders are processed within the specified handling time, avoiding penalties related to late shipments or missed delivery promises.
5. **Useful for Professional Sellers:**

  - Only Professional sellers on Amazon are eligible to use the Order Handling Capacity feature, making it a crucial tool for large-scale operations.

**How Order Handling Capacity Works**

When you set your handling capacity, Amazon automatically adjusts the delivery times for orders that exceed your set limit. For example:

- **Scenario 1:** You set a one-day handling time and an order capacity of 100 orders per day. For the first 100 orders, Amazon will display a one-day handling time. For any order beyond 100, the handling time will extend to two days.
- **Scenario 2:** If your order handling capacity is exceeded by 20 orders, those extra orders will automatically count toward the next day's capacity.

**Note:** This feature does not apply to **Seller Fulfilled Prime (SFP)** orders or orders using the **Premium Shipping** option. These orders are excluded from the daily handling capacity.

**New Updates to Order Handling Capacity (*****Effective February 24, 2025*****)**

Amazon will be updating the **OHC** feature to automatically set a **minimum threshold** for sellers' order handling capacity. This threshold will be recalculated weekly based on your average daily orders over the past 30 days, helping you set a more accurate and realistic limit.

This update ensures that your handling times reflect your true fulfillment capacity, helping protect you from late deliveries and improving the customer experience. If your OHC is set below the minimum threshold, the system will automatically update it.

**Important Notes:**

- The minimum threshold will only apply to **Standard and Free Economy** shipping options.
- You can adjust your OHC above the threshold at any time.
- If you're taking a break, you can enable **Vacation Mode** to prevent automatic adjustments to your OHC while you're away.
