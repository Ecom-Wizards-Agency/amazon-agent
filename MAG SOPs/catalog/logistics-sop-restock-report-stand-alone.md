# Logistics SOP: Restock Report (Stand Alone)

Source: https://sop.myamazonguy.com/books/catalog/page/logistics-sop-restock-report-stand-alone

## **Catalog SOP: Restock Report (Stand Alone)**

**Who is this for?**

Intended for individuals and Specialist doing the restock recommendations.

**Objective:**

To streamline the generation of a comprehensive restock inventory report, enabling users to make well-informed decisions on replenishing inventory levels with efficiency, utilizing this standalone template.

**Template (Make a Copy): [Basic Restock Report Template](https://docs.google.com/spreadsheets/d/<google-sheet-id>/copy)**

**Reminders**:

- Files Needed To Download Per type of restock report

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%205.44.00%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/SW5image.png)

- Follow the correct naming convention for each file

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%205.42.48%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/BlXimage.png)

- For the All listings report make sure to include :

  - SKU
  - item-name
  - quantity
  - open-date
  - fulfillment-channel
  - asin1
- For the Inventory Report make sure to include :

  - SKU
  - quantity

**NOTE :**It is ok to add more attributes/columns to the reports you are downloading as long as the ones needed are included.

- Except for the Business Report, it is best to download everything as .txt

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Business%20Rerport(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/o9fimage.png)

- Use either the 60-Day Restock Report tab or the 90-Day Restock Report tab, depending on the timeframe you are checking.

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/eafimage.png)

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%205.59.24%E2%80%AFAM.png)

**NOTE : You can use it at the same time just download both the 60 and 90 days business reports.**

**How to use:**

1.**Make a copy for each client or use a single copy for all clients.**

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%205.50.08%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/m18image.png)

2. **From Menu Bar Click on : "Actions" > Reset > Reset All!**

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%205.51.05%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/yAZimage.png)

- For first-time use of copy, authorize the script

  - Click on “Ok”

    ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/pmLZqsimage.png)

    ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%206.JPG)
  - Then choose you Email address

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%207.JPG)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Kvbimage.png)

- Then Click on “Allow” for the Spreadsheet Scripts to be enabled.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%208.JPG)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/6LWimage.png)

**3. Once, Template is authorized, we can proceed to download needed report files and update the capacity monitor values.**

- Access Capacity Monitor from Seller Central > Inventory > Shipments

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Shipments.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/KDyimage.png)

- Scroll down and click on “Capacity Monitor”

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%20Capacity%20Monitor%202.JPG)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/4z9image.png)

- Here you will see the the information that you need.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%20Capacity%20Monitor%203.JPG)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/6ekimage.png)

- Alternatively we can also access Capacity Monitor from Inventory Performance Index

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%20Capacity%20Monitor%204.JPG)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/ydDimage.png)
- Update the Template Values

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%20Capacity%20Monitor%205(1).JPG)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/ViTimage.png)

**NOTE :** Check all storage types, and if the storage type is being used or has contents, also fill them out

**4. To populate template’s data either:**

- Download and save all necessary files to a folder, renaming them according to suggested naming conventions like ManageFBA, FBAinv, etc. All Listings and Business reports are fine as is.

  - Upload the files/saving folder to Gdrive

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%2010.JPG)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/OAvimage.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/tneimage.png)

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%2011.JPG)
- Copy uploaded folder link either by Share

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%2012.JPG)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/hpRimage.png)
- Or copy the link from the Address bar.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%2013.JPG)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/J0Pimage.png)

**5. Utilize the “Actions” button : Upload>Upload All Files**

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%205.53.31%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/ZoCimage.png)

**6. The Non-Gray Area will be your Report. Note that column I is enough and we exclude column S and T because we only need to send the higher value recommended based on available capacity limit.**

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%205.54.43%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/ZlJimage.png)

**7. Copy and paste values only to your Client's Excel reporting template, if you have no current available template or format, Copy and paste as is, then paste again as values only. Sort Recommended Shipment from Largest to smallest.**

Don’t forget to update the Date and Save with correct client’s name.

- Excel Reporting Template

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%2018(1).JPG)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/tt0image.png)
- Google Sheets Reporting Tracker (If still needed)

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%2017(2).JPG)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/z5timage.png)
- Then filter/sort it on your reporting excel or reporting tracker, first sort by Recommended Shipment Largest to Smallest then sort by Fulfillment.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Restock%20Report%20Standalone%2019.JPG)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/A9timage.png)

1. **You may now attach the excel file or include the tracker link if needed to the delivery email.**
