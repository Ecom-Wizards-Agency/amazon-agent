## **FBA Customer Returns Reimbursements**

**Who is this for :** Intended for Amazon sellers who use the FBA program to fulfill their orders.

**Objective :** To outline the process of FBA (Fulfillment by Amazon) Customer Returns Reimbursements for Amazon sellers to receive reimbursements for customer returns under the FBA program.

Normally, Amazon refunds the seller when a customer returns a product. This happens automatically, but sometimes customers don't return the product to Amazon. If the item isn't returned within 45 days, Amazon charges the customer and refunds the seller. However, sometimes this refund doesn't happen, so the seller needs to check and request reimbursement if necessary.

If a customer returns a sellable item to the Fulfillment Center within 45 days of the refund, Amazon returns it to the seller's inventory without reimbursement. But if the item is not sellable, Amazon decides who is responsible. If Amazon is at fault, they reimburse the seller and don't add the item back to their inventory.

**Note :** Amazon doesn't take responsibility or reimburse for returned items that are damaged by the customer, recalled, defective, or against Amazon policies. These items go into the seller's unfulfillable inventory.

5. Once downloaded, the seller should compare the Order IDs in the Returns Report against those in the Refund Report to see how many refunded orders are missing from the Returns Report.

- Use the VLOOKUP function in Excel to quickly identify unreimbursed returns on the Returns Report.

**Note:**

1. If an Order ID is marked as "N/A", it has not been refunded yet. Sellers should monitor Order IDs that are more than 45 days old and still marked as "N/A".
2. If an Amazon return has not been refunded after 50 days, the seller should open a case to request reimbursement.
3. To expedite the process, sellers can split their Order IDs into batches of 10-15 and investigate each batch in a separate case. This allows Amazon to attend to each batch more quickly.

FAQs

**1. Which types of products are eligible for FBA return reimbursements?**

- Amazon typically reimburses sellers if the returned product is:

  - Lost or not returned within 45 days.
  - Returned in unsellable condition due to Amazon’s fault.
  - Incorrectly processed by Amazon's fulfillment center.

    - Amazon will not reimburse for items that are:

      - Damaged by customers.
      - Defective, recalled, or restricted.
      - Against Amazon's policies (e.g., hazardous materials).

**2. How long does it take for Amazon to process a reimbursement request?**

- Once a case is opened, Amazon usually reviews and responds within 7-14 business days. However, complex cases may take longer.

**3. What should I do if Amazon rejects my reimbursement claim?**

- If your claim is denied:

  - Review Amazon’s response for specific reasons.
  - Gather additional evidence, such as:
  - Order ID screenshots.
  - FBA Customer Returns Report.
  - Refund Transaction Report.
  - Proof of shipment.
  - Reopen the case and provide additional documentation.
  - If denied again, escalate to Seller Support > Fulfillment by Amazon > Reimbursement Disputes.

**4. What are my options if Amazon refuses to reimburse a lost return?**

- If Amazon wrongfully denies reimbursement, you can escalate the case and request an investigation.
- If Amazon confirms the return is not their responsibility, you may need to file a claim through your insurance provider.

**5. Does opening too many reimbursement cases impact my account health?**

- No, requesting reimbursements does not negatively impact account health, but submitting excessive claims without valid proof may lead Amazon to limit future investigations.

**6. Can I automate the process of checking for missing reimbursements?**

- Yes. Instead of manually comparing reports, sellers can:

  - Use Amazon Seller Tools like Helium 10 or Refund Genie to track missing reimbursements.
  - Use Pivot Tables & VLOOKUP in Excel for faster matching.
  - Set up scheduled reports in Seller Central > Reports > Fulfillment > Customer Returns to review data periodically.

**7. Where do I open a case for a reimbursement request?**

- Go to Seller Central > Help > Contact Us > Fulfillment by Amazon > Investigate Missing FBA Reimbursements.
- Provide Order IDs (split into batches of 10-15).
- Attach relevant transaction & return reports as proof.
