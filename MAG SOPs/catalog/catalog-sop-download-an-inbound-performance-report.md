# Catalog SOP: Download an Inbound Performance Report

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-download-an-inbound-performance-report

## **Download an Inbound Performance Report**

**Who is this for :** For individuals who are responsible for monitoring and analyzing inbound performance metrics.

**Objective :** To provide clear guidelines and instructions for downloading an Inbound Performance Report. The SOP aims to ensure consistency and accuracy in the process of retrieving performance data, enabling stakeholders to make informed decisions and evaluate the effectiveness of inbound operations.

**Instructions**

1. To generate an Inbound Performance Report, go to "Reports" and then select "Fulfillment".

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684197576556.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/UICimage.png)

2. From the Fulfillment reports page, go to Inventory → Inbound Performance

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684198181601.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/8Idimage.png)

3. On the next page, select the Download tab. Choose the "Event Date" from the available drop-down menu. Select "Exact Dates" and enter the "From" and "To" dates respectively using the calendar icon next to each field.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684198223973.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/sWkimage.png)

4. Click on the "Request .csv Download" button, if you want to download the report in an Excel format, and click on the "Request.txt Download" button, for a text format.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684198411534.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/gHGimage.png)

5. The "Report Status" column in the table below will initially show the status as "In Progress".

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684198492784.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/HOdimage.png)

6. Once the report is ready, download your report by clicking the download button.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684198529447.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/vXNimage.png)

FAQs

1. Where can I find the Inbound Performance Report in the Reports section?

You can access the Inbound Performance Report by navigating to Reports > Fulfillment, then selecting Inventory > Inbound Performance from the Fulfillment reports page.

2. Can I select a custom date range when downloading the report?

Yes, you can select "Exact Dates" from the drop-down menu under the Download tab and specify the "From" and "To" dates using the calendar icon.

3. What file formats are available for downloading the Inbound Performance Report?

You can download the report in Excel (.csv) format by clicking the "Request .csv Download" button or in text (.txt) format by selecting "Request.txt Download."

4. How long does it take for the report to be generated?

The report status will initially show as "In Progress" in the Report Status column. The time required depends on the volume of data being processed.

5. How do I know when the report is ready for download?

Once the report status changes from "In Progress" to "Ready", you can download it by clicking the Download button.
