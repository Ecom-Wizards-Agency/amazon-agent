# Catalog SOP: Creating a Promo Code

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-creating-a-promo-code

## **Creating Promotion Code**

**Who is this for?**

All individuals who want to create promotion codes.

**Objective:**

To ensure that Amazon promotions are effectively used to enhance the visibility and sales of products

**Amazon Promotions** can help your products stand out from the competition and stimulate sales.

However, promotions are **most effective when your offer is the Featured Offer** (formerly known as "Buy Box"). Otherwise, the promotional messaging cannot appear on the detail page.

Before creating a Promo Code, it is important to communicate, and get client’s approval first. Include the approval proof (screenshot or fireflies call link) include in Asana as Task Details or as a Comment. Do the Same when there is an edit to be made on running promotions except if the need to edit is critical.

Before creating or implementing new discount, promotion or coupons, check first if there is an existing running or scheduled one to make sure that these don't overlap in a bad way or end up cancelling one another or cause pricing errors.

In a way, stacking or putting in two different price-off strategy could help more rather than a single flat discount or promotion as this will give both strike through and other discount badge on PDP price making it more appealing to buyer, but we have to confirm and get client's approval as they may not really approve such as for coupons which charges $0.60 for every redemption.

Please take time to watch this informative video about discount, coupons and promotions stacking.

**Setting up Social Media Promo Code**

You can create social media promo codes to offer a percentage discount to customers on eligible products. Once you create a social media promo code, you’ll receive a URL for a unique landing page that contains a list of eligible products for the promotion and the promotion details. The landing page exclusively features your offers, so customers do not need to locate your offer in the Offer Display. Share this landing page on social media or marketing channels, such as paid search, to direct customers and promote sales for your products. Customers can add items to their cart directly from this landing page, and the discount will automatically apply.

Social media promo codes can start four hours after they are created and can run up to 30 days. **The discount must be at least 5% and a maximum of 80% off the current price**. You can set the code to be redeemed on one product or multiple products. After successfully checking out, a customer won't be able to redeem the promo code again unless you allow multiple redemptions per customer.

**Setting Up Percentage Off**

Use Advanced options to set up a promotion with a tiered discount structure to encourage customers to buy more because they get a better discount (no Buy Box impact). To discount single-unit purchases, consider implementing a limited-time Sale Price (impacts Buy Box).

1. **Buyer purchases** - we have 3 options to choose from

a. At least the amount purchased (in $)

b. At least the number of items purchased

c. For every quantity of items purchased

1. **Purchased Items** - If the promotion encompasses the entire catalog, please choose it from the dropdown menu. Alternatively, if the promotion does not cover the entire catalog, you can select specific SKUs or ASINs to include in the promotion.

1. Under the Additional Options field you can select one of the three types of claim code:

- **Single Use**: Ideal if you have a limited inventory intended for the promotion. It generates a unique claim code each time a customer uses the promo. You can also limit this to one redemption per customer. However, you don't have the option to show the promotion on the product detail page when this is selected.

This can be risky if somebody posts the claim code on a social media platform if you have limited inventory intended for the promotion. You also have the option here to display the promotion on the PDP.

This could be dangerous if there are multiple promotions set up under the account as all discounts get added up. **NOTE:**

There are 3 options for **Claim Code Combinability**

- **Preferential**: Only 1 promo will be applied to the purchase. If the customer attempts to add multiple promotions, only the one with the greatest discount will be applied. This is a good choice for sellers who are offering multiple types of promotions for the same product selection.
- **Unrestricted**: Be **extremely**cautious before choosing this option, especially when you’re using a “group” claim code. This option means your promotion can be stacked along with other promotions that are running for your products. This could result in you giving multiple discounts for one order, and if you have Group promo codes, people can place unlimited orders.
- **Exclusive**: This is the best option for most sellers. This states that no other promotions can be applied at the same time as this promotion. This prevents stacking promotions, and discounts, and ensures that you are protected from people who would abuse the system.

FAQs

1. What should I do before creating a new promotion?

Before creating a new promotion, check if there are any existing or scheduled promotions to avoid conflicts, pricing errors, or cancellations. Additionally, get client approval and document proof (screenshot or Fireflies call link) in Asana under Task Details or Comments.

2. How do I ensure my promotion appears on the product detail page?

Promotions are most effective when your offer is the Featured Offer (Buy Box). If your product is not the Featured Offer, the promotional messaging may not appear on the product detail page.

3. Can I run multiple promotions on the same product at the same time?

Yes, but you need client approval first. Some stacking strategies can be beneficial, such as combining strike-through pricing with a percentage discount. However, be cautious, as some promotions (like coupons) may incur additional charges.

4. What is the minimum and maximum discount I can set for a Social Media Promo Code?

The discount must be at least 5% and can go up to a maximum of 80% off the current price.

5. How do I prevent customers from using a promo code multiple times?

When setting up your promotion, choose Single Use as the Claim Code type. This generates a unique claim code for each redemption and can be restricted to one use per customer.

6. What happens if I set the Claim Code Combinability to “Unrestricted”?

If you choose Unrestricted, multiple promotions can be stacked together. This can lead to excessive discounts if multiple promotions are running at the same time. Use this option cautiously, especially with Group claim codes.

7. How can I ensure my promotion stays within budget?

Promotional budgets are for planning purposes only and can be exceeded. However, Amazon will automatically take your promotion offline when it reaches 80% of the set budget.

8. What’s the best Claim Code Combinability option to prevent abuse?

The safest option is Exclusive, which ensures no other promotions can be applied at the same time. This prevents stacking and protects against excessive discounts.

9. Can I edit a promotion after it has been submitted?

Yes, but any edits (except for critical changes) must also be approved by the client, and proof of approval should be documented in Asana.

10. How do I cancel or stop a running promotion?

Navigate to Manage Your Promotions in Seller Central, find the promotion you want to cancel, and follow the steps to deactivate it. Keep in mind that promotions might still apply if they were already used before cancellation.
