## **Edit an Automated Pricing Rule via File Template**

**Who is this for :** For Amazon sellers who are responsible for managing pricing rules in an automated pricing system.

**Objective :** To provide guidance on how to edit an existing automated pricing rule using a spreadsheet template. This SOP will ensure that all changes made to pricing rules are documented and updated accurately and efficiently.

**Important:** Prior any auto pricing rule implementation, it is important to inform, brief details and get client’s approval. Include the approval proof (screenshot or fireflies call link) as Task Details or as a Comment. If you have many listings, you may manage your pricing via a spreadsheet. To do this:

1. Access the "Automate Pricing" feature by selecting it from the "Pricing" tab in Seller Central. ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1683906254151.png)

2. To get the Automate Pricing file, click the Edit SKUs button or select the action drop down ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1683907308553.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/HMgimage.png)

3. Click the "Manage SKUs via File Upload" button at the top right of the product table. ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1683907422477.png)

5. Follow the instructions in the template and enter all the required information. ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1683908738410.png)

6. After completing the Automate Pricing file template, save and upload your file by going to the Automate Pricing "Upload Center" page, which can be found by clicking the "Upload Center" tab on the same page you downloaded the template from.

As soon as your file is processed, the rules will be applied to the SKUs you have specified. Automate Pricing adjusts your price according to the pricing rule, minimum price, and maximum price you enter. Additionally, during automated pricing, the price will not fall below the Minimum Price, nor exceed the Maximum Price. To learn more about this, visit Amazon help page: Manage SKUs using the Automate Pricing file - [https://sellercentral.amazon.com/help/hub/reference/G202166010](https://sellercentral.amazon.com/help/hub/reference/G202166010)

FAQs

1. What file format is required for the automate pricing file upload?

The automate pricing file should be uploaded in CSV format. Ensure that the file is saved correctly before uploading to prevent formatting errors.

2. What should I do if the automate pricing file upload fails?

If your file upload fails, check for the following common issues:

- Missing required fields (such as SKU, Minimum Price, or Maximum Price).
- Incorrect file format (ensure it is in CSV).
- Errors in pricing rules that conflict with Amazon policies.
- Try re-downloading the template and re-entering your data before uploading again.

3. How long does it take for Amazon to process the automate pricing file?

Amazon typically processes pricing file uploads within 15 to 30 minutes, but it can take longer depending on the number of SKUs being updated. You can check the status in the Upload Center.

4. How can I confirm that my pricing changes have been applied?

To verify that your pricing updates have been successfully applied:

- Go to the Automate Pricing page and check the status of your upload.
- Review your SKU pricing under Manage Inventory to confirm that the changes reflect correctly.
- If needed, manually adjust any discrepancies.

5. Can I edit an existing pricing rule, or do I need to create a new one?

You can edit an existing rule by downloading the automate pricing file, updating the necessary fields, and re-uploading it. If you want to apply a new strategy, you may need to create a new rule instead of modifying an old one.

6. Does updating pricing rules impact Buy Box eligibility?

Yes, changes in pricing rules can impact Buy Box eligibility. If your minimum price is too high compared to competitors, you may lose the Buy Box. Make sure your pricing strategy is competitive while maintaining profitability.

7. What are the best practices for managing bulk SKU updates?

- Use Excel formulas or scripts to verify pricing consistency before uploading.
- Test changes on a small batch of SKUs before applying them to your full catalog.
- Keep a backup copy of the previous pricing file in case you need to revert.

8. How do I revert pricing changes if needed?

If an update negatively impacts your pricing or sales, you can re-upload a previous version of your pricing file. Always keep a backup of your last uploaded file before making bulk changes.
