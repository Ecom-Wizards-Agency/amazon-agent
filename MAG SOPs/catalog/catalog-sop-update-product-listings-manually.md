# Catalog SOP: Update Product Listings Manually

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-update-product-listings-manually

## **Update Product Listings Manually**

**Who is this for:**For individuals who need to create or edit their product listings manually inside Amazon Seller Central
**Objective:** The objective of this SOP is to provide step-by-step instructions for updating listing information on Amazon by manually editing each listing, accessing the backend, and saving the changes, to ensure that new listing attributes are accurately reflected and absorbed by the Amazon system.

**Instructions:**

1. Secure a fresh copy of the Category Listings Report. To download the report, go to Reports > Inventory Reports > Category Listings Report > Request Report.

2. Go to Inventory > Manage Inventory

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1680130610835.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Jk3image.png)

3. Search the ASIN that you want to update. Make sure the filters are set to “All” for the Status and Fulfilled By.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202024-09-11%20at%209.54.21%E2%80%AFPM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/EG7image.png)

4. Click the “Edit” button. Sometimes, this button is on the far right. If there are dupe SKUs, meaning for one ASIN there are multiple SKUs, we need to update them as well.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202024-09-11%20at%209.55.45%E2%80%AFPM(2).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/5qximage.png)

5. To update the attribute for a product, find the attribute field on the backend and enter the correct attribute value within the contribution box.

- Number and Item are the same.
- Number of pieces - the number of pieces inside the packaging.
- Unit count - Category dependent

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202024-09-11%20at%209.57.47%E2%80%AFPM(5).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/dvwimage.png)

6. For image updates, just go to the Images tab, and click the Upload button to add images.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202024-09-11%20at%209.58.29%E2%80%AFPM(6).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/SVgimage.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/5Zximage.png)

7. Once confirmed that all information is correct, click Save and finish at the bottom.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1680131040436.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/gZmimage.png)

NOTE: Sometimes, it takes 15 minutes or less, and sometimes, it takes hours for the updates to reflect on the product detail page. Monitor and troubleshoot as needed until the correct attributes are successfully posted.

FAQs

**1. What is the Category Listings Report, and why do I need it?**

- The Category Listings Report is a downloadable report from Amazon Seller Central that provides an overview of your current product listings and their attributes. It is essential for verifying product details, identifying discrepancies, and ensuring accurate updates.

**2. What should I do if I cannot find the "Edit" button for a listing?**

- If the "Edit" button is missing, ensure you have the appropriate permissions for the account. If permissions are not the issue, check if the listing is inactive, restricted, or under a gated category that requires approval.

**3. How can I identify and update duplicate SKUs for a single ASIN?**

- Duplicate SKUs can appear when the same product is listed under multiple SKUs in your inventory. Search the ASIN in Inventory > Manage Inventory, and set the filters to “All.” Update the attributes for each SKU associated with the ASIN to ensure consistency.

**4. What should I do if the attribute field I need to update isn’t available?**

- If the required attribute field isn’t visible, it may not be enabled for manual updates or may require a Category Listings Report upload. You can contact Amazon Seller Support to request access to the missing field or use a bulk update file to make the change.

**5. How do I ensure the correct unit count for category-dependent listings?**

- Unit count requirements can vary by category. Review Amazon’s category-specific guidelines in Seller Central to confirm the correct unit count format and value before updating.

**6. What image file types and dimensions are accepted when uploading product images?**

- Amazon accepts images in .jpg, .jpeg, .png, .tif, and .gif formats. The recommended image dimensions are at least 1,000 x 1,000 pixels for optimal zoom functionality. Ensure the image adheres to Amazon’s image standards to avoid upload errors.

**7. What should I do if updates don’t reflect on the product detail page?**

- If updates don’t appear, wait at least 15–60 minutes for Amazon to process the changes. If the issue persists:

  - Clear your browser cache or try a different browser.
  - Revisit the Manage Inventory page and verify that the changes were saved.
  - Open a case with Amazon Seller Support if the updates still fail to appear.

**8. Can I edit product listings for inactive or suppressed listings?**

- Yes, you can edit inactive or suppressed listings by searching for the ASIN in Manage Inventory. Update the necessary attributes and resolve suppression issues if applicable, such as missing images, incorrect attributes, or policy violations.

**9. How do I monitor and troubleshoot incorrect updates?**

- To monitor updates:

  - Check the product detail page periodically after saving changes.
  - Use the "Contributions" view (if available) in the backend to verify your updates were accepted by Amazon.
  - If issues arise, ensure there are no conflicting contributions from other sellers or third-party systems linked to your account.

**10. Can I update multiple attributes at once for a single ASIN?**

- Yes, you can update multiple attributes simultaneously by editing the fields within the backend of a single listing. Make sure to verify each change before clicking “Save and finish.”

**11. What should I do if the system rejects my attribute update?**

- Attribute updates may be rejected due to invalid values, restricted fields, or category-specific rules. Double-check the input against Amazon’s listing requirements, correct the issue, and resubmit. Contact Seller Support for further guidance if necessary.

**12. How do I update listings across multiple marketplaces?**

- If your product is listed in multiple Amazon marketplaces, changes made in one region (e.g., the US) may not automatically apply to others (e.g., the UK). You must manually update the listings for each marketplace or use a global inventory tool if available.

**13. Can I revert changes if I make a mistake?**

- If an error is made, you can return to the Manage Inventory page, edit the listing again, and correct the mistake. Be aware that changes may take time to propagate and override incorrect details.

**14. Are there any risks to manually updating listings?**

- Risks include overwriting important data, losing contributions due to other sellers or automated tools, or violating Amazon’s listing guidelines. Always double-check updates before saving and monitor listings afterward to ensure accuracy.
