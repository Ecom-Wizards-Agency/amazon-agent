## **How to Generate a Removal Order Detail Report**

**Who is this for :** For sellers using Seller Central who need to generate a Removal Order Detail Report. It provides step-by-step instructions on how to access and download the report.

**Objective :** To guide sellers in generating a Removal Order Detail Report in Seller Central. The report provides crucial information about removal orders, including order type, order status, product information, removal fees, and tracking information for returns.

The Removal Order Detail Report is a valuable tool for sellers using Seller Central to gather comprehensive information about their removal orders. This report not only provides details on order types, order status, and quantities by removal type but also offers insights into the associated product information and removal fees charged for completed removal orders. Additionally, sellers can access tracking information for pending and completed returns, providing visibility into the shipment process.

**To generate a Removal Order Detail Report in Seller Central, please follow the steps below:**

1. Go to the "Reports" tab on the top menu and select "Fulfillment" from the drop-down menu.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-15%20at%2012.13.59%E2%80%AFAM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/9baimage.png)
2. Under the "Removals" header on the left side of the page, click the "Removal Order Detail" link.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-15%20at%2012.14.05%E2%80%AFAM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/8OXimage.png)
3. On the following page, choose the event date by selecting from the drop-down menu or selecting "Exact Dates" and entering the "From" and "To" dates using the calendar icons. You can also enter your Merchant SKU, Removal Order ID, or Fulfillment Network SKU to generate a more specific report.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/20241015-034522(2).gif)

  ![How to Generate a Removal Order Detail Report.gif](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/how-to-generate-a-removal-order-detail-report.gif)

**To download a Txt or a CSV copy of the report, follow the instructions below:**

1. On the same page, click "Download".

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-15%20at%2012.15.12%E2%80%AFAM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/rnUimage.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1684340387041(1).png)

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-15%20at%2012.17.59%E2%80%AFAM.png)

  ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/yawimage.png)

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-15%20at%2012.18.07%E2%80%AFAM.png)

FAQs

1. What is the difference between the Removal Order Detail Report and the Removal Shipment Detail Report?

The Removal Order Detail Report provides information on removal orders, including order type, status, product details, removal fees, and tracking information. The Removal Shipment Detail Report focuses on the actual shipments from Amazon’s fulfillment centers, including SKUs, quantities, and disposition of inventory.

2. How long does it take for the report to be generated?

The report is typically available within a few minutes, but larger data sets may take longer. If the status remains "In Progress" for an extended period, refreshing the page or re-requesting the report may help.

3. Can I schedule automated downloads of the Removal Order Detail Report?

No, Seller Central does not currently offer a built-in scheduling feature for automatic downloads of the report. However, you can use Amazon’s Reports API to automate retrieval.

4. How far back can I retrieve a Removal Order Detail Report?

Amazon typically allows access to reports for the last 90 days. For older reports, you may need to contact Amazon Seller Support.

5. What do I do if I don’t see the "Removal Order Detail" link under the Removals section?

Ensure that you have the necessary permissions in Seller Central. If the link is missing, check your account settings or contact your account administrator.

6. Can I generate a report for specific SKUs or order IDs?

Yes, you can filter the report by Merchant SKU, Removal Order ID, or Fulfillment Network SKU when generating it. This allows for more specific data retrieval.

7. Why is my downloaded report missing certain removal orders?

If some removal orders are missing, ensure that the date range selected includes the period when the orders were processed. Also, check if the correct filters (such as SKU or order ID) were applied.

8. What format should I use: CSV or TXT?

CSV format is best for opening in spreadsheet applications like Excel or Google Sheets. TXT format is useful for processing data in text-based applications or scripts. Choose the format based on your data analysis or integration needs.

9. Can I generate the report on a mobile device?

While Seller Central is accessible on mobile, downloading and viewing CSV/TXT reports is easier on a desktop or laptop.

10. How can I use the tracking information in the report?

The tracking information helps you monitor removal shipments, including estimated delivery times. You can use the tracking details to verify receipt of returned items or check carrier updates.
