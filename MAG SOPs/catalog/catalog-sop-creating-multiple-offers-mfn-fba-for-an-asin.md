# Catalog SOP: Creating Multiple Offers (MFN + FBA) for an ASIN

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-creating-multiple-offers-mfn-fba-for-an-asin

## **Creating Multiple Offers (MFN + FBA) for an ASIN**

**Who is this for :** Individuals who will create skeleton listings with multiple offers of different fulfillment methods.

**Objective :** Successfully create accurate listings with duplicate FBM and FBA offers, maximizing listing accuracy and consistency while reducing errors and turnaround time.

**Instructions:**

**REVIEW THE PRE-REQUISITE SOP:** [Amazon Listing Creation](https://sop.myamazonguy.com/books/catalog/page/merchandising-sop-amazon-listing-creation)

**CAUTION:**

- DO NOT create both offers (FBM or FBA) simultaneously to avoid generating two (2) different ASINs with the exact UPC/GTIN/EAN number.
- You can either choose to create first an FBM or FBA listing.

In this example, we will be creating the FBA listings first

**Prepare The Flat File**

1. Download the correct flat file using the "Category Specific" template.

2. Open the flat file template. Cells that are highlighted in red are required fields although we can still create the listing without filling in each cell. You’ll know which columns should have values if you fill out the Product Type first. The columns required will be highlighted in red; this will vary depending on the category.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%203.14.20%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/kxNimage.png)

**Additional Important Notes:**

- Once a listing has been created, it is not possible to alter the Seller SKU.
- The Standard Price and Quantity cells are not highlighted in red but they are still required.
- If the listing you’re creating is FBA, you don’t have to include Quantity. If it’s FBM, you can put “0” if you don't have the current data.
- If the Main Image is required, you'll need to make sure you add one. To add a photo, use the Amazon backend link, then copy and paste the image URL into the respective flat file cell.

3. On the “Update Delete” column, use Update.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%203.14.27%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/B5Uimage.png)

4. Under the Fulfillment Center ID, use AMAZON_NA (NA for North America, CA for Canada, MX for Mexico, etc) if it’s Fulfilled by Amazon and DEFAULT if it’s Fulfilled by Merchant. If it’s left blank, it will automatically be set to FBM.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%203.14.35%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/7B5image.png)

Important Note.
If it’s FBA, you have to fill out the compliance columns for the following:

- Pesticide Registration Status - This product is not a pesticide or pesticide device, as defined under the U.S. Federal Insecticide, Fungicide, and Rodenticide Act.
- Pesticide Marking - EPA registration number
- Is this product a battery or does it utilize batteries? - No
- Applicable Dangerous Goods Regulations - Not Applicable

5. Add “New” to the Item Condition column.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%203.14.43%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/uTZimage.png)

6. If the start date isn't provided on the task, set it to 1 year from the current date.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%203.14.52%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/49ximage.png)

7. Save the flat file and upload it on Seller Central. Go to Inventory > Add Products via Upload > Upload your Inventory File. Refresh the page first before attaching the file and make sure you’re on the right Amazon store. Then click Upload file.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%203.15.01%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/FRVimage.png)

8. Once the file has been uploaded and processed, let it sit for 15 minutes. It can take a little time for the file to finish processing. Download the processing report and troubleshoot as needed. Then, go to the Inventory page and search for the SKU you created and get the ASIN. Note: If you're creating a large number of listings you may download the all listing report and utilize Vlookup to get the ASINs.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%203.15.12%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/MWCimage.png)

9. Go back to the flat file you previously created and input the proposed SKU for FBM (if you created FBA prior) and fill out the required columns if we're creating new listings for the first time.

NOTE: Use the ASIN as the product ID when creating a duplicate listing.

For example, if you created the FBA listing first, use the generated ASIN from the FBA listing to create the FBM listing and vice versa.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%203.15.20%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/cJKimage.png)

10. Save the flat file and upload it on Seller Central as you did in steps 6 and 7. Make sure everything you review that every detail is correct.

11. After uploading, go to Manage All Inventory and search the ASINs to verify if two listing offers (FBM and FBA) are reflected.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-01-28%20at%203.15.28%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/1vuFREimage.png)

12. If the listings are showing up as suppressed, wait for a few minutes and then refresh the page. If it is still suppressed by then, click the suppressed button beside "Edit". It would route to a page that would tell you the information you need to edit to lift the suppression. Supply it, and save.

FAQs

**1. What is the purpose of creating both FBM and FBA offers for the same ASIN?**

- Creating both FBM (Fulfilled by Merchant) and FBA (Fulfilled by Amazon) offers for the same ASIN allows sellers to provide multiple fulfillment options for customers. This ensures flexibility in order fulfillment, helps manage inventory effectively, and maintains listing visibility even during stockouts in one fulfillment method.

**2. Why is it important to avoid creating both FBM and FBA offers simultaneously?**

- Creating both offers simultaneously can generate duplicate ASINs with the same UPC/GTIN/EAN, causing confusion and potential listing errors. It’s essential to create one offer first (FBA or FBM), generate the ASIN, and then use that ASIN to create the second offer to avoid conflicts.

**3. Which offer should I create first: FBM or FBA?**

- You can choose to create either offer first. However, the provided instructions recommend creating the FBA offer first and then using the generated ASIN to create the FBM offer.

**4. What should I do if I don’t have a client-provided SKU?**

- If the client hasn’t provided an SKU, suggest one based on their existing SKU structure. Ensure the SKU is unique and descriptive to prevent future confusion or errors, as the SKU cannot be changed once the listing is created.

**5. Are “Standard Price” and “Quantity” mandatory fields even if they are not highlighted in red in the flat file?**

- Yes, both fields are required. For FBA listings, leave the "Quantity" field blank, as Amazon manages inventory for FBA. For FBM listings, you can initially set the quantity to "0" if the stock isn’t available yet.

**6. What if the Main Image is required for the listing?**

- If a Main Image is required, upload the product photo to a hosting service (e.g., Amazon S3 or an image hosting platform). Copy the image URL and paste it into the appropriate cell in the flat file under the "Main Image URL" column.

**7. How do I handle compliance information for FBA listings?**

- For FBA listings, fill out the compliance-related columns, such as:

  - Pesticide Registration Status: “This product is not a pesticide or pesticide device.”
  - Pesticide Marking: Enter the EPA registration number if applicable.
  - Is this product a battery or does it utilize batteries? Select “No” or provide details if batteries are involved.
  - Applicable Dangerous Goods Regulations: Enter “Not Applicable” if no dangerous goods are involved.

**8. How should I handle the Item Condition and Start Date fields?**

- Item Condition: Set to “New” unless instructed otherwise.
- Start Date: If not provided in the task, set it to one year from the current date to avoid immediate suppression.

**9. What should I do if the listing gets suppressed after uploading?**

- If the listing is suppressed:

  - Wait a few minutes and refresh the page.
  - If still suppressed, click the “Suppressed” button beside “Edit.”
  - Follow the prompts to identify and correct any missing or inaccurate information.

**10. How can I efficiently retrieve the ASINs for multiple listings?**

- If you are creating a large number of listings, download the All Listings Report from Seller Central. Use VLOOKUP in Excel to match the generated SKUs to their corresponding ASINs.

**11. How do I ensure that my listings appear in the inventory after uploading?**

- After uploading the flat file:

  - Wait 15 minutes for processing.
  - Download the processing report to identify any errors and troubleshoot them.
  - Go to Manage All Inventory and search for the ASINs to confirm both FBM and FBA offers are reflected.

1**2. What should I do if I’m still experiencing errors after following all steps?**

- If errors persist, double-check the flat file for missing or incorrect information. Common errors include:

  - Incorrect fulfillment center IDs.
  - Missing mandatory fields (even if not highlighted).
  - Incorrect compliance data for FBA listings.
  - If issues remain unresolved, contact Amazon Seller Support for further assistance.
