# Catalog SOP: Creating an Archived Orders Report

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-creating-an-archived-orders-report

**Creating an Archived Orders Report**

**Who is this for :** This is designed for Amazon sellers seeking to download reports pertaining to their archived orders.

**Objective:** The purpose of this document is to offer a detailed, sequential guide on how to generate and download archived orders from the Amazon Seller Central page. This report serves as a comprehensive record and management system for previous orders, enabling sellers to access and review essential transaction details such as customer information, order dates, purchased items, and shipping information.

This feature assists sellers in tracking their sales history, managing customer inquiries, analyzing purchasing trends, and maintaining accurate financial records.

Instructions:

1. To generate a report containing archived orders, simply adhere to the following instructions: Begin by accessing the Seller Central Menu and selecting "Orders" from the initial dropdown menu. Next, proceed to choose "Order Reports" from the subsequent options.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1685151088138.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/wsBimage.png)

2. On the next page, go to the "Archived Orders" tab, and under the "Request Report" heading, select the month and year from the available drop-Down Menus. Then, click the "Request" button.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1685151694015.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/PdJimage.png)

3. Under the "Download Reports" heading further down the page, your requested report will appear at the top of the table with a Report Status of "Request Submitted" and a Download Action of "Not Ready".

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1685151953785.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/36cimage.png)

4. Click the "Refresh" button located at the top right of the table repeatedly until the Report Status changes to "Processing Complete" and a clickable link appears under the Download Action column, allowing you to download the report.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1685152155866.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/cMUimage.png)

FAQs

1. How long does it take for the archived orders report to be generated?

The processing time for an archived orders report varies depending on the number of orders in the selected time period. It usually takes a few minutes to an hour. If the report status remains “Request Submitted” for an extended period, try refreshing the page or requesting the report again.

2. In what format is the archived orders report downloaded?

The archived orders report is downloaded as a CSV (Comma-Separated Values) file, which can be opened in spreadsheet programs like Microsoft Excel or Google Sheets.

3. What should I do if the report request fails or does not appear?

If your report does not generate or is missing from the "Download Reports" section:

- Ensure that you selected the correct month and year.
- Refresh the page and check again.
- Try submitting a new request.
- If the issue persists, contact Amazon Seller Support for assistance.

4. How long are archived order reports available for download?

Archived order reports remain available for a limited time after they are generated. It is recommended to download and save them immediately, as Amazon may remove older reports after a certain period.

5. Can all Seller Central users access archived orders reports?

Access to archived order reports depends on user permissions. Only users with the necessary order management permissions will be able to generate and download reports. If you cannot access the report, check with your account administrator.

6. Can I request multiple archived orders reports at the same time?

Yes, you can request multiple reports for different months, but each report request must be submitted separately. There is no option to generate a report for multiple months in a single request.

7. Is there another way to access archived order data besides downloading the report?

Yes, sellers with developer access can retrieve order data using the Amazon Selling Partner API (SP-API). Additionally, some third-party seller tools may offer more advanced reporting options.

8. When might I need to generate an archived orders report?

You may need to generate an archived orders report for various reasons, including:

- Resolving customer disputes or inquiries.
- Reviewing sales history for accounting or tax purposes.
- Analyzing purchasing trends to optimize inventory management.
- Auditing shipping and fulfillment details.
