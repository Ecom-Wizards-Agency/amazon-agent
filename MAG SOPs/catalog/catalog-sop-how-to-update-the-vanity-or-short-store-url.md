# Catalog SOP: How to Update the Vanity or Short Store URL

Source: https://sop.myamazonguy.com/books/catalog/page/catalog-sop-how-to-update-the-vanity-or-short-store-url

## How to Update the Vanity or Short Store URL

**Who is this for :** This is intended for Amazon Sellers or associates who want to learn how to change short store URLs.

**Objective :** This SOP aims to provide you with the steps to alter or customize your store's vanity URL

**Note:** A vanity URL is only available for brand stores that have already been published.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-How-to-Update-the-Vanity-or-Short-Store-URL-1.png)

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/d9Qimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/d9Qimage.png)

- A vanity URL is a custom or personalized URL intended to be easy to remember and promote a specific brand.
- When a brand store is created, Amazon automatically generates a short store URL that is unique to every brand. In most cases, the brand name is affixed to it in this format: [https://www.amazon.com/store/page/BrandName](https://www.amazon.com/store/page/BrandName).
- There are some instances, however, that random characters are generated instead, which usually look like this: "[https://www.amazon.com/stores/page/AEI092UHKKKJKDS](https://www.amazon.com/stores/page/AEI092UHKKKJKDS)".
- It usually happens when the brand name contains unique characters like "& %, ( ) * @ !. Other times, we need to update the short URL if the client has specific preferences (upon request). Below is an example of an active brand store whose vanity URL needs updating.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/cKWimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/cKWimage.png)

**Steps in updating the vanity URL:**

1. Confirm with the brand manager or reach out to the client to know if the client has a preferred short URL in mind. You can also make suggestions at this point.

2. File a ticket under Stores by going to Stores → Manage Stores.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Catalog-SOP-How-to-Update-the-Vanity-or-Short-Store-URL-3.png)

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/T0mimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/T0mimage.png)

3. Click on the question mark beside the account name on the top right part of the screen.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/3otYfnimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/3otYfnimage.png)

4. Select Troubleshooting, Brand Management, Short URL for Stores.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/cuAimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/cuAimage.png)

5. It will bring you to the **Short URL for Stores page.**Take note of the current Store Link and your requested short link, then click **“Contact Us”** at the bottom of the page.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/JvQimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/JvQimage.png)

6. Provide a summary of the request. Do not forget to include the current store URL and the requested short URL. If prompted, include a screenshot of the problem.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/63Oimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/63Oimage.png)

7. Review and submit the ticket.

8. Check the status of the ticket or follow up daily until the desired URL reflects.

FAQs

1. **Can all brand stores update their vanity URL?**

- No, only published brand stores are eligible for a vanity URL update. If your store is still in draft mode, you must publish it first before requesting a change.

1. **How long does it take for Amazon to process a vanity URL update?**

- Amazon typically processes requests within 7–14 business days, but this can vary.
- If you do not see the update within two weeks, follow up with Amazon Support.

1. **What happens if Amazon rejects my vanity URL request?**

- If Amazon denies the request:

  - Ask for a reason and see if adjustments can be made.
  - Ensure the URL follows Amazon’s guidelines (avoid special characters, branding conflicts, or existing URLs).
  - Try submitting an alternative vanity URL.

1. **Will my old vanity URL still work after the update?**

- No, once the vanity URL is changed, the old link will not redirect automatically. Any external links (ads, social media, website links) should be updated to reflect the new URL.

1. **Does changing the vanity URL affect my store’s traffic or ranking?**

- Changing the URL itself does not impact Amazon search rankings, but if external links still point to the old URL, traffic may be affected.
- Always update all marketing materials that contain the old URL.

1. **What are the best practices for choosing a good vanity URL?**

- Keep it short and easy to remember.
- Use the exact brand name when possible (e.g., /store/MyBrand).
- Avoid special characters that Amazon may not support (&, %, @, !).
- Ensure it matches existing branding and marketing materials.

1. **What if the “Short URL” category is missing in the Support Center?**

- Clear your browser cache or try a different browser.
- Check if you have the necessary permissions (only brand-registered users can request a change).
- If the option is still missing, submit a ticket under a related category and request redirection to the correct team.

1. **Can I submit multiple URL requests if my first choice is unavailable?**

- Yes, if your preferred vanity URL is unavailable, you can resubmit another request with a different variation until an approved version is granted.
