# Advertising SOP: Uploading Backup File through Bulk Operations

Source: https://sop.myamazonguy.com/books/amazon-advertising/page/advertising-sop-uploading-backup-file-through-bulk-operations

## **Upload backup files through Bulk operations**

**Who is this for?**
For those who will upload backup files in Amazon Advertising Console

**Objective:**
This SOP aims to help individuals to upload bulk spreadsheets after campaign optimizations on a mass scale.

**Three main reasons why to upload bulk operations to the amazon advertising console?**

Answer 1: I wanted to update the changes I made from the Bulk spreadsheet after optimizing certain campaign bids

Answer 2: I wanted to upload the previous data of the specific timeframe of a specific campaign.

Answer 3. Create campaigns directly to Amazon advertising on a mass scale below is a template for auto-campaign creation.

> **IMPORTANT! Make sure you have all of the items checked before uploading:**
>
> - Before Copy-Pasting Data:
>
>   - Ensure **no filters** are selected when copying into a sheet with formulas.
>   - Remove any data where bids were not change before copying.
>   - Always use **Copy → Paste → Values** to prevent formula errors.
> - Formatting & Accuracy:
>
>   - Check **decimal place formatting** to avoid converting cents into whole dollars.
>   - Verify **daily budget decimal formatting** in the campaign column.
>   - Double-check **margins on new bids** (especially if using drag-to-copy).
> - Data Validation:
>
>   - Compare the **new sheet vs. the old sheet** to ensure bid changes were applied correctly.
>   - Verify that each keyword correctly corresponds to its intended bid before uploading.
>   - Ensure that no keywords are mismatched with incorrect bids.
>   - Ensure columns **R, S, and T** are **filtered to "Enabled"**.
> - Final Upload Process:
>
>   - Upload **every bulk file** to the **Asana task** before marking it complete.
>   - For **large bulk files**, upload to the **client’s folder** and paste the link in the Asana task comments.

FAQs

##### **FAQs**

- **Q1: What are the main reasons to upload bulk operations to the Amazon Advertising Console?**

  - **A1:**Here are three key reasons:

    - To update changes made from the bulk spreadsheet after optimizing specific campaign bids.
    - To upload historical data for a specific timeframe and campaign.
    - To create campaigns directly in the Amazon Advertising Console on a mass scale using a template for auto-campaign creation.
- **Q2: Where can I find the Bulk Operations page on Amazon Advertising Console?**

  - **A2:** You can access the Bulk Operations page at [Amazon Advertising Console Bulk Sheet Home Page](https://advertising.amazon.com/bulksheet/HomePage).
- **Q3: What types of files can be uploaded for bulk operations?**

  - **A3:** You can upload files that are saved from the Bulk Spreadsheet history. These files typically contain campaign data optimized for bulk updates.
- **Q4: How do I download a bulk spreadsheet?**

  - **A4:** Navigate to the Bulk Spreadsheet history in the Amazon Advertising Console, select the desired file, download it to your computer, and save it in a folder for later use.
- **Q5: What is the process for uploading a bulk file?**

  - **A5:** Follow these steps:

    - Go to the [Amazon Advertising Console Bulk Sheet Home Page](https://advertising.amazon.com/bulksheet/HomePage).
    - Click the “Choose file” button.
    - Select the bulk file saved on your computer.
    - Upload the file to the Bulk Operations page.
- **Q6: Can I create new campaigns using bulk operations?**

  - **A6:** Yes, you can create campaigns on a mass scale using a specific template for auto-campaign creation.
- **Q7: What should I do if there are errors during the upload?**

  - **A7:** If errors occur:

    - Review the error messages provided by the system.
    - Cross-check your bulk file for formatting issues or invalid data.
    - Make corrections and re-upload the file.
- **Q8: Are there any tips for organizing bulk spreadsheets?**

  - **A8:** Yes, consider the following tips:

    - Keep a well-organized folder for downloaded and optimized bulk spreadsheets.
    - Use consistent naming conventions to differentiate between timeframes and campaigns.
    - Backup your optimized files before uploading.
- **Q9: How often should bulk operations be used?**

  - **A9:** Use bulk operations whenever you need to:

    - Implement changes across multiple campaigns simultaneously.
    - Regularly update or maintain historical data.
    - Launch new campaigns efficiently on a larger scale.
