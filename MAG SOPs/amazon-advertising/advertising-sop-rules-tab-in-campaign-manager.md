# Advertising SOP: Rules Tab in Campaign Manager

Source: https://sop.myamazonguy.com/books/amazon-advertising/page/advertising-sop-rules-tab-in-campaign-manager

##### **Rules Tab in Campaign Manager**

**Steps**

**Step 1: Access the Rules Tab**

- Log in to your Amazon Ads account.
- Navigate to **Campaign Manager**.
- Under the **Targeting tab**, locate and open the **Rules Tab**.

  - Note: Not all accounts currently have access to this feature.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/eDwimage.png)

**Step 2: Understand Rule Types**

Amazon provides three types of rules:

- **Bid Rules** – Automate bid adjustments.
- **Budget Rules** – Automate budget increases/decreases based on schedules or performance.
- **Targeting Rules** – Automate addition of keywords or product targets (for Auto Campaigns with single-product targeting).

**Step 3: Create a Bid Rule**

- Click **Add Rule** → Select **Bid**.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/KVwimage.png)

- Choose **one or multiple campaigns**.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/u3Uimage.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/Vtpimage.png)

- Set **date range**: Daily or Weekly.

  - Option to run all day or set custom start/end times (dayparting).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/44bimage.png)

- Enter **bid adjustment** (% increase/decrease).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/Gxcimage.png)

- Assign a **rule name** for organization (e.g., “Bid Increase – Daily 10%”).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/eiVimage.png)

- Save the rule by clicking Add Rule.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/OfJimage.png)

**Step 4: Create a Budget Rule**

- Click **Add Rule** → Select **Budget**.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/mclimage.png)

- Select **one or multiple campaigns** (e.g., for scheduled/dayparting or performance-based strategies).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/6cyimage.png)

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2026-01/scaled-1680-/Vqlimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2026-01/Vqlimage.png)

- Choose **schedule type**:

  - Daily
  - Weekly
  - Custom (e.g., seasonal events like Father’s Day promo).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/Zg7image.png)

- Set **performance conditions** (optional):

  - Example: ACOS < 25%, ROAS > 3, CVR > 15%.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/FlPimage.png)

- Set **budget adjustment** (% increase/decrease).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/ajRimage.png)

- Assign a **rule name** for clarity (e.g., “Budget Boost – ACOS < 20%”).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/yY0image.png)

- Save the rule by clicking Add Rule.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/ypZimage.png)

**Step 5: Create a Targeting Rule (Auto Campaigns Only)**

- Click **Add Rule** → Select **Targeting**.

  - Note: Only available for **Auto Campaigns with single-product targeting**.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/RQSimage.png)

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2026-01/scaled-1680-/oOVimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2026-01/oOVimage.png)

- Add **Performance filters/criteria**:

  - Example: Clicks > 10 AND Orders > 1.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/AQUimage.png)

- - Lookback window (Performance Duration): up to **65 days**.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/ySIimage.png)

- Choose **target type**:

  - Keywords: Exact, Phrase, or Broad match.
  - Products: ASIN-based targeting.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/nqOimage.png)

- Set **bid adjustments** (percentage increase/decrease).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/dunimage.png)

- Once an ad group is selected, choose your targets.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/dxlimage.png)

- Assign a **rule name** (e.g., “Target Expansion – Auto Campaigns”).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/DUwimage.png)

- Save the rule by clicking Add Rule.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-09/scaled-1680-/WjMimage.png)
