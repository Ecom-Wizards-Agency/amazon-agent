# Advertising SOP: Bulk Negation Tool

Source: https://sop.myamazonguy.com/books/amazon-advertising/page/advertising-sop-bulk-negation-tool

## **Bulk Negation Tool**

**Who is this for :** Individuals who want to save time and increase efficiency by doing bulk negations.

**Objective :**This SOP aims to provide a step-by-step guide on how to generate a Bulk File ready for upload for Customer Search Terms to be negated based on the parameters that you can set (Clicks, Spending, and Orders)

> **IMPORTANT : Before Running your results, Delete the “Retailer” column (G) for the tool to compute the results.**

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Uawimage.png)

**Bulk Negation Tool:** [File](https://drive.google.com/file/d/117nWNmzMk0tovYXVNFOB-LJroyzfRN1j/view?usp=drive_link)

- **Download:**

  - Generate a Sponsored Products Search Term Report (Chosen date range)
  - Bulk File (Any date range)

    ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/w16image.png)

- **Activate Bulk Negation Tool.xlsm Macro**

  - For Windows: Right click > Properties > Check "Unblock" > Click OK

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202024-04-03%20at%203.55.04%E2%80%AFAM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/uFcimage.png)

- **For Mac: Open the file > Enable Macros**

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/h30image.png)

- **Open the files:**

  - The Bulk Negation Tool.xlsm
  - The Search term report.xlsx
  - The Bulk File.xlsx
- **Paste the files**

  - The SP Search term report to STR-Dump sheet
  - The SB Search term report to STR-Dump-SB sheet
  - The Bulk File to Bulk-Dump sheet
- **On Negation sheet, set:**

  - Orders
  - Spend
  - Clicks
  - ACOS

    - Sample settings:

      ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Pi0image.png)
    - Note: Make sure we consider the product price relation to the ad spend when setting this up

      - Ex: if product price is $50 and ACOS goal 40%, the approx. input should be $25 and more
- **Click the Run button**

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/fYWimage.png)

**For SP**

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/mYrimage.png)

**For SB**

- **Validate the Results, next is Copy then Paste to Negation sheet:**

  - Campaign Name
  - Adgroup Name
  - Customer Search Term
- **Choose between Exact or PhraseNegative match types**

  - Filter Negative type column to KW to easily fill out Negative match type column
  - PT must be blank on Negative match type column
- **Copy the Data from SP sheet then paste on a different excel as values**

  - SP - Column A to AH
  - SB - Column A to Y

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/5xCimage.png)

- **Save the new Excel and upload it to Bulk operations**

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/NvGimage.png)

- **Others**

- **Clear ALL**

  - This will clear these sheets:
  - STR-Dump
  - STR-Dump-SB
  - Bulk-Dump
  - Results
  - Negation

- **Optional Clear**

  - This will clear the same sheets but there will be a pop-up asking to clear each sheet or not

FAQs

##### **FAQs**

- **Q1: What are the prerequisites before starting?**

  - **A1:**Files Needed:

    - Sponsored Products (SP) Search Term Report (chosen date range).
    - Bulk File (any date range).
    - **Bulk Negation Tool.xlsm**.
  - **To Activate the Macro:**

    - **For Windows:** Right-click the file, go to **Properties**, check "Unblock," and click OK.
    - **For Mac:** Open the file and enable macros.
- **Q2: What are the additional options for clearing sheets?**

  - **A2: Clear ALL:**

    - Clears data from the following sheets:

      - **STR-Dump**
      - **STR-Dump-SB**
      - **Bulk-Dump**
      - **Results**
      - **Negation**
    - **Optional Clear:**

      - Allows selective clearing of sheets with confirmation pop-ups for each.
- **Q3: What should I keep in mind while setting up the tool?**

  - **A3:**Align spend parameters with product price and ACOS goals to ensure effective negation.
  - Validate results thoroughly before uploading the file to avoid errors.
- **Q4: What match types are available for negation?**

  - **A4: Exact Match**: Prevents the specific search term from triggering your ads.
  - **Phrase Negative Match**: Blocks search terms containing the specified phrase.
- **Q5: How do I handle SP and SB data separately?**

  - **A5:**SP data goes to the **STR-Dump** sheet, and SB data to the **STR-Dump-SB** sheet.
  - Ensure results are copied into separate columns before saving and uploading.
