# Advertising SOP: DEDUPLICATOR TOOL

Source: https://sop.myamazonguy.com/books/amazon-advertising/page/advertising-sop-deduplicator-tool

## **DEDUPLICATOR TOOL**

**Who is this for?**

- This is for Amazon Advertising Specialists who need to check the metrics of specific target/s and verify if a specific keyword or a set of keywords have been targeted on any active campaigns based on a specific match type.

**Description: This SOP will help you outline the steps of using the Deduplicator Tool and efficiently audit the files, information, including links needed:**

**Before you begin, make a copy of**[Deduplicator Tool](https://docs.google.com/spreadsheets/d/1g9tkoscLx9oQGwiRVN452XR7HuqDs-wBtR0R3WXNFQw/copy)

**This guide contains two parts:**

**Section 1:** Generating data for the tool

**Section 2:** Using the Deduplicator tool

FAQs

Generating Data

- **Q1: How do I generate data for the Deduplicator Tool?**

  - **A1:**

    - Generate and download the bulk file with the required inclusions for any date range.
    - Open the bulk file and go to the Sponsored Product Campaigns.
    - Apply filters to Column B (Entity) and select "KEYWORD," "PRODUCT AD," and "PRODUCT TARGETING."
    - Follow the replacement and pasting instructions to copy data into the SP BULK DUMP sheet of the FULL PPC AUDIT AND DEDUPLICATOR TOOL.
- **Q2: What additional steps are needed for Sponsored Brand Campaigns?**

  - **A2:**

    - Filter Column B (Entity) to include "CAMPAIGN," "KEYWORD," and "PRODUCT TARGETING."
    - Filter Column O (State) to include "ENABLED," and Column Q (Campaign Serving Status) to include "RUNNING."
    - Perform keyword replacements in Column W and handle ASINs in Column AL according to the instructions.
    - Paste the data as VALUES into the SB BULK DUMP sheet of the FULL PPC AUDIT AND DEDUPLICATOR TOOL.

Using the Deduplicator Tool

- **Q3: How do I use the Deduplicator Tool?**

  - **A3:**

    - Open the DEDUPLICATOR sheet of the FULL PPC AUDIT AND DEDUPLICATOR TOOL.
    - Select the match type to check using the drop-down in Cell L1.
    - Paste the keywords to be checked as VALUES in Cell L3.
    - Review the numerical values in Columns J (SP CHECKER) and K (SB CHECKER) to determine the keyword usage count.
- **Q4: What does the data in Columns J and K represent?**

  - **A4:**

    - The values indicate how often a keyword has been used for the selected match type.
    - A "0" means the keyword was not used, while other numbers indicate the frequency of use.

Data Verification

- **Q5: How can I check the data per ASIN?**

  - **A5:** Utilize filters in cells B2 to H2 to analyze the data for each ASIN.
