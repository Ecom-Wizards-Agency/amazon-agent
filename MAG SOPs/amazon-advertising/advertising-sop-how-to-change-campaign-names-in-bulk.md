# Advertising SOP: How to Change Campaign Names in Bulk

Source: https://sop.myamazonguy.com/books/amazon-advertising/page/advertising-sop-how-to-change-campaign-names-in-bulk

## **How to Change Campaign Names in Bulk**

**Who is this for?**

Individuals who have a lot of campaigns and want to change their names in bulk.

**Objective:**

The purpose of this document is to explain how to change campaign names in bulk.

**Overview:**

1. Download the Bulk Sheet
2. Filter the campaigns
3. Change the name of the Campaigns and AdGroups
4. Upload the Bulk Sheet back

**STEP 1: Download the Bulk Sheet**

- Download the Bulk Sheet using the new spreadsheet format. Make sure that Terminated Campaigns and Brand Assets Data remain checked and select 30 days date range.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/THlimage.png)

- After we download the Bulk file, we need to filter the campaigns to have a better understanding of the active Campaigns and Ad Groups. We also need to have the Renaming sheet ready.

**STEP 2: Filter the campaigns**

- Open the Bulk file click on "filter" and filter the sheet by "Entity" (Column B).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/rEJimage.png)

- Select "Ad Group" and "Campaign" only.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/1s4image.png)

**STEP 3: Change the name of the Campaigns and Ad Groups**

- Change the name of the Campaigns and Ad Groups. After you filter the Column, you can change the name of the Campaign and the Ad Group.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1712341844213.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/WWAimage.png)

- Change the name of the Campaign
- Change the Ad Group name (1 row below the campaign name).
- Update column C after you change the name of all campaigns and ad groups. You need to name Each Row on column C with “Update” otherwise the changes will not take effect.

**Note:**

- Make sure that you use a renaming sheet to have all the old names and new names ready before you start
- Use Search (Control + F) to search the old campaign name and make sure that you will change the name of the campaign on the correct row.
- Don’t forget to fill column C with the "update" word. Otherwise, your campaign names won't be updated.

> **⚠️IMPORTANT! Make sure you have all of the items checked before uploading:⚠️**
>
> - **Before Copy-Pasting Data:**
>
>   - **Ensure** **no filters** **are selected when copying into a sheet with formulas.**
>   - **Remove any data where bids were not change before copying.**
>   - **Always use** **Copy → Paste → Values** **to prevent formula errors.**
> - **Formatting & Accuracy:**
>
>   - **Check** **decimal place formatting** **to avoid converting cents into whole dollars.**
>   - **Verify** **daily budget decimal formatting** **in the campaign column.**
>   - **Double-check** **margins on new bids** **(especially if using drag-to-copy).**
> - **Data Validation:**
>
>   - **Compare the** **new sheet vs. the old sheet** **to ensure bid changes were applied correctly.**
>   - **Verify that each keyword correctly corresponds to its intended bid before uploading.**
>   - **Ensure that no keywords are mismatched with incorrect bids.**
>   - **Ensure columns** **R, S, and T** **are** **filtered to "Enabled".**
> - **Final Upload Process:**
>
>   - **Upload** **every bulk file** **to the** **Asana task** **before marking it complete.**
>   - **For** **large bulk files, upload to the** **client’s folder** **and paste the link in the Asana task comments.**

**STEP 4: Upload the Bulk Sheet back**

- After you complete the first 3 steps, save the file and upload it to the advertising console. Make sure that the update will finish successfully and that you will have all the rows processed.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202024-08-02%20at%2010.11.06%E2%80%AFPM(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/kEkimage.png)

Congratulations! You managed to change the names of the campaigns in bulk.

FAQs

##### **FAQs**

- **Q1: How do I change the names of campaigns and ad groups?**

  - **A1:**Once the campaigns and ad groups are filtered, change the name of the campaign and its corresponding ad group. Make sure to update column C with "Update" for each row.
- **Q2: Why do I need to update column C with the word "Update"?**

  - **A2:**Column C needs to be filled with "Update" to ensure the changes will take effect. Without it, your campaign names won't be updated.
- **Q3: What should I do before changing the campaign names?**

  - **A3:**Make sure you use a renaming sheet that contains both the old and new names. Use the search function (Control + F) to find the old campaign names and confirm you are changing the correct rows.
- **Q4: What do I do after making the changes?**

  - **A4:**After making the necessary changes, save the file and upload it to the advertising console. Ensure the update process completes successfully, and all rows are processed.
- **Q5: What should I do if the update fails?**

  - **A5:**Ensure that all steps were followed correctly, particularly updating column C with "Update." Re-upload the file if needed, and check for any errors or incomplete rows.
