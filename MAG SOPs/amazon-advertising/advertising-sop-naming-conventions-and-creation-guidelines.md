## **Naming Conventions and Creation Guidelines**

**Who is this for?**: This is for the digital marketing team responsible for creating and managing Sponsored Products, Sponsored Brands, and Sponsored Display campaigns on various e-commerce platforms.

**Objective**: This aims to provide guidelines for naming conventions and the creation of ad groups for Sponsored Products, Sponsored Brands, and Sponsored Display campaigns. The document outlines specific instructions for naming ad groups, using new match types, and targeting options.

FAQs

Campaign Naming Conventions

- **Q1:** How are campaign names automatically generated?

  - **A1:** Input the description of the target product and its ASIN, and the system will automatically generate appropriate campaign names based on the campaign type and matching.
- **Q2:** What should I do if I find duplicate campaign names?

  - **A2:** Manually check for duplicate campaign names and append a number to the end of the name to distinguish between similar campaigns.
- **Q3:** Are there any specific formatting rules for the campaign names?

  - **A3:** Yes, ensure that ad groups contain ASIN(s), there are spaces between dashes, and numbers appear at the end of the name (e.g., `SP - KW - Broad - Red Roses - B0XXXXXXXX - MAG - 1`).

Sponsored Products (SP)

- **Q4:** What is the naming convention for Sponsored Products with automatic targeting?

  - **A4:** For automatic targeting, the name should be formatted as:
    `SP - Auto - Red Roses - B0XXXXXXXX - MAG`.
- **Q5:** How do I name campaigns targeting specific keyword match types in Sponsored Products?

  - **A5:** The naming conventions for keyword match types are as follows:

    - Broad: `SP - KW - Broad - Red Roses - B0XXXXXXXX - MAG`
    - Phrase: `SP - KW - Phrase - Red Roses - B0XXXXXXXX - MAG`
    - Exact: `SP - KW - Exact - Red Roses - B0XXXXXXXX - MAG`
    - Mixed: `SP - KW - Mixed - Red Roses - B0XXXXXXXX - MAG`
- **Q6:** How should I name campaigns for competitor brand keywords?

  - **A6:** For competitor brand targeting, use:

    - Broad: `SP - KW - COMP - Broad - Red Roses - B0XXXXXXXX - MAG`
    - Phrase: `SP - KW - COMP - Phrase - Red Roses - B0XXXXXXXX - MAG`
    - Exact: `SP - KW - COMP - Exact - Red Roses - B0XXXXXXXX - MAG`
    - Mixed: `SP - KW - COMP - Mixed - Red Roses - B0XXXXXXXX - MAG`
- **Q7:** How should campaigns be named for ASIN and category targeting?

  - **A7:** For ASIN and category targeting, use the following formats:

    - ASIN targeting: `SP - ASIN - SELF - Red Roses - B0XXXXXXXX - MAG`
    - Category targeting: `SP - CAT - Red Roses - B0XXXXXXXX - MAG`

Sponsored Brands (SB)

- **Q8:** How should I name Sponsored Brand video campaigns targeting specific keyword match types?

  - **A8:** The naming convention for Sponsored Brand video campaigns with keyword match types is:

    - Broad: `SB - Video - KW - Broad - Red Roses - B0XXXXXXXX - MAG`
    - Phrase: `SB - Video - KW - Phrase - Red Roses - B0XXXXXXXX - MAG`
    - Exact: `SB - Video - KW - Exact - Red Roses - B0XXXXXXXX - MAG`
- **Q9:** How should I name campaigns targeting competitor ASINs in Sponsored Brands?

  - **A9:** For campaigns targeting competitor ASINs in Sponsored Brands, use:
    `SB - Video - ASIN - COMP - Red Roses - B0XXXXXXXX - MAG`
- **Q10:** What is the naming convention for Sponsored Brand Headline campaigns?

  - **A10:** For Sponsored Brand Headline campaigns, use the following formats:

    - Keyword match types: `SB - KW - Broad - Red Roses - B0XXXXXXXX - MAG`
    - ASIN targeting: `SB - ASIN - Red Roses - B0XXXXXXXX - MAG`

Sponsored Display (SD)

- **Q11:** How should I name Sponsored Display campaigns targeting product categories or ASINs?

  - **A11:** Use the following formats for Sponsored Display product targeting campaigns:

    - Category targeting: `SD - PT - VCPM - CAT - Red Roses - B0XXXXXXXX - MAG`
    - ASIN targeting: `SD - PT - VCPM - ASIN - Red Roses - B0XXXXXXXX - MAG`
- **Q12:** What is the naming convention for Sponsored Display audience targeting campaigns?

  - **A12:** For audience targeting campaigns, use these formats:

    - AMZ AUD: `SD - AUD - VCPM - AMZ AUD - Red Roses - B0XXXXXXXX - MAG`
    - Views RM: `SD - AUD - CPC PV - Views RM - Red Roses - B0XXXXXXXX - MAG`
    - Purchase RM: `SD - AUD - CPC - Purchase RM - Red Roses - B0XXXXXXXX - MAG`
- **Q13:** How should I name Sponsored Display Video campaigns targeting audiences?

  - **A13:** For Sponsored Display Video campaigns targeting audiences, use:
    `SD - Video - AUD - VCPM - AMZ AUD - Red Roses - B0XXXXXXXX - MAG`
