# General SOP: How to Make HTML Tags

Source: https://sop.myamazonguy.com/books/general/page/general-sop-how-to-make-html-tags

## **How to Make HTML Tags**

**Who is this for?**This is intended for individuals who are new to HTML programming or are interested in learning how to create HTML tags for formatting HTML content.

**Objective**: This Standard Operating Procedure (SOP) provides instructions on how to create HTML tags using an HTML converter tool and provides a list of common HTML tags that can be used to format HTML content.

Use an HTML converter to help you create the tags. I prefer to use codebeautify.org/htmlviewer. This tool has features that streamline the process. For example, it automatically creates the closing tags once you have opened them.

**Below is a list of HTML tags to use to create an HTML Format:**

<h1></h1> - Headline of the article

<h2></h2> - Secondary headline

<h3></h3> - Generic headline to use inside of a secondary headline

<p></p> - New paragraph

<ol></ol> - Numbered list - each value in the list must start with <li> and end with </li>

<ul></ul> - Generic bullet points - each bullet point must start with <li> and end with </li>

<strong></strong> - use to bold a section of text

<a href="example link"> link text</a> - How to create a hotlink

There should be no spaces or indentations between tags.

*Example*:

Headline 1

Body Paragraph

Headline 2

Body Paragraph

Headline 2

Body Paragraph

1. Item1

2. Item2

3. Item3

Headline 2

Body Paragraph

Headline 3

-Item

-Item

-Item

HTML:

<h1>Headline 1</h1><p>Body Paragraph</p><h2<Headline 2</h2><p>Body Paragraph</p><h2>Headline 2</h2><p>Body Paragraph</p><ol><li>Item1</li><li>Item2</li><li>Item3</li></ol><h2>Headline 2</h2><p>Body Paragraph</p><h3>Headline 3</h3><ul><li>Item</li><li>Item</li><li>Item</li></ul>

FAQs

Using an HTML Converter Tool

- **Q1: How do I use**[**codebeautify.org/htmlviewer**](http://codebeautify.org/htmlviewer)**for creating HTML tags?**

  - **A1:** To use [codebeautify.org/htmlviewer](http://codebeautify.org/htmlviewer), simply input your desired content into the tool, and it will generate HTML tags automatically. It also provides features like auto-closing tags, which streamline the process of writing HTML code.
- **Q2: Are there other HTML converter tools I can use besides**[**codebeautify.org/htmlviewer?**](http://codebeautify.org/htmlviewer?%EF%BF%BCA2)

  - **A2:** Yes, there are other tools like Online HTML Editor, HTML Formatter, and W3Schools HTML Editor. These tools offer similar functionalities to [codebeautify.org](http://codebeautify.org) and can help simplify the process of creating and editing HTML.

Formatting with HTML Tags

- **Q3: What happens if I forget to include a closing tag?**

  - **A3:** Omitting a closing tag can cause formatting errors in your HTML document, as the browser may not interpret the structure correctly. Most modern HTML editors, including [codebeautify.org](http://codebeautify.org), will alert you to missing closing tags or automatically add them for you.
- **Q4: Can I use spaces or indentations within tags?**

  - **A4:** No, spaces or indentations should not be included within HTML tags. For example, `< h1 >` is incorrect, while `<h1>` is correct. However, you can use spaces or indentations outside tags to organize your code for readability.

HTML Tags and Their Usage

- **Q5: Can I use multiple**`<h1>`**tags in one document?**

  - **A5:** While technically possible, using multiple `<h1>` tags is not recommended as it can negatively impact SEO and accessibility. Use `<h2>`, `<h3>`, and other heading tags for additional sections.
- **Q6: What is the difference between**`<ol>`**and**`<ul>`**tags?**

  - **A6:** `<ol>` creates a numbered (ordered) list, while `<ul>` creates a bullet-pointed (unordered) list. Both require `<li>` tags for individual items in the list.
- **Q7: How can I add links within my HTML content?**

  - **A7:** To add a link, use the `<a>` tag in the format `<a href="example link">link text</a>`. Replace "example link" with the URL and "link text" with the text you want displayed as the link.

Troubleshooting HTML Formatting

- **Q8: Why isn’t my HTML formatting displaying correctly on a webpage?**

  - **A8:** Check for common errors such as missing or mismatched tags, improper nesting, or spaces within tags. Tools like [codebeautify.org/htmlviewer](http://codebeautify.org/htmlviewer) can help identify and correct these errors.
- **Q9: How can I test my HTML after creating it?**

  - **A9:** You can copy and paste your HTML code into a browser to view how it renders. Alternatively, use online tools like W3Schools' Tryit Editor to preview your HTML and test its functionality.
- **Q10: Can I format text beyond the basic tags provided in the SOP?**

  - **A10:** Yes, you can use additional HTML tags like `<em>` for emphasis (italic), `<span>` for inline styling, or `<div>` for larger sections. Be sure to follow best practices for HTML structure and styling.
