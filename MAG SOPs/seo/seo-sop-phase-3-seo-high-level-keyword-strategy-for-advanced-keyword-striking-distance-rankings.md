# SEO SOP: Phase 3 SEO - High Level Keyword Strategy for Advanced Keyword Striking Distance Rankings

Source: https://sop.myamazonguy.com/books/seo/page/seo-sop-phase-3-seo-high-level-keyword-strategy-for-advanced-keyword-striking-distance-rankings

## **Phase 3 SEO : High-Level Keyword Strategy for Advanced Keyword Striking Distance Rankings**

**Who is this for?**

- For MAG employees responsible for managing and optimizing backend search terms using MAG’s 3rd SEO strategy.

**Objective:**

- This phase is for mature listings (6+ months old generally) that index for 800+ organic keywords. This phase identifies keywords ranking organically 20-50 and attempts to improve them to rank between 1-19.

**Scope:**

- Redo current backend search terms for mature listings. Must include all variations in a parentage.
- Do this when you have already applied Pink Terms Optimization (SEO Phase 2), or if the parent ASIN has 800+ organic keywords distribution, regardless of the current SEO phase (with the Brand Manager’s approval).
- Must have 50 words in ranks 20-50 before starting.
- Rotate keywords every 90 days; must align with [Phase 4 SEO goals](https://sop.myamazonguy.com/books/seo/page/seo-sop-phase-4-intent-based-strategy-leveraging-conversion-and-click-through-rates-for-market-share-expansion)

  - Phase 3R1 > Phase 4 Round 1A > Phase 4 Round 1B
  - Phase 3R2 > Phase 4 Round 2A > Phase 4 Round 2B
  - Phase 3R3 > Phase 4 Round 3A > Phase 4 Round 3B

    - (The phase cycle will repeat until further decisions are made by the client or Brand Managers)

**Media Learning**:

**Procedure:**

- Create a spreadsheet using this sheet as a template:

  - [Client Name - Phase 3 SEO](https://docs.google.com/spreadsheets/d/<google-sheet-id>/edit?usp=sharing)
- Make sure that you create the spreadsheet in the Clients folder on Google Drive.
- Login to Helium10 and choose the Cerebro tool (reverse ASIN lookup).
- Input your ASIN and click “Get Keywords.”

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/ScPimage.png)

- If the ASIN has fewer than 800 organic keywords that it indexes for, this strategy won’t work. Update your spreadsheet by putting the organic keyword count and move to the next ASIN.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/x2Oimage.png)

- If the ASIN has 800 above organic keyword count then do the following steps.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/sOKimage.png)

- Be sure to set some filters before exporting the data, including:

  - Match Type - should always be organic.
  - Word Count - input 1.
  - Then hit “Apply filters.”

    ![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/UMrimage.png)

- Scroll a little bit down to see a table containing the comprehensive data. Sort the search volume in descending order. And export as a CSV file.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/PYuimage.png)

- Access the downloaded CSV file. In your spreadsheet, select “File” then “Import.”

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/pfJimage.png)

- Upload the downloaded CSV file to your spreadsheet.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/LuMimage.png)

- Select “Insert new sheet(s)” and hit “Import Data”.

  - This avoids replacing the current data

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/jJMimage.png)

- Delete some unnecessary columns except for the columns for Keyword Phrases, Search Volume and Organic Rank.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Ayhimage.png)

- In your spreadsheet, hit the “Data” tab and select “Create a filter.” Filter by condition the column C (Organic Rank). Select “In between” and input “20-50” as shown in the figure below. (Skip this step if you already applied the 20–50 Organic Rank filter in Cerebro before exporting.

  The downloaded file will already reflect only the keywords within that rank range.)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/vRQimage.png)

- Then, make sure that column B (Search Volume) is sorted by descending order.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/OYfimage.png)

- Go to the H10 Keyword Processor (formerly Frankenstein) tool. From your spreadsheet, copy some keywords from the highest search volume and paste them into the Keyword Processor tool.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/bN6image.png)

- Select relevant keywords from the Result (right side of the Keyword Processor).
- Once you have keywords with 246-250 characters (spaces included), paste them to your spreadsheet, Proposed Search Terms column.

  - Search Terms = Take advantage of misspellings, Spanish, etc.

**Note:** Stay under the length limit, which ranges from 246-250 characters, NOT bytes. Amazon has recently become stringent about this process and is now counting white spaces.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/L2himage.png)

- Highlight the phrases used in your search terms with bright yellow.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/9FZimage.png)

- Check the listing if it also has other search attributes (Subject Matter, Intended Use, Target Audience and Other Attributes).

  - If it has, update the other search attributes, copy some keywords again from your spreadsheet and paste it into the Keyword Processor.
  - Select relevant keywords for those attributes.

    - Other search attributes are not as impactful.
    - They should be filled out at the very least.
  - Target Audience - Target Demographics
  - Subject Matter - Occasions
  - Used For/ Intended Use - Common Sense (What is the product used for?)
  - Other Attributes = Features (10 oz, Size, etc)
- Copy the selected keywords to your spreadsheet.

  - Subject Matter – 50 characters (spaces counted)
  - Intended Use – 100 characters (space counted)
  - Target Audience – 50 characters (space counted)
  - Other Attributes – 50 characters (space counted)
- Highlight the phrases used in your other search attributes with light yellow.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/M9Timage.png)

NOTE: **Phase 3 SEO**should be done every 90 days to check for Strikezone keywords. Moreover, Phase 3 - **should not be done** until 50 keywords are in ranks 20-50. Re-run phase 2 or 1 SEO if conditions are not met to do phase 3 and it's been 60 days since phase 2 was run. The goal is to matriculate keywords from the strike zone to the top of the page.

FAQs

General Requirements

- **Q1: What criteria must an ASIN meet to qualify for Phase 3 SEO?**

  - **A1:** The ASIN must index for 800+ organic keywords and have at least 50 keywords ranked between 20-50. If these conditions are not met, you must re-run Phase 1 or 2 SEO if it has been 60 days since their last implementation.
- **Q2: How frequently should Phase 3 SEO optimizations be performed?**

  - **A2:** Phase 3 SEO should be conducted every 90 days to review and optimize Strike Zone keywords.
- **Q3: Can Phase 3 SEO be applied without completing Phase 2 optimization?**

  - **A3:** No, Phase 3 SEO assumes Phase 2 optimization has been completed, except when the ASIN already indexes for 800+ organic keywords and approval has been obtained from the Brand Manager.

Process Guidelines

- **Q4: What filters should be applied in Helium10’s Cerebro tool during keyword research?**

  - **A4:** Set the Match Type filter to “Organic” and input “1” for Word Count before applying the filters. Ensure that the search volume is sorted in descending order.
- **Q5: How are the exported CSV files handled in the spreadsheet?**

  - **A5:** The CSV file should be imported as a new sheet in the client-specific Google Drive folder. Unnecessary columns should be deleted, keeping Keyword Phrases, Search Volume, and Organic Rank. Use filters to identify keywords ranked between 20-50.
- **Q6: What is the character limit for Search Terms, and how is it calculated?**

  - **A6:** Search Terms must stay within 246-250 characters, including spaces. Amazon is strict about this limit and counts white spaces.

Keyword Selection and Optimization

- **Q7: How are keywords selected for inclusion in Search Terms?**

  - **A7:** Keywords are selected from the highest search volume list in the spreadsheet and processed in the H10 Keyword Processor tool. Prohibited keywords are filtered out using the Prohibited Keyword Tool.
- **Q8: Are other search attributes like Subject Matter, Intended Use, and Target Audience mandatory?**

  - **A8:** Yes, these attributes should be updated using relevant keywords. They may not have as much impact as Search Terms but are still essential to complete.

Troubleshooting and Recommendations

- **Q9: What should be done if fewer than 50 keywords are ranked between 20-50?**

  - **A9:** Re-run Phase 1 or 2 SEO if it has been 60 days since the last optimization, or wait until the conditions for Phase 3 are met.
- **Q10: What happens if the character limit for Search Terms is exceeded?**

  - **A10:** Exceeding the character limit may cause Amazon to reject or truncate the search terms. Always ensure compliance with the specified limits to avoid issues.
- **Q11: How does Phase 3 SEO align with Phase 4 goals?**

  - **A11:** Phase 3 SEO prepares keywords in the Strike Zone (ranks 20-50) to matriculate into the top rankings (1-19), directly feeding into Phase 4 goals of maintaining and optimizing top-ranking keywords.
