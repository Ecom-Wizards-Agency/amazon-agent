# **Create Meta Description Using Master Keyword List (MKL) + How to Set the Meta Description of Any Page in a Brand Store**

**Who is this for?**To individuals seeking to further increase their brand's visibility.

**Objective:**To boost your search engine rankings and get higher visibility for your brand name and associated non-brand keywords, both on and off Amazon.

**Actual Sample worksheet:**[**Age of Sage - Meta Description**](https://docs.google.com/spreadsheets/d/1dpUOegV1zyED6i0kJRhvCxvZWjSUaNAHtnzjR-JkJPo/edit?usp=sharing)

**What is a meta description?**

A meta description is a brief summary that describes the content of a web page. It is displayed as part of the search snippet in a search engine results page (SERP) and is meant to give the user an idea of the content that exists within the page and how it is related to their search query.

On the left side of the page, click on any of the store pages to load them. Once it loads, click on the page name again, and the text field for the meta description will appear. See Figure 2.

Next, optimize all pages that doesn’t have high ranking or high search volume keywords. Make a note and add them to your worksheet. You may use the sample worksheet and template provided above.

**II. Create the Meta Description**

To get started, understand the page you are trying to create the meta description for. For example, if it is a water tumbler product, locate the Master Keyword List (MKL) for water tumbler products on the SEO worksheet and make use of the keywords with high search volume and relevancy. The same thing applies if you are creating the meta description for other pages with different product listing categories.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/1b1image.png)

*Figure 3.*

**Here are some tips on creating the right meta description:**

- *On H10 Frankenstein, process the first 100 keyword phrases; get the keyword ideas from the filtered keywords*
- *Keep it up to 150-160 characters as much as possible*
- *Use high-ranking keywords that focus on the category you are targeting*
- *Ensure that your meta description matches the content of your store page*
- *Write in an active voice*
- *Include a call to action wherever necessary*
- *Highlight the listing or category you are working on especially if they are category-specific*

**III. QA and Approval**

- Submit the meta description draft for QA.
- Once QA is completed, POC should deliver to client for approval.
- Once approved, Make sure that a screenshot proof of approval /disapproval or fireflies call link is commented in Asana Tasks.
- Make sure all resources SEO/Copy/PSD works are placed in respective client's folders.

**IV. Publish**

Sign in to the client’s Seller Central account, input the meta description for that page, and publish.

It normally takes 2-3 days for Amazon to review the changes; make sure to monitor. The meta description will reflect automatically once approved.

FAQs

**1. Why are meta descriptions important for SEO and brand visibility?**

- Meta descriptions improve click-through rates (CTR) on search engines:

  - Helping users understand what the page is about before clicking.
  - Boosting visibility for non-brand keywords related to the product.
  - Encouraging engagement with a compelling call to action.
  - While Amazon does not use meta descriptions for internal search rankings, they affect external search engine visibility.

**2. How do I decide which pages to optimize first?**

- Prioritize pages that:

  - Have high search potential but are underperforming in clicks or conversions.
  - Feature top-selling or flagship products.
  - Have outdated or missing meta descriptions.
  - Focus on seasonal or promotional campaigns that need more visibility.

**3. Should I update existing meta descriptions or only add new ones?**

- If the existing meta description is well-optimized and driving traffic, leave it unchanged.
- If the page has low CTR or outdated keywords, update it with high-ranking keywords from the Master Keyword List (MKL).
- Run A/B testing by tracking performance before and after changes.

**4. What are common mistakes to avoid when writing a meta description?**

- Using vague or generic descriptions (e.g., “Best products available now”).
- Stuffing keywords unnaturally, making it unreadable.
- Exceeding 160 characters, which gets cut off in search results.
- Not matching the description with the actual content on the page.

**5. How do I track the performance of a meta description after publishing?**

- Monitor organic search traffic from tools like Google Search Console.
- Check for changes in CTR on Amazon if external traffic is a key driver.
- Use third-party SEO tools to track keyword rankings.
- If traffic does not improve after 30–60 days, consider re-optimizing.

**6. What should I do if Amazon does not approve my meta description update?**

- Ensure the content follows Amazon’s policies (avoid claims, promotional language, and subjective terms).
- Double-check that no prohibited keywords are used.
- Submit a Seller Support ticket if the update does not reflect after 3 days.

**7. Do meta descriptions affect Amazon’s internal search ranking?**

- No, Amazon’s search algorithm does not use meta descriptions for ranking. However, they impact Google and other search engine rankings, driving external traffic to the store.

**8. Can I use the same meta description across multiple pages?**

- No, each page should have a unique meta description:

  - Category pages should target broader, high-volume keywords.
  - Product pages should focus on specific product attributes.
  - Brand pages should highlight brand identity and key selling points.

**9. What if the meta description field is not visible in Seller Central?**

- Try accessing a different page to see if the issue is page-specific.
- Ensure you have the correct admin permissions to edit the store.
- Clear browser cache or switch to another browser.
- If the issue persists, contact Amazon Seller Support.
