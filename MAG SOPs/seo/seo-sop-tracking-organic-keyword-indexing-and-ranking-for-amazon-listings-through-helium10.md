# **Tracking Organic Keyword Indexing and Ranking for Amazon Listings Through Helium10**

**Who is this for?** This is intended for Amazon Sellers and employees responsible to monitor the keyword movement of Amazon product listings.

**Objectives:**

- To track the performance of the keywords used on your listings;
- To have visibility of improvement across the targeted ASINs in the account

Sample: [Age of Sage - Keyword Tracking](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Age%20of%20Sage%20-%20Keyword%20Tracking.xlsx)

Before you begin, identify whether or not the ASINs you’re tracking are child or parent listings. If the ASIN is a child listing, be sure to exclude other variations by ticking the box (*Figure A*). Conversely, do not tick the box if the ASIN is a parent listing.

*Figure 1.*

**Scenario 1: You Own a Helium10 account**

**I. Target ASINs**

Keyword tracking (also known as Indexing Check) is normally done after Phase 1 or Phase 3 SEO has been implemented. In Phase 1 SEO (Master Keyword List or MKL), we typically highlight in bright yellow whatever keywords we used in constructing the search terms *(see Figure 2).* In keyword tracking, we utilize those highlighted keywords and add them to your Helium10 Keyword Tracker *(see Figure 3)*.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/SEO-SOP-Tracking-Organic-Keyword-Indexing-and-Ranking-for-Amazon-Listings-Through-Helium10-2.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/yJGimage.png)

*Figure 2.*

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Zbsimage.png)

*Figure 3.*

**II. Add the Keywords to Helium10 Keyword Tracker**

In the Keyword Tracker tool, input the ASIN that you are planning to track as well as the highlighted keywords from the MKL (Master Keyword List). Do not include the keywords used in generating the other search attributes. You may choose either to track the parent product and all variations or pick a couple of ASINs from the parentage.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/bOximage.png)

*Figure 4.*

If yes, tick the box that says “Track Parent Product and All Variations.” If not, untick it. Start tracking.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/ngLimage.png)

*Figure 5.*

After adding all the ASINs and keywords to Helium10 Keyword Tracker, let Helium10 absorb and gather all the data, and track it after a week.

**III. Track the Keywords**

Go to Helium10 Keyword Tracker and locate the ASIN you want to track. Sort the search volume from Z to A or in descending order, and export the current results.

*Figure 6.*

Access the downloaded file and delete all the columns except *Keyword, Search Volume, and Organic Rank***.**Create a filter *(figure 6)* and sort the organic rank in ascending order *(see example in Figure 7).*Delete those that do not have data under column C.

**Scenario 2: You don't Own a Helium10 account**

The process is pretty much the same, except for the part that we cannot add your target ASINs to Helium10 Keyword Tracker. In this case, we will be utilizing Helium10 Cerebro to get data.

**I. Reverse Product Lookup**

Run the ASIN in Helium10 Cerebro. Do not apply any filters; however, sort the search volume from Z to A or in descending order right away. Export data as a CSV file.

FAQs

- **Q1: How do I determine if the ASINs I am tracking are child or parent listings?**

  - **A1:** To track the performance of the keywords for ASINs, check if the ASIN is a child or parent listing. If it’s a child listing, make sure to exclude other variations by ticking the appropriate box. For a parent listing, do not tick the box.
- **Q2: How do I add keywords to the Helium10 Keyword Tracker?**

  - **A2:** In the Keyword Tracker tool, input the ASIN and the highlighted keywords from the Master Keyword List (MKL). For parent ASINs, you can track all variations, or for specific ASINs, select the variations to track.
- **Q3: How do I track keywords in Helium10 after adding them?**

  - **A3:** Once keywords are added, allow Helium10 to collect data for about a week. After that, access the Keyword Tracker, sort the search volume, and export the results. Clean up the data by deleting unnecessary columns and sorting the organic rank and search volume.
- **Q4: How do I transfer keyword tracking data to Google Sheets?**

  - **A4:** After sorting and filtering the data, transfer the results to Google Sheets. Create a new worksheet named using the format “(Account Name) - Indexing Check.” Customize the frequency of tracking and update all relevant fields, including the summary table.
- **Q5: What is the process for tracking keywords if I don’t own a Helium10 account?**

  - **A5:** If you don’t have a Helium10 account, use Helium10 Cerebro to perform a reverse product lookup. Export the data as a CSV, clean up the file by removing unnecessary columns, and transfer it to Google Sheets for ongoing tracking.
- **Q6: How can I view keyword distribution using Helium10?**

  - **A6:**Keyword distribution, including total keywords, sponsored keywords, organic keywords, and Amazon-recommended keywords, can be found when running the ASIN in Helium10 Cerebro.
