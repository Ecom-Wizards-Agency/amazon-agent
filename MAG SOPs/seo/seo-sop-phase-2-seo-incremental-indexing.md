# SEO SOP: Phase 2 SEO - Incremental Indexing

Source: https://sop.myamazonguy.com/books/seo/page/seo-sop-phase-2-seo-incremental-indexing

## **Phase 2 SEO - Incremental Indexing**

**Who is this for?**For individuals who want to conduct incremental indexing on their listings.

**Objective:**Your objective is to detect, eliminate, and replace repeat keywords *(pink terms)* from the search terms using **H10’s Scribbles Listing Optimizer.**

***NOTE****:*

- *Phase 2 SEO is done at least 30 days after*[*Phase 1 SEO*](https://sop.myamazonguy.com/books/seo/page/seo-sop-phase-1-how-to-construct-master-keyword-list-mkl-and-highly-optimized-backend-search-terms-through-helium10-cerebro-reverse-asin-lookup)*is completed. If titles, bullets, and descriptions (TBD) are not implemented on the backend, do not proceed with Phase 2 yet. Also, make sure that the listing has a****200+ organic keyword count****. Otherwise, Phase 2 must not be done.*
- If you are doing a **Phase 2 redo**, **you need to do another competitor research**. This is to ensure that we have relevant competitors with good BSR, which will help make phase 2 more successful.
- *The Phase 2 process can be skipped as long as the requirements for Phase 3 SEO are met:*

  - *The listing is more than 6 months old*
  - *Organic KW (Keyword) Count is 800+*
  - *Must have 50 words in ranks 20 -50*

**STEPS:**

**I. Prepare your SEO phase 2 worksheet.**

- Access the client’s folder in Google Drive and look for an existing phase 1 SEO worksheet. This is usually found in the “SEO Work” folder within the client’s folder.
- Make a copy of the[Phase 1 SEO](https://sop.myamazonguy.com/books/seo/page/seo-sop-phase-1-how-to-construct-master-keyword-list-mkl-and-highly-optimized-backend-search-terms-through-helium10-cerebro-reverse-asin-lookup) worksheet within the SEO Work folder and name it using the format “*(Client Name)* - Phase 2 SEO.”
- Add another column after the phase 1 search terms column, then label the header “New Search Terms.” Change the header of the existing phase 1 search terms column to “Current Search Terms.”
- Insert a row after the header and put the current date.

***NOTE****: Phase 1 SEO is the prerequisite of Phase 2 SEO. We cannot simply move forward with Phase 2 without establishing the search terms from the previous phase. It will defeat the purpose of Phase 1 SEO in so doing.*

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/d0gimage.png)

**II. Prep the Master Keyword List (MKL) for Phase 2.**

- Locate and open the existing MKL that was used in phase 1 to identify the top 10 competitor ASINs.**Make sure to check if these competitors are still active or relevant**
- Following the steps from [phase 1 SEO](https://sop.myamazonguy.com/books/seo/page/seo-sop-phase-1-how-to-construct-master-keyword-list-mkl-and-highly-optimized-backend-search-terms-through-helium10-cerebro-reverse-asin-lookup), do a reverse ASIN lookup on the same top 10 competitor ASINs through H10’s Cerebro to extract the keywords to create the MKL. Please refer to **item nos. 4 & 5 of the phase 1 SOP** for a step-by-step guide on doing a reverse ASIN lookup and creating an MKL.
- Competitor ASINs’ updated BSR data must be retrieved and used to sort them from smallest to largest

**III. Again, following the same steps from phase 1 SEO, process the keywords from the Master Keyword List (MKL)***(starting from the top of the list/words with the highest MAG score)* **in H10’s Keyword Processor (Frankenstein) to eliminate the duplicate keywords.**Please refer to **items nos. 6 & 7 of the phase 1 SOP** for a step-by-step guide on processing the keywords in Keyword Processor.

**Optional for Brand Awareness :** Make sure to retain all the brand names collated from the Phase 1 worksheet and add them to the last part of the MKL.

**IV. Eliminate the****words that are already in the title or bullet points** *(pink terms)***from the Search Term field.**Please follow the steps below:

**Using H10’s Scribbles Listing Optimizer tool:**

1. the current (phase 1) search terms and paste them into the Original Keywords field of the H10 Keyword Processor. Do the same for the new set of processed keywords *(step 3 resulting keywords)*from Keyword Processor. This process will eliminate the duplicate keywords in the current and the new search terms.

2. From the resulting keywords, remove the irrelevant and [prohibited phrases/keywords](https://sop.myamazonguy.com/books/seo/page/seo-sop-phrases-for-amazon-copy-and-search-terms-words-you-cant-use). Retain the misspellings and Spanish terms.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/QUFimage.png)

3. Import the listing / ASIN to H10 Scribbles.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/HRBimage.png)

4. Copy the resulting keywords from Keyword Processor and paste them into both the Words and Search Terms fields in Scribbles. Eliminate all the repeat keywords *(those with exponents >1)* by removing them from the Search Terms field.

**Note**: Stay under the length limit, which ranges from 246-249 characters, NOT bytes. Amazon has recently become stringent about this process and is now counting white spaces.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/phase-2-seo-updated-12-21-2022-image-4.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Rb5image.png)

5. Copy the resulting keywords from Scribbles and paste them into the [Charcounter](https://charcounter.com/en/) tool. Remove the keywords in excess of the 249-character limit for the search terms. Characters should not exceed 249.

**V. Paste them into the SEO phase 2 worksheet under the New Search Terms column.**

Once done, send your work to your QA Pod Lead for review and approval; your QA Pod Lead will deliver the report to the client.

  FAQs

  Prerequisites

  - **Q1: When should Phase 2 SEO be conducted?**

    - **A1:** Phase 2 SEO should be done at least 30 days after Phase 1 SEO is completed. Ensure the listing has a 200+ organic keyword count, and the titles, bullets, and descriptions (TBD) are implemented on the backend before proceeding.
  - **Q2: Can Phase 2 SEO be skipped?**

    - **A2:** Yes, if the requirements for Phase 3 SEO are met. These include:

      - The listing is more than 6 months old.
      - Organic keyword count is 800+.
      - The listing has 50 words ranking between positions 20-50.
  - **Q3: What should be done if Phase 2 SEO is being redone?**

    - **A3:** Conduct competitor research again to ensure you are comparing relevant competitors with good BSR. This will enhance the success of Phase 2.

  Worksheet Preparation

  - **Q4: How should the Phase 2 SEO worksheet be prepared?**

    - **A4:**

      - Access the client’s folder in Google Drive and locate the Phase 1 SEO worksheet in the “SEO Work” folder.
      - Make a copy of the worksheet and rename it using the format: “(Client Name) - Phase 2 SEO.”
      - Add a column labeled “New Search Terms” and rename the Phase 1 search terms column as “Current Search Terms.”
      - Insert the current date below the headers.
  - **Q5: Why is the Phase 1 SEO worksheet important for Phase 2?**

    - **A5:** Phase 1 SEO establishes the baseline search terms. Skipping this step defeats the purpose of Phase 1 and disrupts the SEO optimization process.

  Master Keyword List (MKL)

  - **Q6: How is the Master Keyword List (MKL) prepared for Phase 2?**

    - **A6:**

      - Locate the existing MKL from Phase 1 to identify the top 10 competitor ASINs.
      - Check the relevance and activity of these competitors.
      - Perform a reverse ASIN lookup on the top competitors using H10’s Cerebro to extract keywords.
      - Use updated BSR data to sort competitors from smallest to largest.
  - **Q7: Should brand names be retained in the MKL?**

    - **A7:** Yes, retain all brand names from the Phase 1 worksheet and add them to the end of the MKL for brand awareness purposes.

  Keyword Processing

  - **Q8: How are duplicate keywords handled in Phase 2?**

    - **A8:** Use H10’s Keyword Processor to process the MKL keywords. This tool helps eliminate duplicate keywords and refine the list.
  - **Q9: What should be done with words already in the title or bullet points?**

    - **A9:** Eliminate these words (pink terms) from the Search Term field using H10’s Scribbles Listing Optimizer tool.

  Scribbles Tool Workflow

  - **Q10: How do you use Scribbles to eliminate repeat keywords?**

    - **A10:**

      - Paste the Phase 1 search terms into the Original Keywords field in H10 Keyword Processor.
      - Add the newly processed keywords to the tool to remove duplicates.
      - Import the ASIN into Scribbles and paste the keywords into the Words and Search Terms fields.
      - Remove all repeat keywords (exponents >1).
  - **Q11: What is the character limit for search terms?**

    - **A11:** The search terms must not exceed 246-249 characters (not bytes), including white spaces.

  Final Steps

  - **Q12: What is the final step in Phase 2 SEO?**

    - **A12:** Copy the finalized search terms into the SEO Phase 2 worksheet under the “New Search Terms” column. Submit the worksheet to your QA Pod Lead for review and approval.
