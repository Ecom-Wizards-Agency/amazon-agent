## Creating Product Variations and Twisters

**Who is this for?** Intended for Amazon sellers who have access to Vendor Central and want to learn how to create variations for their products.

**Objective:** To provide step-by-step guidance on creating a variation in Vendor Central, the Amazon platform, in order to enable sellers to effectively manage their product variations and enhance their online retail experience.

**Instructions:**

1. Access the account by going to [https://vendorcentral.amazon.com/](https://vendorcentral.amazon.com/).

**How to Create variations**

Create a new variation family with new products that aren’t in your catalogue using a Bulk File

To create a product with variations using the bulk listing workflow, follow these steps:

1.Go to **Items** and click Add products.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-46G3JKOA.png)

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/6Gyimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/6Gyimage.png)

2.Under **Add new products,** click Download bulk listing spreadsheet. If you have variations enabled for the product type, the following attributes will display in the downloaded spreadsheet:

- Parentage level
- Child relationship type
- Parent SKU
- Variation theme name

Fill in column A by providing a value for your vendor code. This step will trigger the attribute requirement tags in the spreadsheet and highlight what cells must be filled in.

1. For the parent row, provide a vendor SKU. (A vendor SKU is a stock-keeping unit number or stock number used by selling partners to track and identify the product.) Then fill in the variation attributes, since this will reduce the number of required attributes for the parent ASIN:

- Parentage level: Set it to Parent to create a virtual ASIN to link all the child products under it. Since the parent ASIN is not sellable, do not provide any offer-related information, such as external IDs (UPC, EAN or GTIN), price or cost.

**Note:** Please verify the value provided for the parentage-level attribute, since this attribute cannot be edited by you or Selling Partner Support once the listing is created.

- Child relationship type: Select Variations.
- Parent SKU: Leave this field blank.
- Note: Do not leave the Vendor SKU field blank as it is mandatory to create the parent ASIN.
- Variation theme name: Select the applicable variation theme from the options in the drop-down list.

**Note:** When listing products or SKUs with variations, fill out the attributes under the Variations attribute group first. Based on the inputs provided here, such as whether an SKU is a child or parent, cells across the spreadsheet will be disabled accordingly if no additional input is needed. If you're listing products or SKUs that don't require variations, leave these variation attributes blank and proceed with listing creation.

1. For the other rows, fill in all mandatory attributes for the child ASINs:

- Parentage level: Set it to Child for the SKUs that belong to the variation family. These are actual sellable SKUs that you're listing your offer on.

**Note:** Please verify the value provided for the parentage-level attribute, since this attribute cannot be edited by you or Selling Partner Support once the listing is created.

- Child relationship type: Select Variations.
- Parent SKU: Provide the vendor SKU given to the virtual parent for all the child ASINs you want to group together. In case of multiple variation families, ensure that all the child ASINs that you want to group together are provided with the same parent SKU value.
- Variation theme name: Select the applicable variation theme from the drop-down list that matches the corresponding parent ASIN.

1. Save your completed spreadsheet and upload it.

Create a new variation family with existing individual products in your catalogue

To combine existing ASINs into one variation family, follow these steps:

1. Go to **Items**and select **Catalogue.**

  [![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/POlimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/POlimage.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-UA5FVAY1.png)2. Under **Refine by**, use the search bar to find the products that you want to group.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/DCuimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/DCuimage.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-A5IVOJD4.png)3. Select the ASINs that you want to create variations with and click **Download spreadsheet.**

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-88F6QJB7.png)

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/mUHimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/mUHimage.png)

4. In the downloaded spreadsheet, add a new row at the bottom of the prefilled products to create a virtual parent SKU.

5.For the new row, provide a vendor SKU and fill all other mandatory attributes by repeating steps 3-5 under Create a new variation family instructions above.

Add a new child product to a variation family that you created

You can add a SKU to a variation family that you previously created in Vendor Central.

1. Go to **Items** and select **Catalogue.**

  [![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/OaYimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/OaYimage.png)

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-FY2RXDNQ.png)
2. Under Refine by, use the search bar to find the products that you want to group.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-RLX40OK0.png)

  [![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/qSsimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/qSsimage.png)
3. Select the ASINs you want to group into an existing family and click Download spreadsheet.

  ![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-KBR3VG5C.png)

  [![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/O9Aimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/O9Aimage.png)
4. Provide all the required values in the template by following step 4 of the **Create a new variation family** instructions on this SOP.

**Note:** When you add a new SKU to a variation family, you must provide the details of the parent ASIN to which the child ASINs are to be linked with. Add a new row below the child SKUs and fill all required attributes for the parent ASIN. You can find the parent ASIN details on the **My Catalogue page**.

**Note:**You can verify that you’ve successfully created a variation family by searching with the parent SKU on My Catalogue. However, to confirm the publishing status, you should check the product detail page. Note the following reasons why a variation family may not be published, even it has been successfully created:

- If Amazon does not have inventory available in fulfilment centres, a child ASIN would not reflect on the detail page.

**Note:**The inventory published in the ARA report is a weekly snapshot and doesn’t show the real-time data. If you have inventory in hand, be prepared to fulfil purchase orders when they’re generated. Once we receive the units, the ASIN will reflect on the detail page.

- The child ASINs are suppressed or under review for malpractice.

FAQs

Troubleshooting

- **Q1: What if I am not redirected to the correct account after logging into Vendor Central?**

  - **A1:** If you are not redirected to the correct account after logging in, click on the account name in the top left corner and select "See All" to view a complete list of your accounts. Then, choose the correct account to work on.
    *Category: Account Management*
- **Q2: What should I do if I cannot find the "Variation Template" after searching in the Resource Center?**

  - **A2:** Ensure that you are searching for "Variation Template" in the search box within the View Resource Center. If it still doesn't appear, try checking for any spelling errors or variations in the search term, or contact Amazon support for assistance.

Template Instructions

- **Q3: How do I fill out the Variation Template correctly?**

  - **A3:**Fill out the Variation Template by providing the necessary details for each required column, such as Parent Title, Child Title, Brand, Variation Theme, and Model Name. Make sure to follow the template's formatting and instructions for each field.

Variation Management

- **Q4: Can I create multiple variations in one variation template?**

  - **A4:**Yes, you can create multiple variations within a single variation template, as long as you follow the formatting instructions provided in the template.

Support Case Submission

- **Q5: What should I do if I forget to attach the variation spreadsheet when submitting a support case?**

  - **A5:** If you forget to attach the variation spreadsheet when submitting the support case, you will need to create a new case and ensure the spreadsheet is attached. It’s crucial to include all the necessary grouping information for the ASINs in the variation template.

Product Categorization

- **Q6: How do I know which variation themes to choose for my products?**

  - **A6:** The variation themes for your products will depend on the categories your products are listed under. The Variation Template will provide you with the available options based on these categories. Follow the instructions provided in the spreadsheet.

Process Overview

- **Q7: What happens after I submit the variation template and create a support case?**

  - **A7:** After you submit the variation template and create a support case, Amazon's internal teams will process the request and create the twisters on your product landing pages based on the variations specified in the spreadsheet.

Variation Themes

- **Q8: How many variation themes can I include for a single variation?**

  - **A8:** You can include up to two variation themes for a single variation. Be sure to specify them accurately in the Variation Template.
