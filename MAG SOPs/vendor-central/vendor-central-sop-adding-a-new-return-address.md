# Vendor Central SOP: Adding a New Return Address

Source: https://sop.myamazonguy.com/books/vendor-central/page/vendor-central-sop-adding-a-new-return-address

## Adding a New Return Address

**Who is this for:** For vendors on Amazon who need to add a new return address to their account.

**Objective:** To outline the process for adding a new return address to ensure accurate and efficient handling of outgoing mail and packages

It is important to keep the return addresses for your company up to date in case we need to process returns.

**To add a new return address:**

1. Go to the "Settings" tab on the top menu of Vendor Central and choose "Return Addresses".

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1720632484887(1).png)

2. Click on the "Add a new address" button.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1687310916034.png)

3. Fill out all the necessary information in the text boxes, including the type of return you want to use for this address.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1687310970951.png)

4. Once you have filled out all the required fields, click on the "Add" button located at the bottom of the page.

FAQs

**1. How long does it take for a new return address to take effect?**

- Once added, a new return address typically updates within 24 hours. However, Amazon may review the address for accuracy before activating it.

**2. Can I add multiple return addresses to my Vendor Central account?**

- Yes, Amazon allows vendors to add multiple return addresses for different locations, warehouses, or return types.

**3. How do I set a default return address?**

- After adding multiple addresses, go to Settings > Return Addresses and select the address you want as the default. This ensures Amazon prioritizes that location for return shipments.

**4. Can I modify or delete an existing return address?**

- Yes. To edit an address, go to Settings > Return Addresses, select the address, make your changes, and click Save. To delete an address, click Remove next to it.

**5. What should I do if I can’t add a new return address?**

- If you experience issues:

  - Ensure all required fields are filled correctly.
  - Check if the address is already in use.
  - Try clearing your browser cache or using a different browser.
  - If the issue persists, contact Amazon Vendor Support.

**6. Does Amazon verify the return address before activating it?**

- Amazon may conduct a verification process to ensure the address is valid. Ensure your return address is a physical business location where packages can be received.

**7. Can I assign different return addresses for different return types?**

- Yes. When adding a new address, you can specify whether it should be used for defective, customer-damaged, or carrier-damaged returns.
