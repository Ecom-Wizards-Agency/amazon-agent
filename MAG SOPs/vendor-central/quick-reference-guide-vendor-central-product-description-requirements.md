# Quick Reference Guide: Vendor Central Product Description Requirements

Source: https://sop.myamazonguy.com/books/vendor-central/page/quick-reference-guide-vendor-central-product-description-requirements

## **Vendor Central Product Description Requirements**

**Product Description Guidelines**

- **Length**: 100–200 words.
- **HTML**: Use basic HTML tags like <p>, <br>, <li>, <ul>.
- **Sentence Structure**:

  - Write in complete sentences.
  - End sentences with a full stop.
- **Formatting**:

  - Use sentence case (e.g., “This is a product” instead of “THIS IS A PRODUCT”) except if it is for acronyms such as HDMI, USB, etc.
  - Avoid special characters like ™, €, ®, ¥, etc.

**Prohibited Elements**

- **No Images, Links, or Tables**: Do not use rich (A+) content images, links, or comparison tables.
- **Avoid Certain Phrases**:

  - Do not use phrases like eco-friendly, environmentally friendly, ecologically friendly, anti-microbial, anti-bacterial, Made from Bamboo, contains Bamboo, Made from Soy or contains Soy.
  - Do not include guarantees like ““Full refund” or “If not satisfied, send it back” or “Unconditional guarantee with no limit”
  - Do not include Amazon.com (or any other extension such as .co.uk or .de) or Amazon.
- **No Price, Company, or Website Information**: Do not mention product cost, vendor details, or website links.
- **No ASIN or N/A**: Avoid mentioning the ASIN number or using "not applicable," "NA," or "n/a."

**Content Focus**

- **Product Features & Benefits**: Introduce the product and explain its uses, features, and category relevance.
- **Clarity & Honesty**: Be concise, honest, and friendly in tone.

Use this guide to create a concise, clear, and compliant product description for your Amazon listings in Vendor Central.
