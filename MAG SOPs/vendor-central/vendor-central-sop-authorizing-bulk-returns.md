# Vendor Central SOP: Authorizing Bulk Returns

Source: https://sop.myamazonguy.com/books/vendor-central/page/vendor-central-sop-authorizing-bulk-returns

## Authorizing Bulk Returns

**Who is this for:**This is designed for vendors or sellers who have a business relationship with Amazon and need to authorize bulk returns of excess or damaged products initiated by Amazon.

**Objective:**To provide step-by-step instructions on accessing and reviewing return requests, downloading the request details as an Excel Spreadsheet, filling out the required fields in the spreadsheet, and finally, uploading the authorizations to the Amazon Vendor Central platform.

- If Amazon has excess or damaged products, they may request to return them to you. You will receive an Amazon-initiated Return Request when they want to return products to you. Your return requests will be in one of the following states: New, Approved, or Closed.

1. To view your return requests, go to Vendor Central and select “Returns” from the “Orders” tab's dropdown menu

2. The Returns page lists your return requests with relevant information. Use the “Show Search” button to search for approved or closed requests.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/IHCimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/IHCimage.png)

3. You can download the request as an excel spreadsheet by clicking the “Download as Excel Spreadsheet” button at the bottom of the displayed table.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/KsQimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/KsQimage.png)

- Request ID: A unique identifier for each return request. Click on it to see item-level detail.
- Vendor Code: Your vendor code with Amazon for the specific request.
- Request Type: Indicates the condition of the items being returned (sellable or damaged).
- Condition: Describes if the request is new, approved, or closed.
- Requested Quantity: The quantity of products that Amazon wants to return.
- Approved Quantity: The quantity of products you agree to accept in the return.
- Requested Refund: The amount Amazon requests for the return.
- Approved Refund: The amount you agree to remit to Amazon for the return.
- Next Automated Action: The next system action based on the return status.
- Warehouse: The Amazon fulfillment center where the return will originate.
- Authorization ID: An alpha-numeric string you create to identify the return.

5. After completing the spreadsheet, return to the Returns page and click the “Bulk Authorisation” button at the top.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/NjEimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/NjEimage.png)

6. On the next page, upload your authorizations by clicking the “Choose File” button, locating your spreadsheet, selecting it, and clicking the “Upload File” button.

FAQs

**1. Do all vendors need to authorize returns, or are some automatically approved?**

- Not all vendors need to manually authorize returns. Some vendors may have pre-agreed return policies with Amazon, allowing automatic approval. Check your Vendor Central settings or contact your Amazon representative for details.

**2. How long do I have to respond to a return request?**

- Vendors typically have 5–7 business days to approve or reject a return request. If you do not respond within this time, Amazon may process the return based on their default policies.

**3. What happens if I do not authorize or reject a return request in time?**

- If no action is taken within the given timeframe, Amazon may either proceed with returning the items to you or dispose of them based on their return policy. This could result in additional charges or product loss.

**4. Can I approve only part of a return request?**

- Yes, you can approve only part of a return request. In the spreadsheet, adjust the Approved Quantity and Approved Refund fields to reflect what you agree to accept.

**5. Can I edit or cancel a return authorization after submitting it?**

- No, once a return authorization has been submitted, it cannot be edited. If you need to make changes, contact Amazon Vendor Support as soon as possible.

**6. What file format and size are required for the bulk authorization spreadsheet?**

- The file must be uploaded in Excel (.xlsx or .xls) format and should not exceed 10MB. Ensure all fields are correctly filled to avoid processing errors.

**7. How will I be notified of a new return request?**

- Amazon notifies vendors through email and Vendor Central notifications. You can also manually check for new requests under Orders > Returns in Vendor Central.

**8. What should I do if I disagree with a return request?**

- If you believe a return request is incorrect, you can dispute it by contacting Amazon Vendor Support. Provide clear reasons and supporting documents (e.g., invoices, order details, product condition reports).
