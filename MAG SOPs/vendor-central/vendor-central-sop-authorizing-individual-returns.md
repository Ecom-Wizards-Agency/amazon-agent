# Vendor Central SOP: Authorizing Individual Returns

Source: https://sop.myamazonguy.com/books/vendor-central/page/vendor-central-sop-authorizing-individual-returns

## Authorizing Individual Returns

**Who is this for?:** Intended for individuals responsible for handling and authorizing individual returns within the organization.

**Objective:** To establish a set of guidelines and procedures for authorizing individual returns within an organization, ensuring compliance with legal and regulatory requirements while maintaining accuracy and efficiency in the return authorization process.

**To authorize return requests for individual products, follow these steps:**

1. Navigate to the “Returns” page in Vendor Central by selecting it from the Drop-Down Menu in the “Orders” tab on the top menu.

Note: if your account does not have this menu, file a ticket in Vendor Central.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/nNBimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/nNBimage.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1720667550773(1).png)

2. Click on the “Request ID” link of the return request you want to authorize.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1720667589517(1).png)

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/14Nimage.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/14Nimage.png)

3. In the Authorization Request Items section, you'll see a table with a list of all the products that Amazon wants to return. You can choose to authorize the return of all units for a product or a reduced quantity by changing the value listed in the “Approved Quantity” field. If you want to reject the return of a product entirely, simply enter 0.

[![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Fz5image.png)](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/Fz5image.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1720667750414(1).png)

4. If you change the refund quantity to less than what is requested, you'll be prompted and blocked. You'll need to reach out to your product management team to see if you have a refund reduction agreement with Amazon.

5. To provide a reason for a refund reduction, select the appropriate option from the “Rejection Reason” field. If you select “Other Reasons”, you must also enter a comment for the rejection in the “Rejection Reason Description” field.

6. Enter an alpha-numeric string in the “Authorization ID” field to serve as the Authorization ID for that product. If you want to enter the same Authorization ID for multiple products, select the line items, click “Bulk Edit”, enter the Authorization ID, and click “Apply”.

7. Once you've made all the changes, click “Save Changes”. The values for items are saved for future viewing but have not yet been submitted.

8. Finally, click the “Submit” button to authorize the return request.

**To refund the authorized amount to Amazon, follow these steps:**

1. After authorizing the return request, you'll see two options: “Submit” and “Destroy”.

2. Click “Submit” if you want Amazon to ship the products back to you.

3. Click “Destroy” if you want Amazon to destroy the products instead of shipping them back to you.

4. Review the information and click the “Submit” or “Destroy” buttons again to complete the authorization process.

FAQs

**1. What should I do if Amazon sends back a product that I never authorized for return?**

- If you receive an unauthorized return, first check Vendor Central to confirm whether a return request was mistakenly approved. If not, contact Amazon Vendor Support to dispute the return.

**2. Is there a deadline for authorizing or rejecting a return request?**

- Yes, return requests must be authorized or rejected within the timeframe specified by Amazon, typically within 30 days of the request. If you don’t respond, Amazon may process the return automatically.

**3. How long does Amazon hold returned products before destruction?**

- If you authorize the return but don’t request the items to be shipped back, Amazon typically holds the inventory for 30 days before destruction. Be sure to act promptly if you want the products returned.

**4. Does rejecting return requests affect my Amazon vendor performance metrics?**

- Amazon tracks return approvals and rejections. While rejecting a reasonable number of return requests won’t impact your account, frequent denials may flag your account for review. Consistently rejecting valid returns could lead to additional scrutiny from Amazon.

**5. How can I get a refund reduction agreement with Amazon?**

- Refund reduction agreements must be negotiated with your Amazon Vendor Manager. If your team doesn’t have an existing agreement in place, reach out to your Vendor Manager to discuss eligibility and terms.

**6. Can I modify or cancel a return request after submitting it?**

- No, once a return request has been submitted, it cannot be reversed or edited. Double-check all details before clicking “Submit.” If you need to make changes, contact Amazon Vendor Support immediately for assistance.

**7. How can I track a return shipment once Amazon has sent it back?**

- If you selected the “Submit” option for Amazon to return the products, tracking details will be available in Vendor Central under the Returns section. You can also contact Amazon for tracking updates if needed.
