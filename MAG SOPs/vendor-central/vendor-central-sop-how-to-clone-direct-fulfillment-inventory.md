# Vendor Central SOP: How to Clone Direct Fulfillment Inventory

Source: https://sop.myamazonguy.com/books/vendor-central/page/vendor-central-sop-how-to-clone-direct-fulfillment-inventory

## **How to Clone Direct Fulfillment Inventory**

**Who is this for?**

This SOP applies to Amazon specialists responsible for overseeing and executing the cloning of products into Direct Fulfillment inventory on Vendor Central.

**Objective:**

To guide specialists through the process of cloning products listed in the Retail (PO) catalog for use in Direct Fulfillment.

**Procedure**

**1. Log in to**[**Vendor Central**](https://vendorcentral.amazon.com/home/vc)

**2. Navigate to Direct Fulfillment Inventory**

- From the main menu, hover over the **Items** tab.
- From the dropdown, select **Direct Fulfillment Inventory**.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/vendor-central-sop-how-to-clone-direct-fulfillment-inventory-image-gi3udn8m.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/u2Aimage.png)

**3. Initiate the Cloning Process**

- Click on **Clone ASINs from your Retail (PO) Catalog** to begin cloning products into the Direct Fulfillment inventory.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/vendor-central-sop-how-to-clone-direct-fulfillment-inventory-image-jhrnfekl.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/CR5image.png)

**4. Select the Appropriate PO Vendor Code**

- Choose the correct PO vendor code that corresponds to the catalog where the products are currently listed.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/vendor-central-sop-how-to-clone-direct-fulfillment-inventory-image-xvjcu5kc.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/uKsimage.png)

**5. Download the Cloning File Template**

- Click on **Cloning file template** to download the cloning template.
- This template is required to upload the ASINs to be cloned.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/sn9image.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/vendor-central-sop-how-to-clone-direct-fulfillment-inventory-image-swmtp5rb.png)

**6. Populate the Cloning Template**

- Open the downloaded cloning template on your computer.
- Add the ASINs you wish to clone into the template. Ensure that these ASINs are part of the PO vendor code you selected in Step 4.
- Save the template.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/fmyimage.png)

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/vendor-central-sop-how-to-clone-direct-fulfillment-inventory-image-1v576ecg.png)

**7. Upload the Cloning File**

- Return to Vendor Central and navigate back to the **Upload cloning file** section.
- Click **Browse** to locate and select the saved template file from Step 6.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/vendor-central-sop-how-to-clone-direct-fulfillment-inventory-image-0m60tpes.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/nIeimage.png)

- Upload the template file by clicking **Submit**.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/vendor-central-sop-how-to-clone-direct-fulfillment-inventory-image-xmfv3ryt.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/KnAimage.png)

**8. Monitor the Cloning Process**

- After uploading the template, you can track the status of the cloning process.
- To check the status, navigate back to **Clone ASINs from your Retail (PO) Catalog**.
- Click on **Last cloning request status** to view whether the cloning process has been completed successfully.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/vendor-central-sop-how-to-clone-direct-fulfillment-inventory-image-ourkmtyx.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/9YBimage.png)

**Important Notes**

- **Cost Replication:** The cloning process replicates the product’s ASIN information, including the current cost at the time of cloning. If the cost for Direct Fulfillment differs from the Retail catalog, you must submit a **cost update** after the cloning process is complete.
- **Template Accuracy:** Ensure all ASINs listed in the template are from the selected PO vendor code. Any mismatch could lead to errors in the cloning process.
