# Walmart SOP: How To Update a Listing Manually in Walmart

## **How To Update a Listing Manually in Walmart**

**Who is this for :** For various teams or individuals responsible for updating the content on the Walmart platforms.

**Objective :** To provide step-by-step instructions for updating listing information on Walmart, accessing the backend, and saving the changes, in order to ensure that new listing attributes are accurately reflected and absorbed by the Walmart system.

**Instructions:**

1. Log in to Walmart seller central account. Go to https://seller.walmart.com/

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1710430999201.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Jqmimage.png)

2. Go to Catalog in the main menu, click on a listing or the three dots under the Actions tab, and select Edit Item.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1733784262472.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/cb0image.png)

3. Edit the attributes that need to be updated and click Submit.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1733784382066.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/SZpimage.png)

4. A Feed ID will be generated. To monitor the upload or changes, go to Activity Feed under Catalog and locate the Feed ID.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1733784837430.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Qyjimage.png)

5. If the upload has errors, view the error report to review the issues, make the necessary corrections, and re-submit your upload.

6. If the updated attributes are viewable in the PDP such as title, description, bullets, etc. Visit the product detail page to confirm if the changes were successfully through. Click the item or listing and click View on Walmart.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1733785053130.png)

FAQs

1. Which attributes can I update for a single listing on Walmart?

You can update the following attributes:

- Title: Update product name to be clear and keyword-optimized.
- Description: Improve or change the product description for better clarity and SEO.
- Key Features (Bullet Points): Add or modify bullet points for clarity and SEO.
- Price: Modify the product price based on changes in pricing strategy.
- Images: Update images to reflect accurate product visuals (e.g., new product images or improved resolution).
- Inventory: Update stock levels to reflect the actual availability.

Some attributes like GTIN, SKU, and brand name cannot be updated through the Seller Center interface and require support assistance.

2. Are there any attributes that cannot be updated through Walmart Seller Center?

Yes, certain attributes are locked after the initial listing creation, such as:

- GTIN (UPC/ISBN)
- SKU
- Brand Name

To update these, you'll need to submit a case to Walmart Partner Support or, in some cases, create a new listing.

3. How do I confirm that my changes have been successfully applied to Walmart?

- Check the Product Detail Page (PDP): Click the item in Seller Center and select View on Walmart to confirm if the updates (title, description, bullet points, images) are reflected.
- Activity Feed: Go to Activity Feed under Catalog, and search by the Feed ID to see if the upload was successful.
- Item Report: Download the Item Report to verify that all attributes are updated in the Walmart system.

4. What should I do if my update results in an error?

1. Go to Activity Feed and find the Feed ID for the update.
2. Download the error report and review the issues (e.g., missing required fields, invalid data).
3. Correct the errors (e.g., add missing details, fix formatting) in the listing.
4. Re-upload the file after making the corrections.
  Common errors include:

- Missing required fields (e.g., product identifiers).
- Incorrect formatting (e.g., special characters or exceeding character limits).

5. How long does it take for listing updates to reflect on Walmart?

- Simple updates (price, inventory, title, description): Typically take 15 minutes to 2 hours.
- Content updates (images, key features, bullet points): Can take 2 to 4 hours.
- Category or attribute updates: May take up to 24 hours to fully reflect on Walmart.

6. What are Walmart’s content restrictions when updating listings?

When updating listings, ensure the content adheres to Walmart’s Listing Quality Guidelines, including:

- No promotional language (e.g., “Best Price,” “Limited Time Offer”).
- Avoid using all caps or emojis in the title and description.
- Do not use misleading claims (e.g., “Guaranteed to Cure” for supplements).
- Ensure the images are high quality, at least 1000x1000 pixels, and do not contain watermarks or logos.

7. How do I update the images on Walmart listings?

- Image Requirements: Ensure your images meet Walmart’s minimum resolution of 1000x1000 pixels for zoom functionality.
- File Format: Use JPG, PNG, or GIF formats for images.
- Image Guidelines: The primary image should be the front view of the product with a white background. Add lifestyle images for a more detailed product view.
- Re-upload Images: Go to the Edit Item page and update the image URLs under the “Images” section.

8. What should I do if updates are not reflected after 24 hours?

1. Check the Activity Feed for any errors or failures in the upload.
2. Confirm the changes in the backend by revisiting the listing and reviewing the updated attributes.
3. If updates are still not visible on Walmart.com, submit a case to Walmart Partner Support for further assistance.
4. Escalate the case if the issue is not resolved within 5 business days.

9. Can I update multiple listings at once, or do I have to update each one individually?

- For multiple listings, you can use the bulk update feature via Item Spec Sheets.
- For single listings, use the Edit Item page in Walmart Seller Center.
- For bulk updates, select Update Items in Bulk and use the Update with File option to update attributes across multiple listings at once.

10. What if my listing update includes multiple products and I encounter errors?

- Check for common errors such as missing fields, incorrect formats, or locked attributes.
- Download the error report from Activity Feed to see exactly where the issues lie.
- Fix the errors in the CSV or Excel file and re-upload. If the issues persist, reach out to Walmart support for assistance.
