# Walmart SOP: Create Multiple Items in Walmart via Flat File (Seller Fulfilled Type)

Source: https://sop.myamazonguy.com/books/walmart/page/walmart-sop-create-multiple-items-in-walmart-via-flat-file-seller-fulfilled-type

## **Create Multiple Items in Walmart via Flat File**

##### **(Seller Fulfilled Type)**

**Who is this for?**For individuals who to create new listings on their Walmart Seller Central account.

**Objective :**Successfully create bulk listings in Walmart via flat file and know all the required attributes to be able to create the listing.

The bare minimum requirements in creating Walmart listings are:

- SKU
- Product ID (UPC, EAN, GTIN, ISBN)
- Product ID Type
- Product Name
- Description
- Key Product Features (not required but good to have)
- Brand
- Selling Price
- Main Image URL
- Secondary Images (not required but good to have)
- Package Weight (lbs) if in a different weight unit please convert to lbs.
- Restriction Type (this attribute is category specific)

**Overview**

In this guide, you’ll learn how to upload a bulk listing report or custom spreadsheet from any external marketplace to build your Walmart catalog in Seller Center.

**Instructions on Uploading Items in Bulk:**

**Step 1** — Log in to Walmart seller central account. Go to [https://seller.walmart.com/](https://seller.walmart.com/).

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/thRimage.png)

**Step 2** — In the Seller Center, navigate to your Catalog and select the Add items button.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/wqBimage.png)

**Step 3** — You will be redirected to the Add Items page, and click on the Upload in bulk button.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/9Xvimage.png)

**Step 4** — On the Upload in bulk page, scroll down to Download a Spreadsheet. Click on the Create new items button.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/nzDimage.png)

**Step 5** — Choose Categories, Product Types, and the Fulfillment Type. Once done selecting the Category and Product Types, click on Next. Once selected, click on Download.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/5Zhimage.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/jtqimage.png)

**Step 6** — Open the downloaded flat file and fill out the required columns, then click on Save.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/image-1689272266737(1).png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/HDNimage.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/tIyimage.png)

**Step 7** — Go back to the Add Items page and Upload the filled out template. After uploading the file, a checkmark will appear above the file name. Select the Next button in the bottom right corner of the page to submit the file. To complete the uploading process, select Import items or Activate the experts, depending on the source of your items. If the Activate the experts option is available to you, it means that your spreadsheet will be converted by Walmart experts, at no cost to you.

![image](https://files.document360.io/d9e9b7a2-6758-4c75-97d3-36a6877048cf/Images/Documentation/Screenshot%202025-02-10%20at%2010.19.04%E2%80%AFPM.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/pE3image.png)

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/Ef6image.png)

**Step 8** — Check the Feed ID on Activity Feed under Catalog to see if the update is successfully processed. If the upload has errors, download the error report to read through the issues. Try to fix all issues and re-upload the revised spreadsheet/file.

![image.png](https://sop.myamazonguy.com/uploads/images/gallery/2025-08/scaled-1680-/6SFimage.png)

**Step 9** — To verify if you have successfully created the listings. Go to Catalog and you should be able to see the new listings in this section in 5-10 minutes (depending on the number of listings that have been loaded).

Visit the product detail page to confirm if all information is correctly showing on the PDP. Hover the mouse over the listing and click View on [Walmart.com](http://Walmart.com).

FAQs

1. What should I do if I encounter errors when uploading a flat file?

If your bulk upload has errors, follow these steps:

1. Go to Activity Feed and search by Feed ID.
2. Download the error report. The common errors may include:

  - Invalid Product ID (UPC, GTIN): Ensure the Product ID matches the required format.
  - Missing Fields: Ensure that all mandatory fields (e.g., SKU, Product Name, Price) are filled out.
  - Incorrect Formatting: Ensure that there are no special characters or spaces in fields like the Product ID or Price.
  - Invalid Image URLs: Ensure the image links are properly formatted and meet Walmart's image size requirements (1000x1000 pixels minimum).
3. Fix the issues identified in the error report and re-upload the file.

2. What is the proper file format for Walmart bulk uploads?

- Walmart accepts CSV (Comma-Separated Values) files for bulk uploads.
- Ensure that the data is properly structured with no extra spaces or special characters that may cause issues.
- If you're using Excel, save the file as a CSV format before uploading to ensure compatibility.

3. How do I handle category-specific fields when uploading a flat file?

Each category on Walmart may have unique attributes that must be filled out:

1. Select the correct template when downloading the flat file based on your product category (e.g., Electronics, Clothing).
2. Use Walmart's Category Taxonomy to identify which additional fields are required for your category (e.g., color, size, material).
3. Fill in all mandatory fields specific to the selected category to ensure your upload is successful.

4. How do I manage product variants or multiple SKUs in the flat file?

For multiple SKUs or variants, follow these steps:

1. Each variant should have a unique SKU and must be associated with a parent SKU for grouping.
2. Include the variant attributes (e.g., color, size) in the appropriate columns.
3. If uploading multiple SKUs for the same product, use the parent-child relationship in the flat file for easy management.
4. Ensure that all required attributes (e.g., price, stock quantity, product details) are correctly filled out for each SKU.

5. How long does it take for bulk uploads to be processed on Walmart?

- Small updates (price, inventory) usually take 15 minutes to 2 hours.
- Content-heavy updates (e.g., titles, descriptions, images) can take up to 4 hours.
- Category updates or other more complex changes may take up to 24 hours.

6. How do I verify that the updates were applied correctly after uploading?

- Activity Feed: First, check the Activity Feed using the Feed ID to confirm if the upload was successful.
- Product Detail Page: After confirming the successful upload, click View on Walmart.com to ensure that all information (price, title, description, images) is correctly reflected on the PDP.
- Item Report: For thorough verification, you can also download the Item Report to cross-check that all fields in the flat file were updated correctly in Walmart’s backend system.

7. What are the limitations on the number of SKUs I can upload at once?

- Walmart generally recommends uploading up to 1,000 SKUs per flat file to avoid processing errors.
- For larger files, consider breaking your batch into smaller files to reduce the risk of errors and ensure faster processing.
- If you're uploading multiple variants for products, consider splitting them into smaller batches based on product types or categories to avoid issues.

8. What if the updates are not reflecting after 24 hours?

1. Check the Feed Status in the Activity Feed to see if the feed was successfully processed.
2. Ensure the changes are reflected on the product page by clicking View on Walmart.com.
3. If the updates are still not visible, consider contacting Walmart support for further assistance or troubleshooting.
4. If there is a consistent issue, escalate the case with Walmart's Partner Support.

9. How do I select the correct template for my category?

- When downloading the flat file, you must choose the correct product category from the dropdown options.
- Each category has its own specific template. Walmart will prompt you to select the right category and product type during the download process.
- Make sure you download and use the template for the specific category your products belong to, as it will have the relevant fields for your products.

10. Can I upload listings for multiple categories at once?

- You can upload listings for multiple categories in a single flat file as long as the SKUs are associated with the correct category type.
- However, ensure that the attributes for each category are properly filled out based on Walmart’s category-specific guidelines.
