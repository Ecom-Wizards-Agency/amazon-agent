# Change Your Seller Order ID


When you confirm shipments, you can include your seller order ID. You might want to do this if you use a different ID in your business processes. For example, you might have an automated invoicing system that tracks your orders using your own order ID.

To edit your seller order ID, follow these steps:

Click the Orders tab, and then click Manage Orders.
Click the Order ID link in the Order Details column.
On the Order Details page, click Edit (located next to Your Seller Order ID and below Amazon's Order ID).
Enter your seller order ID.
Note: Your order ID can be up to 50 alphanumeric characters.
Click Save.

You can also edit multiple seller order IDs in the following ways:

Include the seller order IDs when sending an Order Acknowledgment Feed.
