# Country-specific trademark information


To be eligible for Amazon Brand Registry, your brand must have one of the following:

An active registered trademark or pending registration trademark in each country where you would like to enroll
A pending trademark application that is filed through Amazon IP Accelerator

To learn more about the enrollment process go to the Brand Registry application step-by-step guide.

You can also review the latest version of the Brand Registry application guide PDF in English or Chinese.
