# Attributes guide


Product attributes are specific facts unique to your product. This page explains why they're important for your listings, and how to prioritize them effectively. Understanding and implementing comprehensive product attributes can help increase your product visibility in search results and provides customers with the essential information they need to make confident purchasing decisions.

Product attributes vary based on the product type you select, such as the following examples:

Shoes have attributes like color and size.
Lamps have attributes like wattage and bulb type.

Some attributes combine multiple values to provide complete information. For example, unit count consists of both a number value and a unit count type (such as "12 count" or "6 pack").

Benefits of complete attributes

Improved search visibility

Product attributes help index your products, which ensures they appear in relevant customer searches. They also power the refinement options (the filter check boxes in the left panel) that helps customers narrow their search results.

To learn more, go to Keyword attributes explained.

Enhanced product evaluation

When customers visit your product detail page, comprehensive attributes help them understand your product specifications and determine if it meets their needs. The more complete your product attributes, the higher the chance customers have all the information needed to make a buying decision.

AI assistant support

Complete product attributes enable Rufus (Amazon's AI shopping assistant) to provide more accurate context to customers while they search for products or evaluate specific items.

Attribute categories and priorities

Attributes in listings are categorized into three views:

Required: Must be completed for your listing to be created
Recommended: Important attributes that customers typically look for
All: Complete set of available attributes

We continuously curate these attribute lists based on the latest customer trends. We recommend that you provide all attributes in the Recommended view to ensure your products include the information customers typically look for during search and evaluation.

To stay updated with the latest attribute requirements, review the planned changes document.

Troubleshooting

Important: When you list your products, use only Excel templates that are less than 90 days, as older templates may not be in parity with the latest requirements. You can find the version of your template in cell B1 of every template that you upload.

Error code	Description	Cause	Resolution steps
Error 90057	Invalid value error	An attribute contains a value that’s not present in the valid value list.	Replace the invalid value with valid value from the attribute's drop-down list and resubmit the listing. For more information, go to Error 90057.
Error 90220	Missing required attribute	An attribute on the listing is missing.	Provide the missing value and resubmit the listing. If you’re using an older template, ensure you download the latest one.
Error 90225	Character limit exceeded	An attribute value exceeds the maximum allowed character count.	Shorten the attribute value to stay below the allowed threshold.
Error 90248	Conditional field error	A field is provided for an attribute when specific preconditions prohibit it.	Review the attribute's preconditions and remove fields that must not be provided under the current circumstances.
Error 97779	Generic keywords length exceeded	The generic keywords exceed our length limit of 200 bytes.	Reduce the length of keywords to less than 200 bytes.
Error 99001	Missing required value	A required attribute or column value is missing.	Check all cells highlighted in red in your template, fill in all required fields, and resubmit the listing. For more information, go to Error 99001.
Error 99010	Missing conditional group value	One or more required values from a conditionally required group are missing or conflicting.	

Ensure all attributes in the conditionally required group contain valid values.
Check the Data definition and Valid values tabs in category-specific flat files before you upload.
Resubmit with complete information.

For more information, go to Error 99010.

Note: For attributes that have drop-down values, do not copy and paste invalid values that are not a part of the drop-down list. This will result in an error. Select applicable values from the drop-down list only.
