# SOP Updates

This folder is the GitHub-synced audit trail for verified SOP corrections.

Use `sop-updates/` only after a SOP correction has been checked and applied to a tracked source file. Keep one markdown change note per correction.

Do not store screenshots, GIFs, exports, zip files, client deliverables, or other heavy/local artifacts here. Link to or summarize local/pCloud evidence instead.

Filename format:

```text
YYYY-MM-DD_{short-topic}.md
```

New SOP drafts belong in `sop-drafts/`. Local screenshots and working evidence belong in ignored local folders:

```text
output/{client-or-brand-or-general}/sop-maintenance/
evidence/{client-or-brand-or-general}/sop-maintenance/
```
